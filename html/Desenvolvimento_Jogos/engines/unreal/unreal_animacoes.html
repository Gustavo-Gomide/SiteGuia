Animacoes no Unreal Engine 5

Animation Blueprint, State Machine, Montages e IK Rig

O pipeline completo de animacao de personagem no Unreal: AnimBP, blend spaces, montages e procedural IK.

Animacao no Unreal esta distribuida entre varios sistemas que colaboram: o Animation Blueprint orquestra estados e blending, Montages controlam ataques e reacoes interruptiveis, e o IK Rig adiciona correcçao procedural sobre o resultado. Entender quando usar cada um e essencial.

Animation Blueprint: AnimGraph e Event Graph

O Animation Blueprint tem dois grafos com funcoes distintas. O Event Graph roda na game thread — e onde voce le dados de gameplay (velocidade, direcao, is_grounded) e os escreve em variaveis. O AnimGraph roda na animation thread — usa essas variaveis para calcular a pose final via blend nodes, state machines e IK. Nunca faca logica de gameplay no AnimGraph; nunca faca calculo de pose no Event Graph.

State Machine e Blend Spaces

State Machine

Define estados de animacao (Idle, Walk, Run, Jump, Fall) e as transicoes entre eles com condicoes booleanas ou float. Cada estado pode conter um blend space ou clip. Transicoes tem blend time configuravel e pode usar automatic rule (transicao automatica ao terminar) para estados com animacoes one-shot.

Blend Space 1D / 2D

Blend Space 1D interpola clips em funcao de um eixo (velocidade 0 a 600). Blend Space 2D usa dois eixos (velocidade + direcao lateral) para 8-directional movement. Amostras no grid definem qual clip domina em cada posicao. A interpolacao e bilinear — sem snap, sem pop visual entre direcoes.

Montages e Layered Blending

Animation Montage e o sistema para animacoes interruptiveis — ataques, rolls, hit reactions, interacoes. Um Montage e dividido em sections (Windup, Strike, Recovery) e pode ser interrompido, saltado ou encadeado via codigo. Notifies no Montage disparam eventos (som, efeito, hitbox ativo) em frames especificos sem precisar de Blueprint tick.

Layered Blend per Bone combina dois estados: pose do corpo inteiro do state machine + animacao de upper body de um Montage de ataque, blendados no bone da cintura. Resultado: personagem pode correr e atacar simultaneamente sem pose de ataque cancelar a corrida.

IK Rig e Control Rig

O IK Rig (UE5) define solvers de Inverse Kinematics para um esqueleto — Full Body IK, Two Bone IK para membros. Correcoes procedurais como pe que se adapta a inclinacao do terreno e cabeca que olha para o inimigo sao implementadas aqui. O IK e calculado sobre a pose final do AnimGraph — nao substitui animacao, corrige.

O Control Rig vai alem: e um rigging procedural completo dentro do Unreal, usado para criar animacoes in-engine sem DCC externo. Usado pelo time da Epic para Motion Matching e animacoes de cinematica. Para a maioria dos projetos, IK Rig e suficiente — Control Rig tem curva de aprendizado de rigger tecnico.
