Retrofit

Cliente HTTP Tipado no Android

Retrofit simplifica integração REST com contratos claros de API.

Retrofit reduz boilerplate de rede e centraliza serializacao, autenticação e tratamento de erro.

Pecas do stack

  • Interface de endpoints com anotações.
  • Converter JSON (Moshi/Gson/Kotlinx).
  • OkHttp com interceptors para auth e logs.

Tratamento de erro

Padronize timeout, retry e mapeamento de HTTP codes para estados de UI. Erro de rede sem semantica gera mensagens inconsistentes para usuario.
