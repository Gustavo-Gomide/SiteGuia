Jetpack Compose

UI Declarativa para Android Moderno

Compose reduz boilerplate de XML e organiza UI por estado.

Compose muda o modelo mental: em vez de manipular view, voce descreve tela como funcao do estado.

Estado e recomposicao

Quando estado observado muda, Compose recompõe apenas partes necessarias da arvore. Estado unidirecional facilita debug e previsibilidade de UI.

Boas praticas

  • State hoisting para separar UI e logica.
  • Use ViewModel como fonte de estado.
  • Evite side effects fora de APIs de efeito.

Referencias

  • GOOGLE. Jetpack Compose Documentation. Android Developers, 2025.

Compose acelera entrega, mas exige disciplina de estado para escalar.
