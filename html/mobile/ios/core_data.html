Core Data

Persistencia Estruturada no iOS

Gerencie dados locais com modelo versionado e consultas eficientes.

Core Data e uma camada de gerenciamento de objetos persistentes, essencial para apps offline e historico local.

Conceitos centrais

  • NSManagedObjectContext para unidade de trabalho.
  • NSFetchRequest para consultas.
  • Migracao de modelo para evolucao sem perda.

Referencias

  • APPLE. Core Data Documentation. Apple Developer, 2025.
