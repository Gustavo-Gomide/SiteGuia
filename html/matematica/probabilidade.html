    Probabilidade e Estatistica

    Hero explicativo

    Probabilidade e estatistica sustentam previsao, monitoramento e IA. Em TI, elas transformam incerteza em criterio tecnico para decidir.

    Fundamentos

    Probabilidade descreve eventos futuros; estatistica analisa dados coletados. Conceitos centrais: distribuicao, media, variancia, amostragem e inferencia.

    Como funciona / metodo

    Defina a pergunta, selecione metricas, colete dados, estime parametros e valide suposicoes. Esse ciclo evita conclusoes superficiais.

    Exemplo aplicado

    Em 200 requisicoes, 18 falharam. Taxa de falha estimada: 18/200 = 0,09. Em comparacao com meta de 0,05, o sistema exige investigacao de causa.

    Topicos avancados / armadilhas

    Armadilhas comuns: vies de selecao, uso indevido de p-valor, e extrapolacao sem intervalo de confianca. Correlacao nao implica causalidade.

    Referencias

    • DeGroot; Schervish. Probability and Statistics.
    • Wasserman, Larry. All of Statistics.
    • Ross, Sheldon. A First Course in Probability.

      Probabilidade e Estatistica

      Hero explicativo

      Probabilidade e estatistica sustentam diagnostico de sistemas, previsao e IA. Em TI, elas permitem decidir sob incerteza com criterios quantitativos.

      Fundamentos

      Probabilidade modela eventos futuros; estatistica extrai informacao de dados observados. Conceitos basicos: variavel aleatoria, distribuicao, media, variancia e intervalo de confianca.

      Como funciona / metodo

      Fluxo recomendado: formular pergunta, coletar dados confiaveis, escolher metrica/distribuicao, estimar parametros e validar suposicoes do modelo.

      Exemplo aplicado

      Em monitoramento, o tempo de resposta medio subiu de 120 ms para 140 ms. Com desvio padrao de 30 ms e n=100, o erro padrao e 3 ms. A diferenca de 20 ms e grande frente ao erro, indicando mudanca real.

      Topicos avancados / armadilhas

      Evite p-hacking, testes repetidos sem correcao e uso de media em dados muito assimetricos. Em dados de producao, vies de coleta costuma ser o principal risco.

      Referencias

      • DeGroot; Schervish. Probability and Statistics.
      • Wasserman, Larry. All of Statistics.
      • Ross, Sheldon. A First Course in Probability.

      Distribuição Exponencial Exp(λ)

      f(x)= λe−λx , x≥0

      Quando usar

      Tempo de espera entre eventos Poisson com taxa média λ. Processo sem memória.

      Função de distribuição

      F(x)=1−e−λx

      E[X]=1λ e Var(X)=1λ2

      Sem memória

      P(X>s+t | X>s)=P(X>t). O passado não altera o futuro.

      Exemplo

      Com λ=2 chamadas/min, probabilidade de esperar menos de 30s (0,5 min):

      P(X<0.5)=1−e−2·0.5=1−e−1

      Probabilidade: Conceitos básicos

      Espaço amostral S, evento A ⊆ S, probabilidade clássica quando equiprovável:

      P(A) = n(A) n(S)

      Guia rápido de sinais e notação

      P(A), A ∪ B, A ∩ B, Ac, |, Σ, ∫, ≈, ≥ ≤.

      Independência: A ⟂ B significa que P(A|B)=P(A).

      Regra da Adição

      P(A∪B)=P(A)+P(B)−P(A∩B)

      Complemento

      P(Ac)=1−P(A)

      Regras e Teorema de Bayes

      Prob. Condicional

      P(A|B)=P(A∩B)P(B)

      Independência: P(A∩B)=P(A)P(B).

      Teorema de Bayes

      P(A|B)= P(B|A)P(A) ∑ i P(B|Ai)P(Ai)

      Variáveis Aleatórias

      Discretas (p.m.f.) e contínuas (p.d.f.).

      Esperança

      E[X]=∑xxp(x)
      E[X]=∫xf(x)dx

      A esperança (ou valor esperado) é a média teórica de uma variável aleatória. Para variáveis discretas, é a soma ponderada pelos pesos p(x); para contínuas, é a integral de x f(x). Interpretação: se o experimento fosse repetido indefinidamente, a média observada tenderia a E[X].

      Símbolos: E[X], p(x), f(x), ∑/∫.

      Exemplo (discreto)

      Uma moeda viciada dá cara com p=0.6 e coroa com p=0.4. Se X=1 em cara e X=0 em coroa, então

      E[X]=1·0.6+0·0.4=0.6

      MEDIANA - FÓRMULA COMPLETA

      Mediana = se n é ímpar:   x n+1 2 , se n é par:   x n2 + x n2 + 1 2

      Exemplo 1 - n ímpar

      Dados: [8, 3, 5, 1, 9]

      8 3 5 1 9

      Passo 1: Ordenar → [1, 3, 5, 8, 9]

      Passo 2: n = 5 (ímpar)

      Passo 3: Posição = (5+1)/2 = 3ª posição

      Passo 4: Valor na 3ª posição = 5

      Mediana = 5

      Exemplo 2 - n par

      Dados: [12, 7, 15, 3, 8, 10]

      12 7 15 3 8 10

      Passo 1: Ordenar → [3, 7, 8, 10, 12, 15]

      Passo 2: n = 6 (par)

      Passo 3: Posições centrais = 3ª e 4ª

      Passo 4: Valores = 8 e 10

      Passo 5: Mediana = (8 + 10) / 2 = 9

      Mediana = 9

def calcular_mediana(dados):
    dados_ordenados = sorted(dados)
    n = len(dados_ordenados)
    
    if n % 2 == 1:  # n ímpar
        return dados_ordenados[n // 2]
    else:  # n par
        meio1 = dados_ordenados[n // 2 - 1]
        meio2 = dados_ordenados[n // 2]
        return (meio1 + meio2) / 2

      Variância e desvio padrão — Medidas de dispersão

      Variância populacional

      σ2 = ∑ xi − μ 2 N

      Onde:

      • σ2 = variância populacional
      • xi = cada valor
      • μ = média populacional
      • N = tamanho da população

      Variância amostral

      s2 = ∑ xi − x̄ 2 n−1

      Onde:

      • s2 = variância amostral
      • x̄ = média amostral
      • n = tamanho da amostra
      • n−1 = correção de Bessel

      Desvio padrão populacional

      σ = σ2

      Raiz quadrada da variância populacional

      Desvio padrão amostral

      s = s2

      Raiz quadrada da variância amostral

      Outlier (valor atípico)

      Valor que foge ao padrão do conjunto. Às vezes chamado por engano de "outliner" — o termo correto é outlier.

      Como detectar

      • IQR (Q1, Q3): pontos além de Q1−1.5·IQ ou Q3+1.5·IQ.
      • Z-score: |Z|≥3 (regra prática).

      Impacto

      • Aumentam a variância e o desvio padrão.
      • Podem distorcer a média; a mediana é mais robusta.

      O que fazer

      • Verificar se é erro de digitação/medição.
      • Analisar contexto: outliers podem ser fenômenos reais importantes.
      • Usar métricas robustas (mediana, IQR) ou modelos apropriados.

      Valores de P(Z ≤ z) - Probabilidade Acumulada à Esquerda

      Como usar a Tabela Z

      Leitura básica

      Exemplo: Encontrar P(Z ≤ 1.45)

      1. Localize a linha 1.4

      2. Localize a coluna 0.05

      3. Interseção: 0.9265

      4. P(Z ≤ 1.45) = 0.9265

      Cálculos importantes

      P(Z > z): 1 - P(Z ≤ z)

      P(a < Z < b): P(Z ≤ b) - P(Z ≤ a)

      P(Z < -z): 1 - P(Z ≤ z)

      P(-z < Z < z): 2×P(Z ≤ z) - 1

      Valores críticos

      90% Confiança: z = ±1.645

      95% Confiança: z = ±1.96

      99% Confiança: z = ±2.576

      99.9% Confiança: z = ±3.291

      Uso prático com Python

      from scipy import stats
import numpy as np

print("USO PRÁTICO DA DISTRIBUIÇÃO NORMAL")

# EXEMPLO 1: ALTURA DOS ESTUDANTES
print("\nEXEMPLO 1: ALTURA DOS ESTUDANTES")
print("μ = 170cm, σ = 8cm")
print("Qual P(165 ≤ X ≤ 175)?")

# CONVERSÃO PARA Z
z1 = (165 - 170) / 8  # -0.625
z2 = (175 - 170) / 8  # 0.625

# USANDO SCIPY
p_z2 = stats.norm.cdf(z2)   # P(Z ≤ 0.625)
p_z1 = stats.norm.cdf(z1)   # P(Z ≤ -0.625)
probabilidade = p_z2 - p_z1

print(f"Z₁ = {z1:.3f}, Z₂ = {z2:.3f}")
print(f"P(Z ≤ {z2:.3f}) = {p_z2:.4f}")
print(f"P(Z ≤ {z1:.3f}) = {p_z1:.4f}")
print(f"P({z1:.3f} ≤ Z ≤ {z2:.3f}) = {p_z2:.4f} - {p_z1:.4f} = {probabilidade:.4f}")
print(f"Probabilidade: {probabilidade:.2%}")

# EXEMPLO 2: VALORES CRÍTICOS
print("\nEXEMPLO 2: VALORES Z CRÍTICOS")
niveis_confianca = [0.90, 0.95, 0.99, 0.999]

for confianca in niveis_confianca:
    alpha = (1 - confianca) / 2
    z_critico = stats.norm.ppf(1 - alpha)
    print(f"{confianca:.1%} Confiança: z = ±{z_critico:.3f}")

# EXEMPLO 3: CÁLCULOS DIVERSOS
print("\nEXEMPLO 3: CÁLCULOS DIVERSOS")
z = 1.5

p_menor = stats.norm.cdf(z)                    # P(Z ≤ 1.5)
p_maior = 1 - p_menor                         # P(Z > 1.5)
p_entre = stats.norm.cdf(z) - stats.norm.cdf(-z)  # P(-1.5 < Z < 1.5)

print(f"P(Z ≤ {z}) = {p_menor:.4f}")
print(f"P(Z > {z}) = {p_maior:.4f}")
print(f"P(-{z} < Z < {z}) = {p_entre:.4f}")

# EXEMPLO 4: ENCONTRAR Z PARA PROBABILIDADE
print("\nEXEMPLO 4: ENCONTRAR Z PARA PROBABILIDADE")
probabilidades = [0.9, 0.95, 0.975, 0.99, 0.995]

for p in probabilidades:
    z_valor = stats.norm.ppf(p)
    print(f"P(Z ≤ z) = {p:.3f} → z = {z_valor:.3f}")

      Regressão linear simples

      Modelo: ŷ=b0+b1x

      Estimadores

      b1 = ∑ (x−x̄) (y−ȳ) ∑ (x−x̄)2
      b0 = ȳ − b1 x̄

      Correlação e R²

      r = cov(X,Y) s_X⋅s_Y

      Em regressão linear simples, R² = r².
