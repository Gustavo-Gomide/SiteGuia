Enumeração

O que é Enumeração?

Enumeração

Interação ativa com sistemas para descobrir serviços, versões e usuários.

Enumeração é o processo de interagir ativamente com sistemas para descobrir serviços, versões, usuários, shares, configurações e outras informações que não estão em fontes públicas. É a transição entre o reconhecimento passivo e a exploração ativa.

A diferença entre reconhecimento e enumeração é contato direto: enumeração envia pacotes para os sistemas alvo e coleta suas respostas. Pode ser detectada por IDS/IPS e firewall.

Port Scanning com Nmap

# Scan completo (todos os 65535 ports + detecção de versão):
nmap -sV -sC -p- --open -T4 10.10.10.10 -oA scan_completo

# Scan rápido — top 1000 ports:
nmap -sV --open -T4 10.10.10.10

# UDP scan (mais lento, frequentemente esquecido):
sudo nmap -sU --top-ports 50 10.10.10.10

# Script NSE para vulnerabilidades:
nmap --script vuln 10.10.10.10

# Descoberta de hosts na rede:
nmap -sn 192.168.1.0/24

Parâmetros Nmap essenciais: -sV detecta versões de serviço, -sC roda scripts padrão, -p- todos os 65535 ports, -oA salva em todos os formatos (texto, XML, grepable), -T4 velocidade agressiva. Use -T2 para ser mais furtivo.

Enumeração por Protocolo

SMB (porta 445)

File shares do Windows. Enum4linux ou smbclient para listar shares, usuários, grupos, políticas de senha. EternalBlue (MS17-010) explorava SMBv1 — verifique sempre se está ativo.

enum4linux -a 10.10.10.10
smbclient -L //10.10.10.10 -N
crackmapexec smb 10.10.10.10 --shares

LDAP (porta 389/636)

Active Directory expõe enorme quantidade de informações via LDAP, às vezes sem autenticação (null session/anonymous). Usuários, grupos, GPOs, trust, SPNs para Kerberoasting.

ldapsearch -x -H ldap://10.10.10.10 \
    -b "dc=empresa,dc=com,dc=br"
# Com bloodhound-python:
bloodhound-python -d empresa.com.br \
    -u user -p pass -c all

FTP (porta 21)

Verificar login anônimo, listar arquivos acessíveis. FTP envia credenciais em texto claro — sniffável em rede não criptografada. Buscar arquivos de configuração e backups.

ftp 10.10.10.10
# Tente: usuario anonymous
#         senha: qualquer@email.com
nmap -sV --script=ftp-anon 10.10.10.10

SSH (porta 22)

Identificar versão (banner grabbing) para CVEs específicos. Verificar métodos de autenticação aceitos. Enumerar usuários via timing attack (versões antigas do OpenSSH).

ssh 10.10.10.10 -v 2>&1 | grep "Server version"
nmap -p 22 --script ssh-auth-methods 10.10.10.10

SNMP (porta 161 UDP)

Gerenciamento de rede. Community string "public" como padrão em dispositivos antigos expõe informações do sistema, interfaces de rede, processos, rotas de forma completamente não autenticada.

snmpwalk -v2c -c public 10.10.10.10
onesixtyone 10.10.10.10 public private
snmp-check 10.10.10.10 -c public

HTTP/HTTPS (porta 80/443)

Identificar tecnologias (Wappalyzer, whatweb), diretórios ocultos (gobuster/feroxbuster), arquivos sensíveis (robots.txt, .git, .env), APIs, virtual hosts.

whatweb http://10.10.10.10
gobuster dir -u http://10.10.10.10 \
    -w /usr/share/wordlists/dirbuster/med.txt
nikto -h http://10.10.10.10

Active Directory Enumeration

Active Directory é o alvo mais comum em pentests corporativos. Com qualquer credencial de domínio (até usuário comum), é possível fazer enumeração extensa:

# BloodHound — mapeamento visual de attack paths no AD:
# Coletar dados:
bloodhound-python -d empresa.com.br -u usuario -p senha -c all

# PowerView (PowerShell) para enum de AD:
Get-NetUser | select samaccountname, description
Get-NetGroup "Domain Admins" | select member
Find-LocalAdminAccess   # hosts onde temos admin local
Get-NetComputer -Ping   # hosts ativos no domínio

# Kerberosating — enumerar service accounts com SPN:
GetUserSPNs.py empresa.com.br/usuario:senha -dc-ip 10.10.10.10

Enumeração detalhada é a diferença entre um pentest raso e um profundo. Cada porta aberta é uma oportunidade. Explore cada serviço encontrado antes de partir para exploração. Próxima fase: Exploração. Veja também Nmap em detalhes.

"Enumeração lenta é pentest amador. Mas enumeração sem documentação é tempo perdido."
