O que é Nmap?

Nmap — Network Mapper

Descoberta de hosts, portas abertas, serviços e OS detection em redes.

Nmap (Network Mapper) é a ferramenta de descoberta de rede e auditoria de segurança mais usada do mundo. Open source desde 1997, detecta hosts ativos, portas abertas, serviços, versões e sistema operacional. É indispensável em qualquer pentest ou auditoria de rede.

Tipos de Scan

Parâmetros Essenciais

# Scan completo — o mais usado em CTF e pentest:
sudo nmap -sV -sC -p- --open -T4 10.10.10.10 -oA scan

# Explicação de cada flag:
#   -sV   → detectar versão do serviço
#   -sC   → rodar scripts NSE padrão
#   -p-   → todas as 65535 portas (default: apenas top 1000)
#   --open → mostrar apenas portas abertas
#   -T4   → timing agressivo (T0=paranóico, T5=insano)
#   -oA   → salvar em todos os formatos (.nmap, .xml, .gnmap)

# Scan rápido (descoberta inicial):
nmap -sV --open -T4 10.10.10.10

# Só descoberta de hosts na rede:
nmap -sn 192.168.1.0/24

# UDP top 50 portas (mais lento):
sudo nmap -sU --top-ports 50 10.10.10.10

NSE — Nmap Scripting Engine

O Nmap tem mais de 600 scripts Lua para automação de tarefas de enumeração, detecção de vulnerabilidades e exploração básica.

Scripts por categoria

# Todos os scripts de vulnerabilidade:
nmap --script vuln 10.10.10.10

# Scripts de uma categoria:
nmap --script "smb-*" 10.10.10.10
nmap --script "http-*" 10.10.10.10

# Script específico:
nmap --script smb-vuln-ms17-010 10.10.10.10
nmap --script ssl-heartbleed 10.10.10.10

Scripts mais úteis

  • smb-vuln-ms17-010 — EternalBlue
  • smb-enum-shares — listar shares
  • http-title — título das páginas web
  • ftp-anon — login anônimo FTP
  • ssh-auth-methods — métodos auth SSH
  • ssl-heartbleed — Heartbleed
  • dns-zone-transfer — transferência de zona

Firewall Evasion

# Fragmentar pacotes (evitar IDS):
nmap -f 10.10.10.10

# Usar decoys (mascarar IP real):
nmap -D RND:10 10.10.10.10

# Spoofar MAC address:
nmap --spoof-mac Apple 10.10.10.10

# Timing lento (evitar detecção):
nmap -T1 10.10.10.10   # Sneaky
nmap -T0 10.10.10.10   # Paranoid

# Usar porta de origem específica (bypassar firewall):
nmap --source-port 53 10.10.10.10
