O que é Burp Suite?

Burp Suite

Proxy HTTP que intercepta, modifica e re-envia requisições para descobrir vulnerabilidades.

Burp Suite é o suite de ferramentas para pentest de aplicações web mais usado do mundo, desenvolvido pela PortSwigger. Opera como proxy HTTP interceptor entre seu browser e a aplicação — permitindo capturar, modificar e re-enviar requisições para descobrir vulnerabilidades.

Community (Gratuito)

Proxy, Repeater, Decoder, Comparer, Sequencer. Intruder limitado (velocidade limitada). Suficiente para a maioria dos CTFs e aprendizado.

Professional (~$449/ano)

Adiciona scanner automático de vulnerabilidades, Intruder sem limitação de velocidade, Collaborator (para SSRF/XXE OOB), extensões avançadas. Padrão em pentests profissionais.

# 2. Configurar FoxyProxy no Firefox (extensão recomendada):
#    Proxy: HTTP 127.0.0.1 porta 8080

# 3. Instalar certificado CA do Burp:
#    Com proxy ativo, acessar http://burpsuite
#    Baixar CA certificate
#    Firefox: Preferências → Certificados → Importar

# 4. Para HTTPS funcionar corretamente:
#    O Burp gera certificado dinâmico assinado pela CA dele
#    Com o CA instalado, o browser confia nesses certs

Ferramentas Principais

Proxy / Intercept

Captura toda requisição entre browser e servidor. Com Intercept ON, você pode modificar a requisição antes de enviar. Essencial para testar validações do lado do cliente, alterar parâmetros, tokens e cookies.

Repeater

Enviar uma requisição capturada repetidamente com modificações. Ideal para testar injeções manualmente — modifica um parâmetro, envia, analisa resposta, repete. Workflow central do pentest web manual.

Intruder

Automação de requisições com payloads variáveis — brute force, fuzzing, enumeração. Modos: Sniper (1 posição), Battering Ram (mesma string em N posições), Pitchfork (listas sync). Community tem throttling.

Scanner (Pro)

Scan ativo automático de vulnerabilidades web (XSS, SQLi, SSRF, etc). Crawla toda a aplicação e testa vulnerabilidades. Gera relatórios. Não substitui análise manual mas encontra muito automaticamente.

Decoder

Encode/decode rápido de valores: Base64, URL encoding, HTML entities, hex, SHA, MD5. Essencial para manipular tokens, cookies e dados serializados.

Extensions (BApp Store)

JSON Beautifier, JWT Editor (manipular JWTs), ActiveScan++, Logger++, Turbo Intruder (Intruder rápido grátis), Param Miner (descobrir parâmetros ocultos). Estendem o Burp com funcionalidades da comunidade.

Workflow de Pentest Web com Burp

# Exemplo: testar SQLi no Repeater

# 1. Capturar requisição de login no Proxy
# 2. Enviar para Repeater (Ctrl+R)
# 3. No campo username, testar:
#    ' (aspas simples — erro SQL revela SQLi)
#    ' OR '1'='1'--   (bypass de login)
#    admin'--          (comentar resto da query)

# Exemplo: brute force no Intruder
# 1. Capturar requisição de login
# 2. Enviar para Intruder (Ctrl+I)
# 3. Marcar §password§ como posição
# 4. Atacar: Sniper mode
# 5. Payload: lista de senhas comuns
# 6. Filtrar por Status code != 401 ou resposta diferente

# Exemplo: manipulação de JWT com extensão JWT Editor
# 1. Capturar requisição autenticada
# 2. Inspector mostra JWT automaticamente
# 3. Editar claims (ex: "role": "admin")
# 4. Assinar com chave fraca ou None algorithm
