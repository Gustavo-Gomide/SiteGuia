O que é um WAF?

WAF — Web Application Firewall

Filtra tráfego HTTP/S contra ataques web — SQLi, XSS, CSRF e bots.

WAF inspeciona tráfego HTTP/S em busca de ataques como SQLi, XSS, CSRF, path traversal e bots. Diferente de firewalls de rede (Layer 3/4), o WAF opera na Layer 7 entendendo a semântica das requisições web.

ModSecurity + OWASP CRS

# ModSecurity no Nginx (nginx.conf):
modsecurity on;
modsecurity_rules_file /etc/nginx/modsec/main.conf;

# main.conf:
Include /usr/share/modsecurity-crs/crs-setup.conf
Include /usr/share/modsecurity-crs/rules/*.conf

# OWASP CRS — paranoia levels (crs-setup.conf):
# PL1 = mínimo de falsos positivos (recomendado para início)
# PL2 = mais regras, alguns falsos positivos
# PL3 = agressivo
# PL4 = máximo, muitos falsos positivos

SecAction \
  "id:900000,\
  phase:1,\
  nolog,\
  pass,\
  t:none,\
  setvar:tx.paranoia_level=2"

# Modo detecção (log only, sem bloquear):
SecRuleEngine DetectionOnly

# Modo bloqueio:
SecRuleEngine On

Técnicas de Bypass (para Pentesters)

Encoding / Obfuscation

# URL encoding duplo:
%25%32%37 → %27 → '

# Unicode:
ʼ ou ᾿ podem funcionar como aspas

# Case mixing:
SeLeCt vs select

Comment Injection (SQLi)

# Comentários como separadores:
SELECT/**/username/**/FROM/**/users
/*!UNION*/ SELECT 1,2,3

# Whitespace alternatives:
SELECT%09FROM

HTTP Parameter Pollution

# Enviar parâmetro duplicado:
?id=1&id=1 UNION SELECT 1,2
# WAF avalia apenas o primeiro,
# app usa o segundo

Mitigação de Bypasses

  • Aumentar paranoia level do CRS
  • Habilitar normalização de encoding
  • Usar WAF com ML/comportamental
  • Combinar WAF + input validation na app
  • Auditar regras regularmente com testes

WAF não substitui desenvolvimento seguro. Um atacante determinado frequentemente encontra formas de bypass. Use WAF como camada adicional de defesa, não como única proteção contra vulnerabilidades de aplicação. Sempre combine com SAST/DAST e code review.

"WAF é a última linha de defesa, não a primeira. Construa aplicações seguras e use WAF como rede de segurança."
