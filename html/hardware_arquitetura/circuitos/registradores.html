Registradores de Hardware

Grupos de flip-flops para armazenar, deslocar e transferir dados em circuitos digitais

Um registrador de hardware é um grupo de flip-flops D que armazena N bits como uma unidade. Diferente dos registradores de CPU (abstração de ISA), registradores de hardware são circuitos físicos e o conceito é o alicerce tanto dos registradores de CPU quanto de buffers em periféricos, FIFOs e memória.

Banco de Registradores (Register File)

Estrutura

Um banco de registradores é um conjunto de N registradores paralelos com lógica de seleção (decodificadores + MUX). Permite múltiplas leituras e uma escrita simultânea.

  • x86-64: 16 registradores gerais visíveis + dezenas de físicos internos
  • ARM AArch64: 32 registradores gerais (X0–X30 + SP)
  • RISC-V RV64I: 32 registradores (x0–x31)

Implementação

  • N registradores de N bits (ex: 32 × 64 bits = 256 bytes no RISC-V 64)
  • 2+ portas de leitura: permite ler src1 e src2 simultaneamente
  • 1 porta de escrita: escreve resultado do WB no clock
  • Fabricado em SRAM multi-port — mais rápido que cache, menor que L1
