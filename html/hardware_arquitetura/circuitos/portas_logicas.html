Portas Lógicas

Os blocos fundamentais de todo circuito digital: AND, OR, NOT, NAND, NOR, XOR, XNOR

Uma porta lógica é um circuito eletrônico que implementa uma função booleana. Todas as portas são construídas a partir de transistores CMOS: um par PMOS (pull-up) + NMOS (pull-down) por inversor. NAND e NOR são universais: qualquer função lógica pode ser implementada usando apenas uma delas.

NOT (inversor): saída = ¬A. Com entrada simples: NOT(0)=1, NOT(1)=0. A porta XOR é útil para detecção de diferença e para construção de somadores.

Portas Universais — NAND e NOR

Implementando NOT, AND, OR apenas com NAND
NOT(A)    = NAND(A, A)
AND(A,B)  = NOT( NAND(A,B) )   = NAND( NAND(A,B), NAND(A,B) )
OR(A,B)   = NAND( NOT(A), NOT(B) ) = NAND( NAND(A,A), NAND(B,B) )
XOR(A,B)  = AND( OR(A,B), NAND(A,B) )  [com 4 NANDs]

# Por que isso importa?
# - FPGAs usam LUTs (Look-Up Tables) que implementam qualquer função
# - ASICs preferem empregam células padrão (standard cells) NAND/NOR
# - Minimiza área de silício e simplifica síntese lógica
