Hardware e Arquitetura

GPU — Processador Gráfico

Arquitetura paralela, VRAM, CUDA/OpenCL e diferenças fundamentais em relação à CPU

A GPU nasceu para acelerar renderização 3D, mas sua arquitetura massivamente paralela a tornou essencial para IA, simulações científicas e processamento de dados. Onde a CPU tem poucos núcleos poderosos e independentes, a GPU tem milhares de núcleos simples que trabalham em conjunto.

CPU vs. GPU

CPU

  • 4–96 núcleos complexos (latência baixa)
  • Execução out-of-order, branch prediction, cáches grandes
  • Otimizada para fluxo de controle serial
  • Clock: 3–5+ GHz
  • Latência de memória baixa (caches L1/L2/L3)
  • Ideal: tarefas sequenciais, lógica complexa, I/O

GPU

  • Milhares a dezenas de milhares de núcleos simples (throughput alto)
  • SIMT (Single Instruction Multiple Threads): todos os threads executam a mesma instrução
  • Otimizada para dados paralelos (matrizes, pixels, vetores)
  • VRAM dedicada de alta largura de banda (HBM: 3+ TB/s)
  • Ideal: IA/ML, shaders, simulações físicas, FFTs
