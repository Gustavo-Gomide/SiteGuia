Visão Geral dos Componentes

Todos os componentes de um PC moderno: função, interconexão e gargalos

Um computador moderno é um sistema de componentes que se comunicam por barramentos. Cada componente tem função específica e características de desempenho distintas. Identificar o gargalo correto é o primeiro passo para otimizar um sistema.

Latência Comparativa de Memória

Números que todo programador deveria saber
Registrador da CPU:     <1 ns
Cache L1:               ~1 ns
Cache L2:               ~4 ns
Cache L3:               ~10 ns
RAM (DDR5):             ~70 ns
SSD NVMe (PCIe 4):      ~100 μs
SSD SATA:               ~100 μs (seq)
HDD:                    ~5–10 ms
Leitura da rede local:  ~1 ms
Leitura na internet:    10–200 ms

Um acesso a RAM é ~70x mais lento que L1 cache. Um acesso a disco NVMe é ~1.000x mais lento que RAM. Estruturas de dados e padrões de acesso à memória importam muito.

Identif icar Gargalos

CPU-bound

Uso de CPU próximo de 100%, pouco I/O. Melhorar: algoritmos mais eficientes, mais núcleos, compilar com otimizações, usar SIMD.

Memory-bound

CPU esperando por dados da RAM. Melhorar: melhorar localidade de cache, reduzir working set, usar estruturas de dados cache-friendly.

I/O-bound

CPU ociosa esperando por disco ou rede. Melhorar: I/O assíncrono, buffer maior, SSD mais rápido, replicação de dados mais próximos.
