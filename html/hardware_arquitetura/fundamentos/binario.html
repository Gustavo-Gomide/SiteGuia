Binário e Operações Bit a Bit

Sistema de base 2, representação de dados e operadores bitwise

O sistema binário é a linguagem nativa dos computadores: transistores são chaves que representam 0 (desligado) ou 1 (ligado). Operações bit a bit aparecem em máscaras de rede, flags de status, criptografia, compressão, gráficos e otimizações de baixo nível.

Aplicações Práticas

C — técnicas comuns com bits
// Verificar se bit N está setado
int is_set(int val, int n) {
    return (val >> n) & 1;
}

// Setar bit N
int set_bit(int val, int n) {
    return val | (1 << n);
}

// Limpar bit N
int clear_bit(int val, int n) {
    return val & ~(1 << n);
}

// Alternar bit N
int toggle_bit(int val, int n) {
    return val ^ (1 << n);
}

// Verificar se é potência de 2 (truque clássico)
bool is_power_of_two(unsigned int n) {
    return n != 0 && (n & (n - 1)) == 0;
}

// Máscara de sub-rede IPv4
uint32_t mask = 0xFFFFFF00;  // /24 = 255.255.255.0
uint32_t ip   = 0xC0A80101;  // 192.168.1.1
uint32_t net  = ip & mask;   // 192.168.1.0

Flags com Bitmask

C — enum de flags
#define PERM_READ    (1 << 0)  // 0001
#define PERM_WRITE   (1 << 1)  // 0010
#define PERM_EXEC    (1 << 2)  // 0100
#define PERM_OWNER   (1 << 3)  // 1000

int perm = PERM_READ | PERM_WRITE;  // 0011

// Testar
if (perm & PERM_READ)  printf("pode ler\n");
if (perm & PERM_EXEC)  printf("pode executar\n");  // não entra

// Revogar
perm &= ~PERM_WRITE;  // 0001

Esse padrão é amplamente usado em kernels de SO (flags de página de memória, flags de processo), protocolos de rede (TCP flags), e drivers de hardware.
