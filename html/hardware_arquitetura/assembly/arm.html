Assembly ARM (AArch64)

RISC de 64 bits: instruções fixas de 32 bits, load/store, NEON e AAPCS64

O ARM AArch64 é a ISA RISC de 64 bits que domin a o mercado mobile (iOS, Android), servidores (AWS Graviton, Ampere Altra) e desktops (Apple Silicon M-series). Todas as instruções têm 32 bits de tamanho fixo; apenas LOAD e STORE acessam memória; a ALU só opera sobre registradores.

Instruções Fundamentais

Exemplo: Hello World Linux AArch64

GNU Assembler (GAS) AArch64 — syscall Linux
.section .data
msg: .ascii "Hello, ARM!\n"
len = . - msg

.section .text
.global _start
_start:
    // write(1, msg, len)
    mov x8, #64        // syscall: write
    mov x0, #1         // fd: stdout
    adr x1, msg        // endereço do buffer
    mov x2, #len       // tamanho
    svc #0             // chamada ao kernel

    // exit(0)
    mov x8, #93        // syscall: exit
    mov x0, #0         // exit code
    svc #0

// Build:
// as -o hello.o hello.s
// ld -o hello hello.o
// ./hello

Os números de syscall no Linux AArch64 diferem do x86-64. Use /usr/include/asm/unistd.h ou a tabela em syscall.tbl do kernel para a lista completa.
