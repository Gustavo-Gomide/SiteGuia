Instruções Assembly

Categorias de instruções x86-64: transferência, aritmética, lógica, desvio, pilha

As instruções x86-64 são agrupadas por função. Cada uma tem um opcode binário de tamanho variável (1–15 bytes). A sintaxe Intel é: MNEM dest, src. A sintaxe AT&T (GAS padrão) inverte a ordem e adiciona sufixos de tamanho (movq, addl).

Exemplos Práticos

NASM x86-64 — padrões comuns
; Zerar um registrador (XOR é menor e mais rápido que MOV rax, 0)
xor rax, rax

; Carregar endereço efetivo (LEA não acessa memória, apenas calcula)
lea rsi, [rel array]   ; RIP-relative: endereço de 'array'

; Loop simples (RCX como contador, JNZZ como condicional)
    mov rcx, 10
.loop:
    ; ... corpo do loop ...
    dec rcx
    jnz .loop

; Trocar dois registradores sem temporário
xchg rax, rbx
; Ou com XOR swap:
xor rax, rbx
xor rbx, rax
xor rax, rbx

; Testar se bit N está setado
test rax, (1 << 3)   ; bit 3
jz  bit_nao_setado
