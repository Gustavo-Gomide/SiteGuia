Modos de Endereçamento

Como operandos são especificados: imediato, registrador, memória e suas variações

O modo de endereçamento define como cada operando de uma instrução é obtido. Na arquitetura x86-64, a fórmula geral para um operando memória é: [base + index * scale + displacement], onde cada campo é opcional. ARM usa BaseReg + offset (imediato ou registrador) com pre/post-indexing.
