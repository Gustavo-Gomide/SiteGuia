pip — Instalador de Pacotes Python

pip

O gerenciador de pacotes padrão do Python — instala, atualiza e remove pacotes do PyPI e outras fontes

pip (Pip Installs Packages) é a ferramenta oficial para instalar pacotes Python do PyPI (Python Package Index — pypi.org), o repositório com mais de 500.000 projetos. pip é incluído automaticamente com o Python 3 (desde a versão 3.4).

Ambientes Virtuais — A Base do Ecossistema Python

O problema central do Python é o isolamento de dependências entre projetos. A solução padrão é o ambiente virtual (venv):

Fluxo de trabalho — venv + pip
# 1. Criar ambiente virtual (uma vez por projeto)
python -m venv .venv

# 2. Ativar (toda vez que abrir o terminal)
source .venv/bin/activate       # Linux/macOS
.venv\Scripts\activate          # Windows

# 3. Instalar dependências
pip install requests
pip install "django>=4.2,<5.0"
pip install -r requirements.txt

# 4. Salvar dependências
pip freeze > requirements.txt

# 5. Desativar
deactivate

Sempre adicione .venv/ ao seu .gitignore. Nunca commite o ambiente virtual; commite apenas o requirements.txt.

Comandos Essenciais

pip — referência rápida
# Instalar
pip install numpy                 # última versão
pip install numpy==1.24.0        # pinado
pip install "numpy>=1.20"        # mínimo
pip install -e .                 # editable install (desenvolvimento)
pip install -r requirements.txt  # do arquivo

# Listar e inspecionar
pip list
pip show requests
pip outdated

# Desinstalar
pip uninstall numpy

# Atualizar
pip install --upgrade numpy
pip install --upgrade pip        # atualizar o próprio pip

# Verificar segurança das dependências
pip audit                        # requer pip-audit

requirements.txt e Reprodutibilidade

Boas práticas com requirements

Uma prática recomendada é separar as dependências em dois arquivos:

  • requirements.in — dependências diretas sem versões fixas
  • requirements.txt — gerado por pip-compile com todas as versões pinadas (inclui transitivas)

Isso combina facilidade de atualização com reprodutibilidade total.
