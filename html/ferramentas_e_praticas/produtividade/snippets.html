Snippets

Modelos de código ativados por palavras-chave que eliminam a digitação repetitiva e padronizam padrões em toda a equipe

Snippets são modelos de código parametrizáveis que você ativa digitando um prefixo e pressionando Tab. São diferentes de templates de projeto — snippets atuam no nível de construções de código (funções, classes, blocos try-catch) dentro de um arquivo existente.

Snippets no VSCode

VSCode usa snippets no formato JSON com suporte a placeholders, tab stops e variáveis especiais. Para criar: Ctrl+Shift+P → Snippets: Configure User Snippets.

python.json — snippets Python customizados 
{
  // Snippet: função com docstring
  "Func with docstring": {
    "prefix": "defds",
    "body": [
      "def ${1:nome_funcao}(${2:params}) -> ${3:None}:",
      "    \"\"\"${4:Descrição da função}.",
      "",
      "    Args:",
      "        ${2:params}: ${5:Descrição}.",
      "",
      "    Returns:",
      "        ${3:None}: ${6:O que retorna}.",
      "    \"\"\"",
      "    ${0}"
    ],
    "description": "Função Python com docstring Google Style"
  },

  // Snippet: bloco try-except-finally
  "Try-Except-Finally": {
    "prefix": "tef",
    "body": [
      "try:",
      "    ${1:pass}",
      "except ${2:Exception} as ${3:e}:",
      "    ${4:raise}",
      "finally:",
      "    ${0:pass}"
    ],
    "description": "Try-except-finally com placeholders"
  },

  // Snippet: classe Python
  "Class": {
    "prefix": "cls",
    "body": [
      "class ${1:NomeDaClasse}${2:(${3:Base})}:",
      "    \"\"\"${4:Descrição da classe.}\"\"\"",
      "",
      "    def __init__(self, ${5:params}) -> None:",
      "        ${0:pass}"
    ],
    "description": "Classe Python com __init__"
  }
}

Boas Práticas

Quando criar snippets

  • Qualquer bloco que você digita mais de 3 vezes por semana
  • Padrões da equipe (docstrings, headers de arquivo, padrões de log)
  • Boilerplate de frameworks (controllers, migrações, tests)
  • Estruturas que você sempre esquece a sintaxe exata

Compartilhando snippets na equipe

Use o recurso de project-level snippets do VSCode: crie o arquivo .vscode/nome.code-snippets no repositório. Todos os membros da equipe terão os mesmos snippets automaticamente ao abrir o projeto.
