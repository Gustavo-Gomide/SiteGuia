Snippets — Código Reutilizável

Snippets

Modelos de código ativados por palavras-chave que eliminam a digitação repetitiva e padronizam padrões em toda a equipe

Snippets são modelos de código parametrizáveis que você ativa digitando um prefixo e pressionando Tab. São diferentes de templates de projeto — snippets atuam no nível de construções de código (funções, classes, blocos try-catch) dentro de um arquivo existente.

Snippets no VSCode

VSCode usa snippets no formato JSON com suporte a placeholders, tab stops e variáveis especiais. Para criar: Ctrl+Shift+P → Snippets: Configure User Snippets.

python.json — snippets Python customizados
{ // Snippet: função com docstring "Func with docstring": { "prefix": "defds", "body": [ "def ${1:nome_funcao}(${2:params}) -> ${3:None}:", " \"\"\"${4:Descrição da função}.", "", " Args:", " ${2:params}: ${5:Descrição}.", "", " Returns:", " ${3:None}: ${6:O que retorna}.", " \"\"\"", " ${0}" ], "description": "Função Python com docstring Google Style" }, // Snippet: bloco try-except-finally "Try-Except-Finally": { "prefix": "tef", "body": [ "try:", " ${1:pass}", "except ${2:Exception} as ${3:e}:", " ${4:raise}", "finally:", " ${0:pass}" ], "description": "Try-except-finally com placeholders" }, // Snippet: classe Python "Class": { "prefix": "cls", "body": [ "class ${1:NomeDaClasse}${2:(${3:Base})}:", " \"\"\"${4:Descrição da classe.}\"\"\"", "", " def __init__(self, ${5:params}) -> None:", " ${0:pass}" ], "description": "Classe Python com __init__" } }

Exemplo real: classe Carro

Agora o exemplo é um snippet de verdade: você digita carro, pressiona Tab e ele insere a classe, a instância e as três chamadas de uma vez.

Snippet pronto para expandir

python.json — prefixo carro
{ "Carro completo": { "prefix": "carro", "body": [ "class Carro:", " def entrar(self) -> None:", " print(\"Entrando no carro.\")", "", " def dirigir(self) -> None:", " print(\"Dirigindo o carro.\")", "", " def parar(self) -> None:", " print(\"Parando o carro.\")", "", "carro = Carro()", "carro.entrar()", "carro.dirigir()", "carro.parar()", "${0}" ], "description": "Expande uma classe Carro com instância e chamadas prontas" } }

Resultado ao expandir

main.py — código inserido pelo snippet
class Carro: def entrar(self) -> None: print("Entrando no carro.") def dirigir(self) -> None: print("Dirigindo o carro.") def parar(self) -> None: print("Parando o carro.") carro = Carro() carro.entrar() carro.dirigir() carro.parar()

Exemplos práticos dos parâmetros

Veja cada parâmetro funcionando em um snippet real que você pode usar hoje.

Tab stops com placeholder e escolhas

Snippet JSON
{ "Loop personalizado": { "prefix": "loop", "body": [ "for ${1:i} in ${2|range,enumerate|}(${3:items}):", " ${0:pass}" ], "description": "Loop com escolha entre range/enumerate" } }
Digitando: loop → Tab
for i in [range|enumerate](items): # ${1:i} está destacado # ${2} mostra menu de escolha pass

Variáveis especiais (arquivo, ano, clipboard)

Snippet JSON
{ "Header com contexto": { "prefix": "head", "body": [ "# $TM_FILENAME", "# Criado em $CURRENT_YEAR", "# Autor: ${1:seu_nome}", "# Conteúdo copiado: $CLIPBOARD" ], "description": "Header com variáveis automáticas" } }
Digitando: head → Tab
# meu_script.py # Criado em 2026 # Autor: seu_nome # Conteúdo copiado: def exemplo():

Transformação com regex

Snippet JSON
{ "Classe do nome do arquivo": { "prefix": "classnm", "body": [ "class ${TM_FILENAME/(.*)\\..*/\\u$1/}:", " \"\"\"${1:Descrição.}\"\"\"", " def __init__(self) -> None:", " ${0:pass}" ], "description": "Classe com nome derivado do arquivo, formatado em PascalCase" } }
Arquivo: usuario.py → Digitando: classnm → Tab
class Usuario: """Descrição.""" def __init__(self) -> None: pass

Múltiplos tab stops em sequência

Snippet JSON
{ "Função com tipos": { "prefix": "fn", "body": [ "def ${1:funcao}(${2:params}) -> ${3:str}:", " \"\"\"${4:Descrição da função.}\"\"\"", " return ${0:resultado}" ], "description": "Função com 5 pontos de edição sequencial" } }
Digitando: fn → Tab → Tab → Tab → Tab → Tab
def funcao(params) -> str: """Descrição da função.""" return resultado # cada Tab salta para ${1}, ${2}, ${3}, ${4}, ${0}

Boas Práticas

Quando criar snippets

  • Qualquer bloco que você digita mais de 3 vezes por semana
  • Padrões da equipe (docstrings, headers de arquivo, padrões de log)
  • Boilerplate de frameworks (controllers, migrações, tests)
  • Estruturas que você sempre esquece a sintaxe exata

Compartilhando snippets na equipe

Use o recurso de project-level snippets do VSCode: crie o arquivo .vscode/nome.code-snippets no repositório. Todos os membros da equipe terão os mesmos snippets automaticamente ao abrir o projeto.
