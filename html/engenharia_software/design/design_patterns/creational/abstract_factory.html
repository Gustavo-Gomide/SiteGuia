Fábricas de famílias de objetos compatíveis

Abstract Factory

Criacional — GoF

Abstract Factory fornece uma interface para criar famílias de objetos relacionados sem especificar suas classes concretas. Garante que os objetos criados sejam compatíveis entre si — por exemplo, todos os componentes de UI para o mesmo tema ou plataforma.

Fábricas concretas por família

Fábrica Botão Checkbox Input
UIFactoryLight LightButton LightCheckbox LightInput
UIFactoryDark DarkButton DarkCheckbox DarkInput
UIFactoryMobile MobileButton MobileCheckbox MobileInput

Exemplo — temas de UI

interface Botao   { renderizar(): string; }
interface Tooltip { renderizar(): string; }

interface UIFactory {
  criarBotao(): Botao;
  criarTooltip(): Tooltip;
}

class LightBotao implements Botao {
  renderizar() { return '<button class="btn-light">'; }
}
class LightTooltip implements Tooltip {
  renderizar() { return '<div class="tooltip-light">'; }
}

class LightUIFactory implements UIFactory {
  criarBotao()   { return new LightBotao(); }
  criarTooltip() { return new LightTooltip(); }
}

// DarkUIFactory seguiria a mesma estrutura

// Cliente — não sabe qual fábrica está usando
function renderizarApp(factory: UIFactory): void {
  const botao   = factory.criarBotao();
  const tooltip = factory.criarTooltip();
  console.log(botao.renderizar(), tooltip.renderizar());
}

// Troca de tema: só mudar a fábrica
renderizarApp(new LightUIFactory());

Quando usar

Use quando

Há famílias de objetos compatíveis que devem ser usadas juntas. Temas, plataformas, providers de infraestrutura.

Consistência garantida

A fábrica assegura que botões, tooltips e inputs da mesma família sejam compatíveis entre si.

OCP puro

Adicionar nova família (ex: HighContrastUIFactory) não modifica o código cliente nem as fábricas existentes.

Evite quando

Apenas um produto existe ou não há famílias. Factory Method ou inversão de dependência simples resolvem.

Abstract Factory é uma extensão do Factory Method. Use quando os produtos precisam ser compatíveis entre si por família.

"A fábrica abstrata garante que os objetos criados formem um conjunto coerente — sem misturar componentes de famílias diferentes."
