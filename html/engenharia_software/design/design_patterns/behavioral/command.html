Operações como objetos — enfileirar, desfazer, refazer

Command

Comportamental — GoF

Command encapsula uma operação como objeto com método execute(). Isso permite tratar ações como dados: enfileirar, registrar, desfazer (undo), refazer (redo) e transferir entre partes do sistema.

Partes do padrão

Parte Função Exemplo
Command (interface) Define execute() e opcionalmente undo(). interface Command
ConcreteCommand Implementa a operação e guarda estado para undo. SaveCommand, DeleteLineCommand
Invoker Executa commands e mantém histórico (undo stack). Toolbar, Keyboard shortcut handler
Receiver Objeto que efetivamente executa o trabalho. Editor, Repository

Exemplo — undo/redo em editor

interface Command {
  execute(): void;
  undo(): void;
}

class EscreverTextoCommand implements Command {
  private textoAnterior: string;

  constructor(private editor: Editor, private texto: string) {}

  execute(): void {
    this.textoAnterior = this.editor.conteudo;
    this.editor.conteudo += this.texto;
  }

  undo(): void {
    this.editor.conteudo = this.textoAnterior;
  }
}

class HistoricoEditor {
  private stack: Command[] = [];

  executar(cmd: Command): void {
    cmd.execute();
    this.stack.push(cmd);
  }

  desfazer(): void {
    const cmd = this.stack.pop();
    cmd?.undo();
  }
}

// Uso
const historico = new HistoricoEditor();
historico.executar(new EscreverTextoCommand(editor, 'Olá'));
historico.executar(new EscreverTextoCommand(editor, ' mundo'));
historico.desfazer(); // editor.conteudo = 'Olá'

Quando usar

Undo/Redo

Editores de texto, CAD, planilhas. A operação precisa ser reversivel e o estado anterior deve ser guardado.

Filas e agendamento

Jobs, task queues, workflows assincronos. O command é serializado e executado depois ou em outro processo.

Desacoplamento

Quem dispara (botão, atalho, API) não sabe quem executa. O Invoker só chama execute().

Evite quando

A operação não precisa ser tratada como dado. Uma chamada de função direta é mais simples e legível.

Command é amplamente usado em CQRS (Command Query Responsibility Segregation). Veja Chain of Responsibility para pipelines de execução.

"Command transforma verbo em substantivo — a ação vira um objeto que pode ser guardado, transferido e revertido."
