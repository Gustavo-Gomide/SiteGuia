Snapshot do estado para desfazer sem violar encapsulamento

Memento

Comportamental — GoF

Memento captura o estado interno de um objeto em um snapshot (Memento) sem expor seus detalhes. Um Caretaker guarda esses snapshots e os devolve ao Originator para restaurar estados anteriores — sem violar o encapsulamento.

Três participantes

Participante Função Analogia
Originator Cria e restaura Mementos do próprio estado. Editor de texto
Memento Snapshot imutável do estado do Originator. Arquivo de backup
Caretaker Armazena Mementos sem inspecioná-los. Pilha de undo, histórico de versões

Exemplo — undo em editor

// MEMENTO — snapshot imutável
class EditorMemento {
  constructor(readonly conteudo: string, readonly posicaoCursor: number) {}
}

// ORIGINATOR
class Editor {
  private conteudo = '';
  private cursor = 0;

  escrever(texto: string): void {
    this.conteudo += texto;
    this.cursor = this.conteudo.length;
  }

  salvar(): EditorMemento {
    return new EditorMemento(this.conteudo, this.cursor);
  }

  restaurar(m: EditorMemento): void {
    this.conteudo = m.conteudo;
    this.cursor = m.posicaoCursor;
  }

  imprimir(): void { console.log(this.conteudo); }
}

// CARETAKER
class Historico {
  private pilha: EditorMemento[] = [];

  salvar(editor: Editor): void { this.pilha.push(editor.salvar()); }

  desfazer(editor: Editor): void {
    const estado = this.pilha.pop();
    if (estado) editor.restaurar(estado);
  }
}

// Uso
const editor = new Editor();
const historico = new Historico();

editor.escrever('Olá');
historico.salvar(editor);
editor.escrever(' mundo');
editor.imprimir(); // 'Olá mundo'

historico.desfazer(editor);
editor.imprimir(); // 'Olá'

Quando usar

Use quando

Precisa de undo/redo. Snapshotting de estado para rollback transacional. Histórico de versões de documento.

Encapsulamento seguro

O Originator controla o que vai no Memento. O Caretaker não vê os detalhes internos do estado.

Custo de memória

Cada snapshot consome memória. Para estados grandes, considere salvar apenas diffs (delta encoding).

vs Command

Command implementa undo executando a operação inversa. Memento guarda e restaura estado direto. Escolha por custo.

Memento e Command são complementares para undo/redo complexos: Command para operações invertíveis, Memento para snapshots completos.

"Memento externaliza o estado sem expor a estrutura interna — quem guarda não precisa entender o que guardou."
