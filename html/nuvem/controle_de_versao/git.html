Git

Controle de Versao Distribuido

Historico imutavel de mudancas com colaboracao por branches e merge.

Git organiza a evolucao do software com rastreabilidade de cada alteracao.

Fundamentos

Commit

Snapshot versionado de um estado do projeto.

Branch

Linha de desenvolvimento paralela para novas mudancas.

Merge

Integracao de historicos para consolidar trabalho colaborativo.

  • LOELIGER, Jon; MCCULLOUGH, Matthew. Version Control with Git. O'Reilly, 2012.
  • TORVALDS, Linus; HAMANO, Junio. Git Documentation.
