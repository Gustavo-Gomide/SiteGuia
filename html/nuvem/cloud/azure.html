Azure

Microsoft Azure

A nuvem para ambientes corporativos e arquiteturas híbridas.

Azure: introducao

O Azure é uma das maiores nuvens do mercado e tem forte adoção em empresas que já usam Windows Server, Active Directory e Microsoft 365.

O foco histórico da plataforma em integração corporativa torna migração e governança mais diretas para muitos cenários enterprise.

Resource Groups

Todo recurso (VM, rede, banco, disco) pertence a um grupo lógico. Isso facilita gestão de custo, ciclo de vida e limpeza completa do ambiente.

Region Pairs

Regiões pares ajudam estratégias de recuperação de desastre: replicação e recuperação priorizam o par geográfico designado.

Maquinas Virtuais (VMs)

As VMs do Azure seguem famílias por perfil de carga: burstable, uso geral, computação e memória.

Series comuns

  • B: econômica para dev e cargas variáveis.
  • D: equilíbrio entre CPU e RAM.
  • F: otimizada para CPU.
  • E: otimizada para memória.

Alta disponibilidade

Availability Set distribui instâncias em domínios de falha no mesmo datacenter.

Availability Zones distribui entre datacenters diferentes na mesma região, com SLA maior.

VM Scale Sets (VMSS)

Permite auto scaling e auto healing em grupos de VMs idênticas com gerenciamento centralizado.

Armazenamento

Servicos da Storage Account

  • Blob: armazenamento de objetos.
  • Files: compartilhamento SMB/NFS.
  • Queues: mensageria simples.
  • Tables: NoSQL chave-valor.

Camadas e redundancia

Hot, Cool e Archive equilibram custo e latência.

LRS, ZRS e GRS definem o nível de replicação local, zonal e geográfica.

Azure Functions

Azure Functions executa código orientado a eventos sem gerenciamento direto de servidor.

O diferencial é o uso de triggers e bindings para reduzir boilerplate de integração.

Fluxo tipico

Evento de entrada (HTTP, Blob, fila) aciona a função, que processa dados e envia para um destino (banco, fila, storage).

Planos de hospedagem

  • Consumption: escala a partir de zero com cobrança por execução.
  • Premium: menor cold start e recursos avançados.
  • App Service Plan: capacidade dedicada.

Durable Functions

Suporta orquestrações stateful e retomada de fluxos longos após reinício.
