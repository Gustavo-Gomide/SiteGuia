Sistemas Operacionais

Introdução ao Windows

Windows é o sistema operacional da Microsoft com maior fatia de mercado em desktops (~72%). Baseado no kernel NT desde 2001 com o Windows XP, cada versão traz novas APIs, melhorias de segurança e suporte a hardware mais moderno.

Ferramentas Essenciais

Task Manager (Ctrl+Shift+Esc)

Monitorar processos, CPU, memória, rede, disco e startup apps.

Event Viewer (eventvwr.msc)

Logs do sistema, aplicações e segurança com IDs de eventos.

Device Manager (devmgmt.msc)

Gerenciar drivers de hardware, identificar conflitos.

Services (services.msc)

Iniciar, parar, configurar serviços do sistema e de terceiros.

Registry Editor (regedit)

Editar HKLM e HKCU; fazer backup antes de qualquer mudança.

PowerShell

Automação, administração remota, gerenciamento de AD e Azure AD.

Estrutura de Discos e Sistema de Arquivos

Estrutura de Diretórios Windows
C:\ (volume do sistema)
├── Windows\        - arquivos do SO
├── Program Files\  - apps 64-bit
├── Program Files (x86)\ - apps 32-bit  
├── Users\          - perfis de usuário
│   ├── Public\
│   └── USERNAME\
│       ├── Desktop\, Documents\, Downloads\
│       ├── AppData\Local\    - dados locais
│       ├── AppData\Roaming\  - dados sincronizados
│       └── AppData\LocalLow\ - baixo nível de integridade
└── ProgramData\    - dados de apps para todos os usuários

# Sistema de arquivos padrão: NTFS
# - journaling, ACLs, compressão, criptografia (EFS)
# - suporta arquivos > 4GB (diferente do FAT32)
# - permissões granulares por arquivo e pasta
