Sistemas Operacionais

PowerShell

PowerShell é um shell de linha de comando e linguagem de script baseada em .NET. Diferente do bash que trabalha com texto, o PowerShell usa objetos .NET no pipeline — cada cmdlet recebe e emite objetos com propriedades tipadas. Disponível no Windows (PowerShell 5.1 embutido) e multiplataforma (PowerShell 7+ para Linux/macOS).

Pipeline com Objetos

PowerShell — Pipeline com Objetos
# Diferença fundamental: objetos, não texto
Get-Process | Where-Object {$_.CPU -gt 10} | Sort-Object CPU -Descending | Select-Object -First 5 Name, CPU, WorkingSet

# Trabalhar com serviços
Get-Service | Where-Object {$_.Status -eq 'Running'} | Measure-Object

# Processar arquivos
Get-ChildItem C:\Logs -Filter "*.log" -Recurse |
  Where-Object {$_.LastWriteTime -lt (Get-Date).AddDays(-30)} |
  Remove-Item -WhatIf

# Consultar informações do sistema
Get-CimInstance -ClassName Win32_OperatingSystem |
  Select-Object Caption, Version, BuildNumber, TotalVisibleMemorySize, FreePhysicalMemory

# Gerenciar usuários locais
Get-LocalUser | Where-Object Enabled -eq $true
New-LocalUser -Name "deploy" -Password (Read-Host -AsSecureString) -FullName "Deploy User"

# Exportar para CSV
Get-Process | Select-Object Name, ID, CPU | Export-Csv -Path C:\processos.csv -NoTypeInformation

Scripts e Automação

PowerShell — Scripts e Automação
# Definir política de execução (necessário para scripts)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Estruturas de controle
$services = Get-Service | Where-Object Status -eq 'Stopped'
foreach ($svc in $services) {
    Write-Host "Serviço parado: $($svc.Name)"
}

# Funções com parâmetros tipados
function Get-DiskUsage {
    param(
        [string]$Path = "C:\",
        [switch]$IncludeHidden
    )
    $params = @{ Path = $Path; Recurse = $true }
    if ($IncludeHidden) { $params['Force'] = $true }
    (Get-ChildItem @params | Measure-Object -Property Length -Sum).Sum / 1GB
}

# Try/Catch
try {
    Get-Item "C:\arquvio_inexistente.txt" -ErrorAction Stop
} catch [System.IO.FileNotFoundException] {
    Write-Error "Arquivo não encontrado: $_"
} finally {
    Write-Host "Execução concluída"
}

# Módulos
Import-Module ActiveDirectory
Get-ADUser -Filter {Enabled -eq $true} -Properties LastLogonDate |
  Sort-Object LastLogonDate | Select-Object -Last 10

PowerShell Remoting

PowerShell — Remoting
# Habilitar PowerShell Remoting (executar como admin)
Enable-PSRemoting -Force

# Sessão remota interativa
Enter-PSSession -ComputerName SERVIDOR01 -Credential domain\admin

# Executar comando em múltiplos servidores
$servidores = "SRV01", "SRV02", "SRV03"
Invoke-Command -ComputerName $servidores -ScriptBlock {
    Get-Service -Name "wuauserv"
}

# Sessão persistente
$session = New-PSSession -ComputerName SERVIDOR01
Invoke-Command -Session $session -ScriptBlock { Get-Process }
Remove-PSSession $session
