Administração do Windows

Administrar o Windows envolve gerenciar usuários, grupos, políticas de segurança, monitorar desempenho e automatizar tarefas. As ferramentas MMC (Microsoft Management Console) consolidam snaps-in especializados, enquanto o PowerShell permite automação em escala.

Gerenciamento de Usuários Locais

PowerShell — Usuários Locais
# Listar usuários locais
Get-LocalUser | Select-Object Name, Enabled, LastLogon, PasswordLastSet

# Criar novo usuário
$senha = ConvertTo-SecureString "Senha@2025" -AsPlainText -Force
New-LocalUser -Name "suporte" -Password $senha -FullName "Suporte TI" -Description "Conta de suporte"

# Adicionar ao grupo Administradores
Add-LocalGroupMember -Group "Administrators" -Member "suporte"

# Desabilitar conta sem deletar
Disable-LocalUser -Name "usuario_antigo"

# Verificar membros de um grupo
Get-LocalGroupMember -Group "Administrators"

# Política de senhas (via secedit ou GPO)
net accounts /minpwlen:12 /maxpwage:90 /minpwage:1 /uniquepw:10

Tarefas Agendadas

PowerShell — Tarefas Agendadas
# Criar tarefa agendada simples
$action  = New-ScheduledTaskAction -Execute "powershell.exe" -Argument "-File C:\Scripts\backup.ps1"
$trigger = New-ScheduledTaskTrigger -Daily -At "02:00AM"
$settings = New-ScheduledTaskSettingsSet -ExecutionTimeLimit (New-TimeSpan -Hours 1)
Register-ScheduledTask -TaskName "BackupDiario" -Action $action -Trigger $trigger -Settings $settings -RunLevel Highest

# Listar todas as tarefas
Get-ScheduledTask | Where-Object State -ne Disabled | Select-Object TaskName, State, TaskPath

# Executar tarefa manualmente
Start-ScheduledTask -TaskName "BackupDiario"

# Ver último resultado de execução
Get-ScheduledTaskInfo -TaskName "BackupDiario" | Select-Object LastRunTime, LastTaskResult, NextRunTime
# LastTaskResult: 0 = sucesso, 1 = processo ainda rodando, outros = erro

# Via schtasks.exe (compatibilidade)
schtasks /create /tn "BackupDiario" /tr "powershell.exe -File C:\Scripts\backup.ps1" /sc daily /st 02:00

Monitoramento de Desempenho

Task Manager (Ctrl+Shift+Esc)

Visão rápida de CPU, RAM, disco e rede por processo; aba Performance mostra histórico e heatmap; aba Startup gerencia apps de inicialização.

Resource Monitor (resmon)

Correlaciona I/O de disco com processos, mostra conexões de rede ativas por processo e uso de RAM detalhado (Working Set, Commit).

Performance Monitor (perfmon)

Adicionar contadores para monitoramento granular: Processor\% Processor Time, Memory\Available MBytes, PhysicalDisk\Disk Reads/sec.

Windows Admin Center

Interface web moderna para gerenciar servidores remotamente sem RDP; inclui PowerShell, Event Viewer, Storage e Performance in-browser.

PowerShell Active Directory

"Um administrador Windows que não usa PowerShell está trabalhando o dobro do necessário."
