Sistemas Operacionais

Registro do Windows

O Registro é o banco de dados hierárquico centralizado onde o Windows armazena configurações do sistema, de aplicações e preferências de usuário. Substituiu os arquivos .INI do Windows 3.x. Editar incorretamente pode tornar o sistema não inicializável — sempre exporte uma chave antes de modificá-la.

Hives Principais

Hive Abreviação Conteúdo
HKEY_LOCAL_MACHINEHKLMConfigurações do sistema: hardware, drivers, serviços, software instalado para todos os usuários
HKEY_CURRENT_USERHKCUConfigurações do usuário logado: preferências, apps, variáveis de ambiente do usuário
HKEY_USERSHKUTodos os perfis de usuário carregados; HKCU é um atalho para o subkey do usuário atual
HKEY_CLASSES_ROOTHKCRAssociações de extensão de arquivo e informações de COM; fusão de HKLM\Software\Classes e HKCU\Software\Classes
HKEY_CURRENT_CONFIGHKCCConfiguração de hardware atual (perfil de hardware); mapeado de HKLM\SYSTEM\CurrentControlSet\Hardware Profiles\Current

Tipos de Valores

Tipo Código Descrição
REG_SZ1String de texto simples (terminada em nulo)
REG_EXPAND_SZ2String com variáveis de ambiente expansíveis (%SystemRoot%)
REG_BINARY3Dados binários brutos
REG_DWORD4Inteiro 32-bit sem sinal (0–4294967295)
REG_QWORD11Inteiro 64-bit
REG_MULTI_SZ7Múltiplas strings separadas por nulo
REG_LINK6Link simbólico para outra chave

Chaves Importantes

Registro — Chaves de Referência
# Startup de programas (persistência):
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\RunOnce

# Serviços do Windows:
HKLM\SYSTEM\CurrentControlSet\Services\NOME_DO_SERVIÇO
  Start: 0=Boot, 1=System, 2=Automatic, 3=Manual, 4=Disabled
  Type: 1=Kernel driver, 2=File system driver, 16=Win32 own process, 32=Win32 shared

# Associação de extensão de arquivo:
HKCR\.pdf = AcroExch.Document
HKCR\AcroExch.Document\shell\open\command = "C:\...\AcroRd32.exe" "%1"

# Informações do SO:
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion
  ProductName, CurrentVersion, CurrentBuild, ReleaseId

# Configurações de rede:
HKLM\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters
  Hostname, Domain, SearchList, NameServer

# Variáveis de ambiente do sistema:
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Environment

Scripts PowerShell para Registro

PowerShell — Gerenciamento do Registro
# Ler valor
Get-ItemProperty -Path "HKLM:\SOFTWARE\Microsoft\Windows NT\CurrentVersion" -Name ProductName

# Criar ou modificar valor
Set-ItemProperty -Path "HKCU:\Software\MyApp" -Name "ContagemAcessos" -Value 1 -Type DWord

# Criar nova chave
New-Item -Path "HKCU:\Software\MyApp\Config"

# Exportar chave (backup) — SEMPRE antes de editar
Export-Clixml -Path "backup_run_key.xml" -InputObject (Get-ItemProperty "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Run")
# Ou via regedit: File > Export

# REG.EXE na linha de comando
reg query "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion" /v ProductName
reg add "HKCU\Software\MyApp" /v Enabled /t REG_DWORD /d 1
reg export "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" C:\backup_run.reg
reg import C:\backup_run.reg
