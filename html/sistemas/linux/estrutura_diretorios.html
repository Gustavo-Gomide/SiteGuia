Estrutura de Diretórios Linux — FHS

O FHS (Filesystem Hierarchy Standard) define onde cada tipo de arquivo deve ficar no Linux. Conhecer essa estrutura evita confusão ao localizar configurações, binários, logs e dados de aplicação. Diferente do Windows com suas letras de drive, o Linux usa uma árvore única com raiz em /, e todos os sistemas de arquivos são montados dentro dela.

Mesmo pen drives, discos externos e partições de rede aparecem como subdiretórios dentro dessa árvore, montados em pontos como /mnt ou /media.

/proc e /sys — Sistemas Virtuais

/proc e /sys são sistemas de arquivos virtuais criados em memória pelo kernel. Não ocupam espaço em disco e expõem informações do kernel e dos processos como arquivos comuns de texto.

/proc e /sys — leitura e manipulação
# /proc — informações de processos e kernel (virtual, criado em memória)
cat /proc/cpuinfo         # informações de cada núcleo da CPU
cat /proc/meminfo         # uso de memória RAM
cat /proc/loadavg         # carga do sistema (1, 5, 15 min)
cat /proc/uptime          # segundos desde o boot
ls /proc/1/               # informações do processo com PID 1 (systemd)
cat /proc/1/cmdline       # linha de comando do processo 1
cat /proc/net/if_inet6    # endereços IPv6 das interfaces
cat /proc/sys/net/ipv4/ip_forward   # parâmetro do kernel (0 ou 1)

# /sys — interface ao kernel via sysfs (hierarquia de objetos do kernel)
ls /sys/class/net/        # interfaces de rede
cat /sys/block/sda/size   # tamanho do disco em setores de 512B
cat /sys/class/backlight/*/brightness   # brilho da tela
echo 1 > /sys/class/net/eth0/flags      # manipular flags de interface

# Verificar sistemas de arquivo montados
lsblk
mount | column -t
findmnt --tree

/var — Dados Variáveis

/var — logs, cache e dados de serviços
# Logs do sistema
ls /var/log/
tail -f /var/log/syslog               # log geral do sistema
journalctl -f                         # logs do systemd (equivalente moderno)
tail -100 /var/log/auth.log           # tentativas de autenticação

# Cache de pacotes
ls /var/cache/apt/archives/           # pacotes .deb baixados pelo apt
du -sh /var/cache/                    # tamanho do cache

# Dados de serviços
ls /var/www/html/                     # raiz padrão do Apache/Nginx
ls /var/lib/mysql/                    # dados do MySQL/MariaDB
ls /var/lib/postgresql/               # dados do PostgreSQL
ls /var/spool/cron/                   # crontabs dos usuários
