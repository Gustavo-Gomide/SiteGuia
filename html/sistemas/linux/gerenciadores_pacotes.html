Gerenciadores de Pacotes Linux

APT, DNF, Pacman, Snap, Flatpak e AppImage — instalar e gerenciar software no Linux

O gerenciador de pacotes e uma das maiores vantagens do Linux sobre outros sistemas operacionais. Com um unico comando voce instala, atualiza, remove e busca software, com resolucao automatica de dependencias. Cada familia de distribuicoes possui seu proprio gerenciador, mas todos compartilham os mesmos conceitos fundamentais.

APT — Debian e Ubuntu

O APT (Advanced Package Tool) gerencia pacotes .deb nas distribuicoes baseadas em Debian. Ele e um frontend de alto nivel para o dpkg.

APT — referencia de comandos
# Atualizar lista de pacotes dos repositorios
sudo apt update

# Atualizar todos os pacotes instalados
sudo apt upgrade

# Atualizar + remover pacotes obsoletos e fazer transicoes
sudo apt full-upgrade

# Instalar um pacote
sudo apt install nginx
sudo apt install nginx=1.24.0-1  # versao especifica

# Remover pacote (mantem arquivos de configuracao)
sudo apt remove nginx

# Remover pacote + configuracoes
sudo apt purge nginx

# Remover dependencias nao mais necessarias
sudo apt autoremove

# Buscar pacote
apt search "servidor web"

# Informacoes sobre um pacote
apt show nginx

# Listar pacotes instalados
dpkg -l | grep nginx

# Instalar pacote .deb local
sudo dpkg -i pacote.deb

# Busca offline no cache local
apt-cache search nginx
apt-cache show nginx

Os repositorios sao definidos em /etc/apt/sources.list e em arquivos dentro de /etc/apt/sources.list.d/. PPAs (Personal Package Archives) do Ubuntu sao adicionados via add-apt-repository ppa:usuario/ppa.

DNF / YUM — Red Hat e Fedora

O DNF (Dandified YUM) e o gerenciador de pacotes das distribuicoes Red Hat desde o Fedora 22. Ele e o sucessor do YUM com melhor resolucao de dependencias, suporte a modulos e melhor performance.

DNF — referencia de comandos
# Instalar pacote
sudo dnf install nginx

# Remover pacote
sudo dnf remove nginx

# Atualizar todos os pacotes
sudo dnf update

# Buscar pacote
dnf search nginx

# Informacoes detalhadas
dnf info nginx

# Historico de transacoes (e possivel desfazer)
dnf history
dnf history undo 5

# Instalar grupo de pacotes
sudo dnf group install "Development Tools"

# Listar repositorios configurados
dnf repolist

# Listar pacotes de um repositorio especifico
dnf repo-pkgs epel list

Os repositorios ficam em /etc/yum.repos.d/. O EPEL (Extra Packages for Enterprise Linux) expande significativamente o repertorio disponivel no RHEL/CentOS: sudo dnf install epel-release.

Pacman — Arch Linux

O pacman e o gerenciador de pacotes do Arch Linux e derivados (Manjaro, EndeavourOS). E conhecido pela velocidade, simplicidade e acesso ao AUR (Arch User Repository).

Pacman — referencia de comandos
# Instalar pacote
sudo pacman -S nginx

# Remover pacote
sudo pacman -R nginx

# Remover pacote + dependencias orfas
sudo pacman -Rs nginx

# Atualizar todo o sistema
sudo pacman -Syu

# Buscar pacote nos repositorios
pacman -Ss nginx

# Informacoes sobre pacote dos repositorios
pacman -Si nginx

# Informacoes sobre pacote instalado
pacman -Qi nginx

# Listar todos os pacotes instalados
pacman -Q

# Listar apenas pacotes instalados explicitamente
pacman -Qe

# Instalar pacote do AUR (com yay)
yay -S pacote-do-aur

# Construir e instalar manualmente do AUR
git clone https://aur.archlinux.org/pacote.git
cd pacote && makepkg -si

Snap, Flatpak e AppImage

Formatos de distribuicao universais permitem que um pacote rode em qualquer distribuicao Linux, contornando fragmentacao de versoes de bibliotecas.

Caracteristica Snap Flatpak AppImage
Isolamento (sandbox)Sim (AppArmor)Sim (Bubblewrap)Nao — executa direto
Tamanho tipicoGrande (inclui runtime)Grande (runtime compartilhado)Medio (portable)
Velocidade de inicioLenta (squashfs mount)ModeradaRapida
Canais de updateSim (stable, beta, edge)Sim (stable, beta)Nao automatico
Suporte offlineLimitadoSim (sideload)Total
Onde funcionaLinux (Canonical)Linux (Flathub)Qualquer Linux x86_64
Snap e Flatpak — comandos essenciais
# SNAP
snap find vlc
snap install vlc
snap remove vlc
snap list
snap refresh vlc          # atualizar especifico
snap refresh              # atualizar todos

# FLATPAK
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub org.videolan.VLC
flatpak uninstall org.videolan.VLC
flatpak list
flatpak update

# APPIMAGE — apenas tornar executavel e rodar
chmod +x Aplicativo-x86_64.AppImage
./Aplicativo-x86_64.AppImage

Use AppImage para binarios portáteis sem instalacao — ideal para ferramentas que voce quer rodar ocasionalmente. Prefira pacotes nativos para softwares criticos de servidor. Flatpak e Snap sao boas opcoes para aplicativos desktop de terceiros que nao estao nos repositorios oficiais.
