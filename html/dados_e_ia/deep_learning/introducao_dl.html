Deep Learning

Quando a máquina começa a imitar o cérebro.

1. Uma subárea do Machine Learning

Se Machine Learning é uma boneca russa, Deep Learning é a bonequinha menor dentro dela. Ele foca exclusivamente em algoritmos inspirados na estrutura biológica do cérebro: as Redes Neurais Artificiais.

O termo "Deep" (Profundo) vem da quantidade de camadas (layers) entre a entrada e a saída. Quanto mais camadas, mais "profunda" e inteligente é a rede.

2. Arquitetura Neural

Informação entra (Input), processa nas camadas escondidas (Hidden) e sai uma decisão (Output).

3. A Grande Diferença

Machine Learning Clássico

  • Funciona bem com poucos dados.
  • Requer Feature Engineering manual (ex: você cria a coluna "IMC" a partir de peso/altura).
  • Treinamento rápido (CPU).
  • Interpretável (dá pra entender o porquê).

Deep Learning

  • Precisa de MUITOS dados.
  • Faz Feature Extraction sozinho (aprende o que é importante pixel a pixel).
  • Treinamento pesado (precisa de GPU).
  • Caixa Preta (difícil explicar a decisão).

4. A Simplicidade do Keras

Criar um cérebro artificial hoje em dia requer poucas linhas de código.

import tensorflow as tf
from tensorflow.keras import layers, models

# Criando a estrutura (Sequencial)
model = models.Sequential([
# Camada de Entrada + Oculta 1 (128 neurônios)
layers.Dense(128, activation='relu', input_shape=(784,)),
# Camada Oculta 2 (64 neurônios)
layers.Dense(64, activation='relu'),
# Camada de Saída (10 classes - ex: dígitos 0-9)
layers.Dense(10, activation='softmax')
])

# Compilando
model.compile(optimizer='adam',
loss='sparse_categorical_crossentropy',
metrics=['accuracy'])
