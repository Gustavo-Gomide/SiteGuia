Redes Neurais (MLP)

Multi-Layer Perceptron: A base do Deep Learning.

1. A Soma Ponderada

Uma Rede Neural é composta por camadas de neurônios. Mas o que um neurônio faz? Ele é uma calculadora simples que realiza 3 passos:

  1. Recebe Inputs ($x$): Dados brutos ou saídas da camada anterior.
  2. Pondera ($w$): Multiplica cada input por um "peso" (importância).
  3. Adiciona Viés ($b$): Soma um valor extra para deslocar o resultado.
$z = (x_1 \cdot w_1) + (x_2 \cdot w_2) + ... + b$

2. Por dentro do Neurônio

Veja os dados fluindo, sendo multiplicados e somados.

3. O Gatilho (Ativação)

A soma ponderada é apenas matemática linear. Para a rede aprender coisas complexas (curvas, formas), precisamos de Não-Linearidade.

A Função de Ativação decide se o neurônio "dispara" ou não.

  • ReLU: Se negativo vira zero, se positivo mantém. (Padrão ouro).
  • Sigmoid: Espreme entre 0 e 1 (Usado para probabilidade).
  • Softmax: Transforma números em uma distribuição de probabilidade (Output final).

4. Dense Layer no Keras

Uma camada "Densa" (Fully Connected) conecta todos os inputs a todos os neurônios.

from tensorflow.keras import layers

# Criando uma camada com 32 neurônios
# Cada um terá seus próprios pesos e viés
layer = layers.Dense(
units=32, # Quantidade de neurônios
activation='relu', # Função de ativação (o gatilho)
use_bias=True # Incluir o termo B (padrão é True)
)

# O número de parâmetros treináveis será:
# (Inputs * 32) + 32 (vieses)
