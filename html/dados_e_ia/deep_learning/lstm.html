LSTM

Long Short-Term Memory: A cura para a amnésia das redes neurais.

1. A Superestrada da Informação

O segredo da LSTM é o Cell State (Estado da Célula). Imagine uma esteira rolante que atravessa toda a rede.

A informação pode viajar por essa esteira praticamente inalterada. Isso permite que a rede "lembre" de algo que aconteceu há 1000 passos atrás (como o gênero do sujeito em um texto longo) sem que o sinal se perca.

2. Por dentro da Célula

Observe como a linha superior (Dourada) flui direto, sofrendo apenas pequenas intervenções dos portões inferiores.

3. As Três Válvulas

1. Forget Gate (Esquecimento)

Decide o que jogar fora da memória antiga.
Ex: "O sujeito da frase mudou, esqueça o gênero antigo."

ft = σ(Wf • [h, x])

2. Input Gate (Entrada)

Decide o que adicionar de novo à memória.
Ex: "Guardar a informação de que o novo sujeito é plural."

it = σ(Wi • [h, x])

3. Output Gate (Saída)

Decide o que vamos entregar como resultado (Hidden State) agora, baseado na memória filtrada.

ot = σ(Wo • [h, x])

4. Keras

Usar uma LSTM é tão simples quanto usar uma RNN comum.

from tensorflow.keras import layers

# return_sequences=True: Retorna output para cada passo (Seq2Seq)
# return_sequences=False: Retorna apenas o último output (Classificação)

layer = layers.LSTM(
units=64,
return_sequences=True,
input_shape=(100, 1) # 100 timesteps
)
