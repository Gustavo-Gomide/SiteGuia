Introdução a Big Data

Armazenar, processar e extrair valor de dados em grande escala.

1. O que é Big Data?

Big Data é um conjunto de práticas e tecnologias para lidar com dados que excedem a capacidade de ferramentas tradicionais (por volume, velocidade, variedade ou requisitos de confiabilidade/custo).

2. Componentes típicos

  • Ingestão: batch e/ou streaming.
  • Armazenamento: data lake, warehouse, bancos distribuídos.
  • Processamento: Spark, Flink, SQL distribuído.
  • Orquestração: agendamento e pipelines (ETL/ELT).
  • Governança: catálogo, qualidade, segurança e lineage.

3. Boas práticas

  • Defina contratos de dados e camadas (raw/bronze/silver/gold).
  • Monitore qualidade e custos (storage/compute/egress).
  • Versione esquemas e trate compatibilidade.
