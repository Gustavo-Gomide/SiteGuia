Coleta de Dados

Onde encontrar dados e como extraí-los com segurança.

1. Fontes e formatos

Antes de coletar, defina o formato e a origem dos dados.

  • Estruturados: tabelas (SQL), CSV/Parquet, planilhas.
  • Não estruturados: texto livre, imagens/vídeos, documentos e logs.
  • Semi-estruturados: JSON, XML e eventos.

2. Métodos comuns

  • SQL: consultas em bancos relacionais e data warehouses.
  • APIs: endpoints REST/GraphQL retornando JSON.
  • Arquivos: CSV/Parquet exportados de sistemas ou pipelines.
  • Logs/eventos: telemetria de apps, sensores e sistemas.
SELECT *
FROM clientes

3. Web scraping (quando não há API)

Web scraping é extrair informações de páginas web. Use com responsabilidade: respeite termos de uso, limites de requisição e privacidade (ex.: LGPD).

4. ETL/ELT (pipeline)

  • Extract: buscar dados (SQL, API, arquivos).
  • Transform: limpar, padronizar e enriquecer.
  • Load: salvar em um destino para análise/consumo.

Em alguns cenários, faz sentido ELT: carregar primeiro e transformar depois.
