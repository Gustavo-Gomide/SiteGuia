Ciencia de Dados

1. O que é?

Ciência de Dados combina estatística, programação e entendimento do problema para transformar dados em insights e decisões. Em geral, o objetivo é responder perguntas, medir impacto e orientar ações.

2. Pilares essenciais

  • Matemática e Estatística: probabilidade, inferência, amostragem e testes.
  • Computação: Python/SQL, dados tabulares, pipelines e automação.
  • Negócio/Contexto: métrica certa, restrições, custo/benefício e comunicação.

3. Próximos passos

  • Ciclo de vida dos dados
  • Coleta de dados
  • Limpeza de dados
  • Exploração (EDA)
  • Visualização
