Exploração de Dados (EDA)

Faça perguntas antes de treinar modelos.

1. Objetivo da EDA

EDA é a etapa em que você investiga dados para entender distribuições, variabilidade, relações entre variáveis e anomalias.

  • Quais colunas existem e o que significam?
  • Há valores faltantes, duplicatas ou categorias inconsistentes?
  • Existem outliers ou faixas improváveis?
  • Quais variáveis parecem ter relação com o alvo?

2. Estatística descritiva

Alguns conceitos que aparecem o tempo todo:

  • Média vs mediana: diferenças grandes sugerem assimetria/outliers.
  • Desvio padrão: dá uma noção de dispersão.
  • Percentis: ajudam a entender caudas e extremos.
# resumo estatístico
df.describe(include='all')

3. Correlação

Uma forma comum de medir relação linear é o coeficiente de Pearson ($r$), que varia de -1 a 1. Use com cuidado: correlação não implica causalidade. 

# matriz de correlação (numéricas)
df.corr(numeric_only=True)

4. Ferramentas

  • Pandas: leitura, limpeza e agregações.
  • Matplotlib/Seaborn: gráficos rápidos (histograma, boxplot, scatter, heatmap).
  • Relatórios automatizados: ajudam a acelerar a exploração em datasets grandes.
import seaborn as sns
import matplotlib.pyplot as plt

sns.heatmap(df.corr(numeric_only=True), annot=False)
plt.show()
