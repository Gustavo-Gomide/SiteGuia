Introdução ao Machine Learning

Modelos que aprendem padrões a partir de dados.

1. O que e?

Machine Learning é uma área da IA em que construímos modelos que aprendem com exemplos para fazer previsões, classificações ou recomendações. Em vez de escrever regras fixas para cada cenário, ajustamos um modelo com dados e avaliamos como ele generaliza.

2. Tipos de aprendizado

  • Supervisionado: aprende com dados rotulados (classificação e regressão).
  • Não supervisionado: encontra estrutura sem rótulos (clusterização, redução de dimensionalidade).
  • Por reforço: aprende por tentativa e erro com recompensas.
