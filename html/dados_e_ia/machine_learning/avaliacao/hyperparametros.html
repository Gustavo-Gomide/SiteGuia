Hiperparâmetros

Configurações que controlam como o modelo aprende.

1. Conceito

Parâmetros são aprendidos a partir dos dados (por exemplo, pesos de um modelo). Hiperparâmetros são definidos por você antes do treino (por exemplo, profundidade máxima de uma árvore).

2. Exemplos comuns

  • K no KNN: quantos vizinhos influenciam a decisão.
  • max_depth em árvores: controla complexidade e risco de overfitting.
  • C (SVM/logística) e alpha (regularização): força da penalização.
  • n_estimators em ensembles: número de árvores/modelos.

3. Como ajustar

O ajuste (tuning) testa combinações de hiperparâmetros e escolhe as melhores com base em validação. Em geral, use validação cruzada para reduzir variação. 

from sklearn.model_selection import GridSearchCV
from sklearn.svm import SVC

param_grid = {
  'C': [0.1, 1, 10],
  'kernel': ['rbf', 'linear']
}

grid = GridSearchCV(SVC(), param_grid, cv=5, scoring='f1')
grid.fit(X, y)

print(grid.best_params_)
