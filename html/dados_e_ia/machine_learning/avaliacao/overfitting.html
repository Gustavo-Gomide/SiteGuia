Overfitting

Quando o modelo decora e não generaliza.

1. Definicao

Overfitting acontece quando o modelo se ajusta demais ao treino, aprendendo também ruído. O resultado típico é: desempenho bom no treino e ruim em dados novos.

Underfitting é o oposto: o modelo é simples demais e não captura padrões úteis.

2. Sinais comuns

  • Grande diferença entre métrica de treino e validação/teste.
  • Métrica de validação piora enquanto o treino melhora.
  • Modelo muito complexo para o volume/qualidade dos dados.

3. Como reduzir overfitting

  • Mais dados ou dados mais variados (quando possível).
  • Validação cruzada para estimar desempenho com mais estabilidade.
  • Regularização (por exemplo, L2 em modelos lineares).
  • Reduzir complexidade (profundidade, número de parâmetros, features).
