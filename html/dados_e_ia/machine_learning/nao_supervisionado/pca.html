PCA

Projetar dados para componentes que capturam variancia.

1. Intuicao

O PCA encontra novas direcoes (componentes) que explicam o maximo de variancia dos dados. Voce representa cada exemplo por um conjunto menor de componentes.

  • PC1: direcao com maior variancia.
  • PC2: segunda direcao, ortogonal a PC1, e assim por diante.

2. Variancia explicada

A variancia explicada ajuda a decidir quantos componentes manter. Um criterio comum e manter a menor quantidade de PCs que atinge um alvo (ex.: 95%).

Como regra pratica: padronize features antes (media 0, desvio 1).

3. Exemplo (scikit-learn)

from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

X_scaled = StandardScaler().fit_transform(X)

pca = PCA(n_components=0.95, random_state=42)
X_reduced = pca.fit_transform(X_scaled)

print("componentes:", pca.n_components_)
print("variancia acumulada:", pca.explained_variance_ratio_.sum())

Para plotar em 2D, use PCA(n_components=2).
