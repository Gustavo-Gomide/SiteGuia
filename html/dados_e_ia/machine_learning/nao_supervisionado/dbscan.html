DBSCAN

Clustering por densidade com rotulo de ruido.

1. Ideia central

O DBSCAN encontra clusters como regioes densas no espaco. Pontos fora de regioes densas viram ruido.

  • eps: raio de vizinhanca.
  • min_samples: minimo de pontos no raio para considerar densidade.

2. Vantagens e limitacoes

  • Pro: nao precisa definir $K$; lida bem com formatos irregulares; identifica outliers.
  • Contra: sensivel a escala; sofre quando densidades variam muito entre clusters.

3. Ajuste de parametros

  • Padronize features antes de ajustar eps.
  • Use k-distance plot como heuristica para eps (dependendo do dataset).
  • Comece com min_samples entre 5 e 20 e ajuste.

4. Exemplo (scikit-learn)

from sklearn.cluster import DBSCAN
from sklearn.preprocessing import StandardScaler

X_scaled = StandardScaler().fit_transform(X)

model = DBSCAN(
        eps=0.4,
        min_samples=10,
)

labels = model.fit_predict(X_scaled)

# rotulo -1 costuma significar ruido
noise_ratio = (labels == -1).mean()
print("ruido:", noise_ratio)
print("clusters:", len(set(labels)) - (1 if -1 in labels else 0))
