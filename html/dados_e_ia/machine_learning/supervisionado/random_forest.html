Random Forest

Bagging de árvores com aleatoriedade controlada.

1. Ideia central

Random Forest combina muitas árvores treinadas em amostras bootstrap. Em cada split, usa um subconjunto aleatório de features. Isso reduz correlação entre árvores e melhora generalização.

2. Hiperparâmetros importantes

  • n_estimators: número de árvores.
  • max_depth: complexidade de cada árvore.
  • max_features: features candidatas por split.
  • min_samples_leaf: suaviza folhas e reduz overfitting.

3. Exemplo (scikit-learn)

from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(
    n_estimators=400,
    random_state=42,
    n_jobs=-1,
)

model.fit(X_train, y_train)
print(model.score(X_test, y_test))
