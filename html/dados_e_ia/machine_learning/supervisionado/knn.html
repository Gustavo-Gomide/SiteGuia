KNN

Prediz usando os vizinhos mais próximos.

1. Ideia central

No KNN, não existe um treino pesado: para prever, você procura os $k$ exemplos mais próximos (por uma métrica de distância) e agrega os rótulos.

  • Classificação: voto dos vizinhos.
  • Regressão: média/mediana dos valores.

2. Hiperparâmetros

  • n_neighbors (k): k pequeno pode overfitar; k grande suaviza demais.
  • metric: euclidiana, manhattan
  • weights: uniforme ou ponderado pela distância.

Como usa distância, escalar as features normalmente é essencial.

3. Exemplo (scikit-learn)

from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier

model = make_pipeline(
    StandardScaler(),
    KNeighborsClassifier(n_neighbors=7, weights='distance')
)

model.fit(X_train, y_train)
print(model.score(X_test, y_test))
