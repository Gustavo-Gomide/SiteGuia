Árvores de Decisão

Regras aprendidas por divisões (splits).

1. Como funciona

Uma árvore cria regras do tipo se então, dividindo os dados em nós até chegar a folhas. Cada split escolhe uma feature e um limiar para melhorar a separação.

  • Classificação: critérios comuns são Gini e Entropia.
  • Regressão: busca reduzir variância/erro (ex.: MSE) nas folhas.

2. Overfitting e controle

Árvores podem crescer demais e overfitar. Os controles mais comuns:

  • max_depth
  • min_samples_split / min_samples_leaf
  • max_leaf_nodes

3. Exemplo (scikit-learn)

from sklearn.tree import DecisionTreeClassifier

model = DecisionTreeClassifier(
    max_depth=4,
    random_state=42,
)

model.fit(X_train, y_train)
print(model.score(X_test, y_test))
