Regressão Logística

Classificação baseada em probabilidade.

1. O que e

Apesar do nome, regressão logística é usada principalmente para classificação. Ela modela $P(y=1|x)$ usando a função sigmoide e decide a classe com um limiar (ex.: 0.5).

2. Função de perda e regularização

O treino costuma minimizar uma forma de log-loss. Na prática, é comum usar regularização (L2 por padrão no scikit-learn) para reduzir overfitting.

3. Exemplo (scikit-learn)

from sklearn.linear_model import LogisticRegression
from sklearn.metrics import classification_report

model = LogisticRegression(max_iter=1000)
model.fit(X_train, y_train)

y_pred = model.predict(X_test)
print(classification_report(y_test, y_pred))

Em dados desbalanceados, avalie além de acurácia: precisão, recall e F1.
