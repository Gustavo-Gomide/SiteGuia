SVM

Separação por margem máxima (com ou sem kernel).

1. Ideia central

A SVM busca um hiperplano que separa as classes com a maior margem possível. Com kernels, ela pode criar fronteiras não lineares sem explicitar novas features.

2. Hiperparâmetros

  • C: controla o trade-off entre margem e erros (C alto tende a ajustar mais).
  • kernel: linear, rbf, poly
  • gamma (rbf): controla o alcance da influência (gamma alto pode overfitar).

Normalização/escala de features costuma ser importante para SVM.

3. Exemplo (scikit-learn)

from sklearn.pipeline import make_pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.svm import SVC

model = make_pipeline(
    StandardScaler(),
    SVC(kernel='rbf', C=2.0, gamma='scale')
)

model.fit(X_train, y_train)
print(model.score(X_test, y_test))
