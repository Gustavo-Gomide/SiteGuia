Regressão Linear

O modelo base para prever valores numéricos.

1. O que e

Regressão linear modela uma saída contínua $y$ como uma combinação linear das entradas $x$. Na forma mais simples (uma variável): $y = w\,x + b$.

2. Como treina (MSE)

Uma escolha comum de perda é o erro quadrático médio (MSE). Treinar significa encontrar $w$ e $b$ que minimizam essa perda no conjunto de treino. 

from sklearn.linear_model import LinearRegression

model = LinearRegression()
model.fit(X_train, y_train)

y_pred = model.predict(X_test)

3. Cuidados práticos

  • Escala: para alguns cenários (regularização), normalizar pode ajudar.
  • Outliers: podem puxar a reta e degradar o ajuste.
  • Generalização: valide (holdout ou validação cruzada) e meça métricas.
