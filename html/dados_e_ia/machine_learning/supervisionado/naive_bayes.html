Naive Bayes

Probabilidades simples que funcionam surpreendentemente bem.

1. Ideia central

Naive Bayes aplica o Teorema de Bayes assumindo independência condicional entre features. Essa suposição é ingênua, mas o método costuma ser forte como baseline, especialmente em texto.

2. Variantes

  • GaussianNB: features contínuas (distribuição normal).
  • MultinomialNB: contagens (muito comum em NLP/bag-of-words).
  • BernoulliNB: presença/ausência (binário).

3. Exemplo (texto)

from sklearn.pipeline import make_pipeline
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.naive_bayes import MultinomialNB

model = make_pipeline(
    CountVectorizer(),
    MultinomialNB()
)

model.fit(textos_treino, y_treino)
print(model.score(textos_teste, y_teste))
