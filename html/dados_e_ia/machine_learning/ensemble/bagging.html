Bagging

Ensemble para reduzir variancia com treino paralelo.

1. Ideia central

Bagging (Bootstrap Aggregating) treina varios modelos do mesmo tipo em amostras diferentes do conjunto de treino e depois agrega as previsoes.

  • Bootstrap: cada modelo ve uma amostra com reposicao.
  • Agregacao: media (regressao) ou voto (classificacao).
  • Efeito tipico: reduz variancia e melhora estabilidade.

2. Quando funciona melhor

Bagging costuma ajudar quando o modelo base e instavel: pequenas mudancas nos dados geram modelos bem diferentes.

  • Arvores de decisao (ex.: base do Random Forest).
  • Modelos com alta variancia em dados ruidosos.

3. Exemplo (scikit-learn)

from sklearn.ensemble import BaggingClassifier
from sklearn.tree import DecisionTreeClassifier

base = DecisionTreeClassifier(random_state=42)
model = BaggingClassifier(
    estimator=base,
    n_estimators=200,
    max_samples=0.8,
    max_features=1.0,
    bootstrap=True,
    random_state=42,
    n_jobs=-1,
)

model.fit(X_train, y_train)
print(model.score(X_test, y_test))

Dica: com arvores, o mais comum na pratica e usar RandomForest, que e um tipo de bagging com amostragem de features em cada split.
