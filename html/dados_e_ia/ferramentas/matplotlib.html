Matplotlib

Guia Prático — Matplotlib é o “controle fino” dos gráficos. Aprenda o modelo Figure/Axes e você consegue construir qualquer visualização sem se perder na API global.

Matplotlib

Visao Geral

1. Modelo mental (Figure e Axes)

Regra de ouro

Pense em Figure (tela) e Axes (quadro). Desenhe sempre no ax.

Exemplo mínimo

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 200)
y = np.sin(x)

fig, ax = plt.subplots()
ax.plot(x, y, label="sin(x)")
ax.set(title="Exemplo", xlabel="x", ylabel="y")
ax.legend()
fig.tight_layout()
plt.show()

2. Receitas (subplots e gráficos comuns)

  • Linha: tendências.
  • Scatter: relação entre variáveis.
  • Barras: categorias.
  • Hist: distribuição.
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 10, 200)
y = np.sin(x)

fig, axs = plt.subplots(2, 2, figsize=(10, 6))

axs[0, 0].plot(x, y)
axs[0, 0].set_title("Linha")

axs[0, 1].scatter(np.random.randn(200), np.random.randn(200), alpha=0.5)
axs[0, 1].set_title("Scatter")

axs[1, 0].bar(["A", "B", "C"], [5, 2, 7])
axs[1, 0].set_title("Barras")

axs[1, 1].hist(np.random.randn(500), bins=30)
axs[1, 1].set_title("Hist")

fig.tight_layout()
plt.show()

3. Exportação e acabamento

Salvar para web e para print

fig.savefig("grafico.png", dpi=150)
fig.savefig("grafico.svg")
fig.savefig("grafico.pdf")
  • PNG: web.
  • SVG/PDF: vetor (bom para relatório/print).

4. Erros comuns

  • Pyplot global confuso: use fig, ax e desenhe no ax.
  • Tamanho ruim: ajuste figsize.
  • Cortes ao salvar: aplique tight_layout() antes de savefig.
