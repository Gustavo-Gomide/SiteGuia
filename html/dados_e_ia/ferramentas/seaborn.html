Seaborn

Visao Geral

1. Modelo mental (antes de plotar)

O que o Seaborn espera

  • Dados “tidy/long”: 1 coluna por variável, 1 linha por observação.
  • Mapeamento estético: colunas viram eixos e canais como hue, style, size.
  • Integração: Seaborn desenha e devolve um Axes; Matplotlib finaliza detalhes.

Exemplo (long-form)

# “Long-form” (recomendado)
# colunas: data, segmento, vendas

# “Wide-form” (às vezes aparece)
# colunas: data, vendas_A, vendas_B
# para usar hue, você normalmente converte com melt()

2. Setup mínimo (boilerplate)

Configuração padrão

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

sns.set_theme(style="darkgrid")
plt.rcParams["figure.figsize"] = (10, 5)

Dica: se você quer consistência no projeto, padronize tema e tamanho de figura.

3. Receitas essenciais (com exemplos)

3.1 Distribuição (histograma + KDE)

# Entender concentração e outliers
sns.histplot(df, x="idade", bins=30, kde=True)
plt.title("Distribuição de idade")
plt.show()

3.2 Comparar grupos (boxplot)

# Comparar uma métrica por categoria
sns.boxplot(df, x="categoria", y="nota")
plt.xticks(rotation=30)
plt.show()

Boxplot: mediana/quartis; bom para comparar grupos rapidamente.

3.3 Relação entre variáveis (scatter + hue)

sns.scatterplot(df, x="renda", y="gasto", hue="segmento", alpha=0.7)
plt.title("Renda vs gasto (por segmento)")
plt.show()

3.4 Tendência no tempo (lineplot)

# Atenção: se houver várias linhas com a mesma data,
# o seaborn agrega por padrão.
sns.lineplot(df, x="data", y="vendas")
plt.xticks(rotation=30)
plt.show()

Se você não quer agregação “automática”, agregue no pandas antes.

4. Correlação (heatmap) do jeito certo

Receita clássica

num = df.select_dtypes(include="number")
corr = num.corr(numeric_only=True)

sns.heatmap(corr, annot=True, fmt=".2f", cmap="viridis")
plt.title("Correlação (variáveis numéricas)")
plt.show()

Correlação não prova causalidade. Use como triagem e valide com método/experimento.

5. Facetas (small multiples) para comparar padrões

Quando usar

Quando um único gráfico fica poluído, repita o mesmo gráfico por grupo (segmento/cidade/etc.).

Exemplo

g = sns.FacetGrid(df, col="segmento", col_wrap=3, height=3)
g.map_dataframe(sns.histplot, x="gasto", bins=25)
plt.show()

6. Erros comuns (e como corrigir)

Problemas que deixam o gráfico “bonito e inútil”

  • Dados wide em vez de long: use melt() para criar colunas “variável/valor”.
  • Agregação escondida: em lineplot com x repetido, entenda como o Seaborn agregou.
  • Overplotting: ajuste alpha, amostre dados, ou use KDE/hexbin.
  • Leitura errada: não conclua causalidade a partir de correlação.

Exemplo (convertendo wide → long)

# wide: data, vendas_A, vendas_B
df_long = df.melt(id_vars=["data"], var_name="serie", value_name="vendas")
sns.lineplot(df_long, x="data", y="vendas", hue="serie")
plt.show()
