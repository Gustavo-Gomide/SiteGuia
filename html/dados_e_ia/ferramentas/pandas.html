Visao Geral

1. Do arquivo ao DataFrame (fluxo real)

Carregar e inspecionar

import pandas as pd

df = pd.read_csv("dados.csv")
print(df.head())
print(df.info())
print(df.describe(include="all"))

Se você não inspeciona (head/info/describe), você sempre vai “descobrir” bugs tarde.

Conceitos

  • Series: uma coluna (1D) com índice.
  • DataFrame: tabela (2D) com colunas e índice.

2. Operações que você vai usar todo dia

2.1 Selecionar, filtrar e criar colunas

# Seleção de colunas
df[["cliente", "valor"]]

# Filtrar linhas
df_filtrado = df.loc[df["valor"] > 100]

# Criar coluna derivada
df["ticket_medio"] = df["valor"] / df["itens"]

2.2 Missing e tipos

# Quantidade de valores ausentes por coluna
df.isna().sum()

# Remover linhas sem valor
df = df.dropna(subset=["valor"])

# Otimizar/organizar tipo
df["categoria"] = df["categoria"].astype("category")

2.3 GroupBy (agregação)

resumo = (
  df.groupby("categoria", as_index=False)
    .agg(vendas_total=("valor", "sum"), ticket_medio=("valor", "mean"), n=("valor", "size"))
)
print(resumo)

2.4 Merge/Join (unir tabelas)

clientes = pd.read_csv("clientes.csv")

# how="left": mantém todas as linhas de df
df2 = df.merge(clientes, on="cliente_id", how="left")

2.5 Datas e resample

df["data"] = pd.to_datetime(df["data"])

serie = (
  df.set_index("data")
    .resample("D")["valor"]
    .sum()
)
print(serie.head())

3. Boas práticas (pandas de produção)

  • Evite vazamento: se for ML, split antes de calcular estatísticas (média/escala).
  • Use .loc para atribuir e filtrar (menos armadilhas).
  • Prefira vetorização: evite apply sem necessidade.
  • Formato: Parquet para volume e performance; CSV para compatibilidade.
  • Memória: considere category para strings repetidas.

4. Erros comuns (e como não cair)

  • SettingWithCopyWarning: evite “cadeias” de indexação; use .loc e/ou crie cópia.
  • Datas como string: converta com to_datetime antes de resample/janelas.
  • Join explodindo linhas: chaves duplicadas geram multiplicação; valide unicidade.
