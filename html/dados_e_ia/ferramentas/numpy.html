NumPy

Guia Prático — NumPy é a base de computação numérica. Se você entende arrays, broadcasting e agregação por eixo, você evita 80% dos bugs e ganha performance sem esforço.

NumPy

Visao Geral

1. Começo rápido

Criar arrays

import numpy as np

a = np.array([1, 2, 3])
b = np.arange(0, 10)
m = np.random.default_rng(42).normal(size=(3, 4))

print(a.shape, a.dtype)
print(m.shape, m.dtype)

Guarde isso: shape (forma) e dtype (tipo) determinam quase tudo.

2. Operações essenciais (com exemplos)

2.1 Indexação e máscaras

x = np.array([10, 20, 30, 40])
print(x[1])        # 20
print(x[1:3])      # [20 30]

mask = x >= 30
print(x[mask])     # [30 40]

2.2 Broadcasting (por que NumPy é rápido)

A = np.ones((3, 4))
v = np.array([1, 2, 3, 4])

# v é “esticado” para (3,4) sem copiar explicitamente
Z = A * v
print(Z.shape)

2.3 Agregações por eixo (o lugar onde mais erra)

M = np.arange(12).reshape(3, 4)
print(M)

print(M.sum())        # soma total
print(M.sum(axis=0))  # soma por coluna
print(M.mean(axis=1)) # média por linha

2.4 Produto matriz-vetor (muito comum em ML)

X = np.random.default_rng(0).normal(size=(5, 3))
w = np.array([0.2, -0.1, 0.7])

y = X @ w
print(y)

3. Cuidados (bugs clássicos)

  • Dtype: float32 economiza memória, mas pode afetar precisão.
  • View vs cópia: slices podem compartilhar memória (alterar um altera outro).
  • Reprodutibilidade: use default_rng(seed) em vez da API antiga.
  • Loops Python: reescreva com vetorização; quando precisar, use np.where/np.clip.
