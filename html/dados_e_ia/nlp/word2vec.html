Word2Vec

Embeddings de palavras para capturar semântica.

1. Visao geral

Word2Vec é uma família de modelos que aprende uma representação vetorial (embedding) para cada palavra a partir do contexto em que ela aparece. A ideia central é que palavras usadas em contextos semelhantes tendem a ter significados relacionados.

2. Intuição (alto nível)

  • Entrada: texto tokenizado (sequências de palavras).
  • Objetivo: prever uma palavra pelo contexto (ou o contexto pela palavra).
  • Saída: vetores densos em que proximidade/cosseno refletem similaridade.

3. Variantes comuns

  • CBOW: prevê a palavra central a partir do contexto.
  • Skip-gram: prevê palavras de contexto a partir da palavra central.

Na prática, o treinamento costuma usar otimizações como amostragem negativa (negative sampling) para ficar viável em vocabulários grandes.

4. Usos típicos

  • Features para modelos clássicos (classificação de texto, clustering).
  • Busca semântica simples (aproximação via similaridade de vetores).
  • Base para médias/combinações de embeddings em documentos.

5. Limitações e próximos passos

  • Embedding é estático: a mesma palavra tem o mesmo vetor em qualquer contexto.
  • Para contexto dinâmico, veja modelos baseados em Transformer (embeddings contextuais).
