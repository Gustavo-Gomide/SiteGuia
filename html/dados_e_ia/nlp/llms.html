LLMs (Large Language Models)

Modelos de linguagem que geram e entendem texto em escala.

1. O que sao?

LLMs são modelos treinados para prever o próximo token, aprendendo padrões de linguagem a partir de grandes corpora. Eles podem ser adaptados para tarefas como chat, sumarização, extração e classificação.

2. Usos comuns

  • Assistentes, suporte e automação de texto.
  • RAG (buscar documentos e gerar resposta com base neles).
  • Extração estruturada (campos) e reescrita.

3. Limitações e riscos

  • Alucinação: pode inventar fatos com confiança.
  • Context window: limite de tokens de entrada/saída.
  • Privacidade: cuidado com dados sensíveis no prompt.
  • Custo/latência: depende do tamanho e do tipo de inferência.

4. Boas práticas

  • Comece com prompts simples e avaliações objetivas (conjunto de testes).
  • Use RAG para reduzir alucinação quando há base de conhecimento.
  • Defina guardrails: validação, checagem de formato e fallback.

Referencias
