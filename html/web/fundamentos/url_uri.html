URL e URI

Identificação de Recursos na Web

Compreenda como recursos são nomeados, localizados e resolvidos em aplicações distribuídas.

URI identifica um recurso; URL é um tipo de URI que também informa como localizá-lo. Esse detalhe impacta modelagem de rotas, APIs e interoperabilidade entre sistemas.

Estrutura de uma URL

Uma URL típica contém esquema, host, porta opcional, caminho, query string e fragmento. Exemplo: https://exemplo.com:443/produtos?categoria=web#topo.

Boas Práticas de Endereçamento

  • Use caminhos legíveis e estáveis ao longo do tempo.
  • Evite parâmetros redundantes e ambíguos.
  • Padronize lowercase e separação por hífen em slugs.
  • Defina política de canonical URL para SEO e consistência.

Erros Comuns

Endpoints que mudam sem versionamento, encoding incorreto de caracteres e inconsistência entre rotas internas e públicas são causas frequentes de quebra de integração e indexação ruim.

Endereçamento bem definido simplifica manutenção, integração entre serviços e evolução de produto.
