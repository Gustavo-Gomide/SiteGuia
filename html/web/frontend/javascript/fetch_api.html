    Fetch API

    Requisições HTTP Modernas no JavaScript

    Fetch substitui XHR com interface baseada em Promises.

    Fetch simplifica integração com APIs e funciona de forma natural com async/await.

    Uso

    Configure método, headers e corpo conforme o endpoint, e trate erros de rede e status HTTP explicitamente.
