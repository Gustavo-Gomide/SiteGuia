    DOM

    Modelo de Documento da Página

    O DOM representa a árvore de elementos renderizados pelo navegador.

    Manipular o DOM com critério é fundamental para desempenho e previsibilidade da interface.

    Operações Comuns

    • Selecionar e atualizar nós.
    • Criar e remover elementos.
    • Modificar classes e atributos.
