SSG

Static Site Generation

Páginas pré-geradas no build e servidas com latência mínima.

SSG combina simplicidade operacional e ótima performance para conteúdo majoritariamente estático, como documentação, marketing pages e portais de conteúdo.

Pontos Fortes

  • Tempo de resposta baixo com CDN.
  • Custos operacionais previsíveis.
  • Boa segurança por superfície reduzida no runtime.

Limites

Atualização de dados em tempo real requer estratégias híbridas, como revalidação incremental, client fetching e endpoints dinâmicos complementares.

Referências

  • JAMSTACK. Modern Web Development. 2025.
  • VERCEL. Static Generation Docs. 2025.

SSG é excelente para escalar conteúdo com previsibilidade de custo e entrega.
