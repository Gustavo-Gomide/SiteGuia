Caching

Menos Latência, Menos Custo, Mais Escala

Cache reduz trabalho repetido e melhora resposta percebida.

Caching eficiente combina headers HTTP, invalidação correta e política por tipo de recurso.

Camadas de Cache

  • Navegador.
  • CDN/reverse proxy.
  • Aplicação e banco de dados.

Boas Práticas

Use Cache-Control, ETag, versionamento de assets e invalidação seletiva para evitar conteúdo desatualizado.

Referências

  • RFC 9111. HTTP Caching. IETF, 2022.
  • MDN. HTTP Cache. 2025.

Cache sem estratégia gera inconsistência; com estratégia, vira vantagem competitiva.

"Performance sustentável nasce de política de cache explícita."
