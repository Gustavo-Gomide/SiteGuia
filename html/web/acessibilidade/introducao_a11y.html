Introdução A11y

Acessibilidade como Requisito de Engenharia

Interfaces acessíveis ampliam alcance, qualidade e conformidade legal.

Acessibilidade web não é camada extra de UX. Ela deve existir desde arquitetura de informação até componentes visuais e validação contínua.

Princípios Centrais

  • Perceptível.
  • Operável.
  • Compreensível.
  • Robusto.

Impacto em Produto

Melhora experiência geral, reduz abandono em fluxos críticos e aumenta compatibilidade com tecnologias assistivas e diferentes contextos de uso.

Referências

  • W3C WAI. Accessibility Fundamentals. 2025.
  • WCAG 2.2. Web Content Accessibility Guidelines. 2023.

Produto acessível é produto tecnicamente mais consistente para todos.

"Qualidade de interface se mede também por quem não ficou de fora."
