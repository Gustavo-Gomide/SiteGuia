O que são Autômatos Finitos

Autômatos Finitos

Máquinas de estados que reconhecem linguagens regulares — a base das expressões regulares e da análise léxica.

Um autômato finito é um modelo matemático de computação com quantidade finita de memória (os estados). Ele lê uma cadeia símbolo a símbolo e decide ao final se a aceita ou rejeita. É o reconhecedor exato das linguagens regulares.

Definição formal: M = (Q, Σ, δ, q₀, F)

  • Q — conjunto finito de estados
  • Σ — alfabeto de entrada (conjunto finito de símbolos)
  • δ: Q × Σ → Q — função de transição (para AFD) ou relação δ ⊆ Q × (Σ ∪ {ε}) × Q (para AFN)
  • q₀ ∈ Q — estado inicial
  • F ⊆ Q — estados de aceitação (finais)

O autômato aceita a cadeia w se existe um caminho de q₀ a algum estado em F lendo todos os símbolos de w.

AFD — Autômato Finito Determinístico

No AFD, para cada par (estado, símbolo) existe exatamente uma transição. O comportamento é totalmente determinado — para a mesma entrada, sempre o mesmo resultado.

AFD que aceita L = {w ∈ {a,b}* | w termina em “ab”}. Estados: q0 (inicial), q1 (leu a), q2 (leu ab — aceitação).

A tabela de transições é uma representação alternativa equivalente ao diagrama:

Estadoab
→ q0q1q0
q1q1q2
q2 *q1q0

→ indica estado inicial; * indica estado de aceitação.

AFN — Autômato Finito Não-Determinístico

No AFN, cada par (estado, símbolo) pode ter zero, uma ou várias transições possíveis. Além disso, podem existir transições-ε (muda de estado sem consumir símbolo).

Poder de reconhecimento

AFN e AFD reconhecem exatamente as mesmas linguagens — as linguagens regulares. O AFN pode ser exponencialmente mais compacto para certas linguagens.

Construção de subconjuntos

Todo AFN pode ser convertido em AFD equivalente pelo algoritmo de construção de subconjuntos (subset construction): cada estado do AFD representa um conjunto de estados do AFN. No pior caso, gera 2^|Q| estados.

ε-fecho

O ε-fecho(q) de um estado q é o conjunto de todos os estados alcançáveis por zero ou mais transições-ε a partir de q. Usado na conversão AFN→AFD.

AFN que aceita cadeias terminadas em “a” ou em “b”. Estado q0 tem transições-ε para dois sub-autômatos paralelos.

Relação com Expressões Regulares

Expressões regulares e autômatos finitos têm exatamente o mesmo poder descritivo. Toda regex pode ser convertida em AFN (via construção de Thompson), e todo AFD pode ser convertido em regex (via eliminação de estados).

Regex → AFN (Thompson)

Cada operação da regex (concatenação, união, Kleene) gera sub-autômatos com transições-ε que são compostos hierarquicamente. Resultado: AFN com ≤ 2|r| estados onde |r| é o tamanho da expressão.

AFN → AFD (Subconjuntos)

Estados do AFD = subconjuntos de Q. Estado inicial = ε-fecho(q₀). Para cada estado S e símbolo a: δ_DFA(S, a) = ε-fecho(⋃ δ_NFA(q, a) para q ∈ S).

AFD → Regex (Eliminação)

Equações de sistema linear sobre linguagens onde cada variável é o conjunto de cadeias que chegam a um estado. Solução pelo Teorema de Arden ou eliminação de estados.

# Python — verificar string com regex (internamente usa AFD)
import re
pattern = re.compile(r'(a|b)*ab')  # Compilado em AFD automaticamente
print(pattern.fullmatch('aab'))    # Match
print(pattern.fullmatch('ba'))     # None

Minimização de AFD

Para cada linguagem regular existe um AFD mínimo único (a menos de renomeação de estados). O algoritmo de minimização (Hopcroft ou por refinamento de partições):

  1. Partição inicial: {F} e {Q \ F}
  2. Refinamento: dois estados p, q são distinguíveis se existe alguma cadeia w que os leva a resultados diferentes (um aceita, outro não)
  3. Estados indistinguíveis são fundidos em um único estado no AFD mínimo

Teorema de Myhill-Nerode

Uma linguagem L é regular se e somente se a relação de equivalência de Myhill-Nerode (≡_L) tem índice finito. O número de classes de equivalência = número de estados do AFD mínimo para L.

x ≡_L y se e somente se para todo z: xz ∈ L ⟺ yz ∈ L
