Computação Quântica

Do qubit ao algoritmo de Shor — como a mecânica quântica redefine os limites do que é computável

A computação quântica explora os princípios da mecânica quântica — superposição, emaranhamento e interferência — para executar certos tipos de cálculo exponencialmente mais rápido do que qualquer computador clássico. Não é uma substituição do computador tradicional, mas um co-processador especializado para problemas específicos: fatoração de números enormes, simulação molecular, otimização combinatória e criptografia.

Bits Clássicos vs Qubits

O Bit Clássico

Um bit clássico está em exatamente um de dois estados: 0 ou 1. N bits clássicos representam um único número de N dígitos binários — ou seja, um estado por vez, mesmo que N bits possam representar 2ᴺ estados possíveis.

O Qubit

Um qubit é um sistema quântico de dois níveis. Antes de ser medido, ele existe em uma superposição de |0⟩ e |1⟩:

|ψ⟩ = α|0⟩ + β|1⟩, onde |α|² + |β|² = 1

Isso não significa que um qubit armazena 2 valores simultâneos — significa que a amplitude de probabilidade de cada estado pode ser manipulada antes da medição colapsar o estado.

Vantagem Quântica

N qubits em superposição representam uma superposição de 2ᴺ estados simultaneamente. Um computador quântico com 300 qubits pode representar mais estados do que átomos no universo observável — mas a medição colapsa tudo para um único resultado.

A arte está em usar interferência quântica para amplificar as amplitudes dos estados que correspondem à solução correta e cancelar os outros.

Emaranhamento

Dois qubits emaranhados têm estados correlacionados instantaneamente, independente da distância. Isso permite ao computador quântico realizar operações sobre múltiplos qubits de forma não local — ampliando o espaço de estados explorado.

Algoritmos Quânticos Fundamentais

Algoritmo de Shor (1994)

Fatoração de inteiros grandes em tempo polinomial O(log³ N) — comparado a sub-exponencial O(exp(N^(1/3))) nos melhores algoritmos clássicos.

Impacto: um computador quântico suficientemente grande (milhares de qubits lógicos sem erro) pode quebrar RSA-2048 em horas. Isso tornou a criptografia pós-quântica (Post-Quantum Cryptography — PQC) uma prioridade global para o NIST.

Algoritmo de Grover (1996)

Busca não estruturada em banco de dados de N elementos em O(√N) — comparado a O(N) clássico. Quadrático, não exponencial — por isso não quebra AES completamente, apenas dobra o tamanho de chave necessário (AES-128 → efetivamente AES-64 para Grover → requer AES-256).

Simulação Quântica

Feynman (1982) propôs usar computadores quânticos para simular sistemas quânticos. Aplicações imediatas: design de superfármacos, catálise para energia limpa (ex: fixação de nitrogênio em temperatura ambiente), materiais supercondutores.

QAOA e VQE

Algoritmos variacionais híbridos (quântico + clássico) para otimização combinatória (QAOA) e química quântica (VQE). São candidatos à vantagem quântica de médio prazo — antes de termos qubits lógicos perfeitos.

QML — Quantum Machine Learning

Algoritmos para acelerar ML em certas tarefas (multiplicação de matrizes, estimação de distâncias). Ainda em debate se oferece vantagem sobre ML clássico em problemas práticos.

Circuito de Bell — Emaranhamento com Qiskit (Python)
from qiskit import QuantumCircuit
from qiskit.visualization import plot_histogram
from qiskit_aer import AerSimulator

# Criar circuito com 2 qubits e 2 bits clássicos
qc = QuantumCircuit(2, 2)

# Hadamard no qubit 0 → superposição
qc.h(0)

# CNOT → emaranhamento (estado de Bell |Φ+⟩)
qc.cx(0, 1)

# Medir ambos os qubits
qc.measure([0, 1], [0, 1])

# Simular 1024 execuções
simulator = AerSimulator()
result = simulator.run(qc, shots=1024).result()
counts = result.get_counts()
print(counts)  # → {'00': ~512, '11': ~512} — nunca 01 ou 10!

Hardware Quântico

Tecnologias de Qubit

  • Supercondutor (IBM, Google): qubits em circuitos supercondutores a ~15 mK (mais frio que o espaço exterior). Alta velocidade, difícil escalamento por coerência.
  • Íons Aprisionados (IonQ, Quantinuum): átomos ionizados suspensos por campos magnéticos. Alta fidelidade, mais lentos, mais fáceis de conectar.
  • Fotônico (PsiQuantum, Xanadu): qubits de fótons. Opera em temperatura ambiente, difícil de escalar com baixo erro.
  • Neutro Atômico (Atom Computing, QuEra): átomos neutros em matrizes de tweezer laser. Potencial para escalar para milhares de qubits físicos.

Estado Atual (2025)

  • IBM Heron (2024): 133 qubits, melhor fidelidade de porta 2-qubit até hoje na linha IBM.
  • Google Willow (2024): 105 qubits; demonstrou escalonamento subexponencial de erro com correção quântica de erros.
  • IonQ Forte: 36 qubits #AQ (métrica de qualidade efectiva) — hardware de íons mais avançado comercialmente.

Barreira: Correção de Erros

Qubits físicos têm alta taxa de erro (~0,1-1%). Um qubit lógico tolerante a falhas requer ~1.000 qubits físicos. Para Shor em RSA-2048 são necessários ~4.000 qubits lógicos = ~4 milhões de qubits físicos. Ainda estamos na era NISQ (Noisy Intermediate-Scale Quantum).
