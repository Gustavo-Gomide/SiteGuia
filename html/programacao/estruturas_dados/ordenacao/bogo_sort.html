Bogo Sort

Bogo Sort: embaralhe e torça pela sorte

O algoritmo de ordenação humorístico que embaralha o array aleatoriamente até acidentalmente estar ordenado. Não use em produção. Jamais.

O Bogo Sort (também chamado de Stupid Sort, Monkey Sort ou Shotgun Sort) é o epítome do não-algoritmo: verifique se está ordenado; se não estiver, embaralhe aleatoriamente e tente de novo. Sua complexidade de tempo esperada é O((n+1)!) — para n=10 são em média ~3.6 milhões de tentativas. Para n=20 são ~2.4 quintilhões. Existe puramente para fins didáticos e humorísticos, para ilustrar o quanto um algoritmo inteligente vale.

O que é o Bogo Sort?

Inspirado no Teorema do Macaco Infinito: se um macaco bater aleatoriamente em uma máquina de escrever por tempo suficiente, eventualmente produzirá as obras completas de Shakespeare. O Bogo Sort é a versão de ordenação: se embaralharmos aleatoriamente por tempo suficiente, eventualmente obteremos a ordem correta.

Existem variantes como o Bozo Sort (troca dois elementos aleatórios em vez de embaralhar tudo) e o hilarante Quantum Bogo Sort (que assume que em algum ramo do universo quântico o array está ordenado após um embaralhamento, e destrói todos os outros ramos — O(n) garantido, mas inaplicável com tecnologia atual).

Aviso: O Bogo Sort não tem garantia de terminar em tempo finito determinístico. Com azar perpétuo (probabilidade zero, mas não impossível), pode executar para sempre. Em prática, para n > 8 é completamente impraticável.

Como Funciona: Passo a Passo

Tentando ordenar [3, 1, 2] — 3! = 6 permutações possíveis. A chance de acertar em cada tentativa é 1/6 ≈ 17%.

Implementação em Python

bogo_sort.py
import random


def esta_ordenado(arr: list) -> bool:
    return all(arr[i] <= arr[i + 1] for i in range(len(arr) - 1))


def bogo_sort(arr: list) -> list:
    tentativas = 0
    while not esta_ordenado(arr):
        random.shuffle(arr)
        tentativas += 1
    print(f"Ordenado em {tentativas} tentativas")
    return arr


# Variante determinística (mais lenta): permutação sistemática
from itertools import permutations

def bogo_sort_deterministic(arr: list) -> list:
    """Testa todas as permutações até encontrar a ordenada."""
    for perm in permutations(arr):
        if esta_ordenado(list(perm)):
            return list(perm)


if __name__ == "__main__":
    dados = [3, 1, 2]
    print(bogo_sort(dados.copy()))   # resultado imprevisível de tentativas

Valor Didático

O que o Bogo Sort ensina

Ilustra perfeitamente por que algoritmos importam. Um algoritmo O(n log n) como o Merge Sort para n=1000 leva ~10.000 operações. O Bogo Sort leva ~1000! operações — um número com 2.568 dígitos.

Conexão com probabilidade

O número esperado de tentativas é e·n! (base do logaritmo natural × n fatorial). Em cada tentativa a probabilidade de acertar é 1/n!, então a distribuição de tentativas segue uma distribuição geométrica.

Nunca use: Bogo Sort não tem garantia de terminar (probabilisticamente pode rodar para sempre). Tempo esperado O((n+1)!) torna impossível ordenar até mesmo 20 elementos em tempo hábil. É exclusivamente um exemplo humorístico e didático.
