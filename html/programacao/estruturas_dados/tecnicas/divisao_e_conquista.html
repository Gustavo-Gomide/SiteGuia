Divisão e Conquista

Dividir, Conquistar e Combinar

Quebrar o problema em partes menores, resolver recursivamente e fundir as soluções.

Divisão e Conquista é o paradigma recursivo que substituiu algoritmos quadráticos por soluções em O(N log N). Cada problema é partido até atingir um caso-base trivial; as soluções parciais são combinadas de forma eficiente para produzir o resultado final.

O que é?

Um paradigma de projeto de algoritmos baseado em três passos recursivos: dividir o problema em subproblemas menores e independentes; conquistar cada subproblema recursivamente (caso base quando trivial); combinar as soluções parciais para obter a solução do problema original.

Segundo Cormen et al. (2022), a chave para a eficiência está na independência dos subproblemas: ao contrário da Programação Dinâmica, aqui os subproblemas não se sobrepõem — cada um é resolvido uma única vez.

A complexidade de Divisão e Conquista é analisada pelo Teorema Mestre: T(n) = a·T(n/b) + f(n), onde a é o número de subproblemas, n/b o tamanho de cada um e f(n) o custo da combinação.

Visualização — Merge Sort

O array [8, 3, 5, 1] é dividido recursivamente até subarrays unitários (caso-base). Na fase de subida, cada par é intercalado em ordem crescente.

Os Três Pilares

  1. Dividir: particione o problema em N subproblemas menores e independentes. No Merge Sort: corte o array ao meio — O(1).
  2. Conquistar: resolva recursivamente cada subproblema. Quando o tamanho for trivial (array de 1 elemento), retorne imediatamente.
  3. Combinar: funda as soluções parciais para produzir o resultado final. No Merge Sort: a função merge() intercala duas listas ordenadas em O(N).

Teorema Mestre

Para recorrências da forma:

T(n) = a · T(n/b) + f(n)

a = n.º de subproblemas, n/b = tamanho de cada um, f(n) = custo da combinação.
Merge Sort: a=2, b=2, f(n)=O(n) → T(n) = O(n log n).

Implementação — Merge Sort

merge_sort.py
def merge_sort(arr):
    """Ordena arr em ordem crescente usando Divisão e Conquista."""
    # 1. Caso Base (Conquista Trivial)
    if len(arr) <= 1:
        return arr

    # 2. Dividir ao meio
    meio = len(arr) // 2
    esq  = merge_sort(arr[:meio])   # conquista esquerda
    dir  = merge_sort(arr[meio:])   # conquista direita

    # 3. Combinar (Merge)
    return merge(esq, dir)

def merge(esq, dir):
    """Intercala duas listas ordenadas — O(n)."""
    resultado, i, j = [], 0, 0
    while i < len(esq) and j < len(dir):
        if esq[i] <= dir[j]:
            resultado.append(esq[i]); i += 1
        else:
            resultado.append(dir[j]); j += 1
    return resultado + esq[i:] + dir[j:]

print(merge_sort([8, 3, 5, 1]))  # [1, 3, 5, 8]

Complexidade e Comparação

Característica Selection Sort (Força Bruta) Merge Sort (Divisão e Conquista)
Estratégia Varre tudo repetidamente Quebra em partes, combina
Complexidade de tempo O(N²) O(N log N)
Complexidade de espaço O(1) — in-place O(N) — array auxiliar
Estabilidade Instável Estável
