Algoritmos Gulosos

Escolha Localmente Ótima em Cada Passo

Pegar sempre a melhor opção imediata e confiar que o ótimo local leva ao global.

Um Algoritmo Guloso constrói a solução peça por peça, sempre escolhendo a opção de maior benefício imediato, sem reconsiderar escolhas feitas. Quando o problema tem escolha gulosa válida e subestrutura ótima, essa abordagem é correta e muito mais eficiente que Programação Dinâmica.

O que é?

Um algoritmo é guloso quando, em cada etapa de construção da solução, escolhe o candidato que parece melhor localmente, sem olhar para o futuro e sem desfazer escolhas anteriores. A solução é construida de forma incremental e irrevogável.

Segundo Cormen et al. (2022), algoritmos gulosos são mais simples de implementar que Programação Dinâmica e geralmente mais eficientes — mas só são corretos quando o problema tem a propriedade da escolha gulosa. Exemplos clássicos: Dijkstra, Huffman, Kruskal e Prim.

Algoritmos gulosos são mais rápidos que Programação Dinâmica, mas exigem prova formal de que a escolha local é sempre compatível com o ótimo global. Sem essa garantia, o guloso pode retornar soluções subótimas.

Quando Funciona?

Para um algoritmo guloso ser correto, o problema precisa de duas propriedades:

  • Propriedade da Escolha Gulosa: a escolha localmente ótima sempre faz parte de alguma solução ótima global — não precisamos do futuro para decidir.
  • Subestrutura Ótima: a solução ótima do problema contém soluções ótimas para cada subproblema (mesmo requisito da Prog. Din., mas sem sobreposição).

Problemas com solução gulosa correta: Huffman (compressão), Kruskal/Prim (MST), Dijkstra (caminhos em grafos com arestas não-negativas), escalonamento de intervalos.

Implementação — Troco e Escalonamento

guloso.py
def troco_guloso(valor: int, moedas: list) -> list:
    """Troco com menor número de moedas (moedas padrão)."""
    moedas = sorted(moedas, reverse=True)
    resultado = []
    for moeda in moedas:
        while valor >= moeda:
            resultado.append(moeda)
            valor -= moeda
    return resultado

# Moedas padrão → ótimo
print(troco_guloso(87, [1, 5, 10, 25, 50]))  # [50, 25, 10, 1, 1]
# Moedas arbitrárias → pode falhar!
print(troco_guloso(6,  [1, 3, 4]))          # [4, 1, 1] — não ótimo

# Escalonamento de tarefas por prazo (Guloso correto)
def escalonar(tarefas: list) -> list:
    """Seleciona tarefas por maior lucro dentro do prazo.
       tarefas: lista de (prazo, lucro).
    """
    tarefas.sort(key=lambda x: x[1], reverse=True)
    n_slots = max(p for p, _ in tarefas)
    slots   = [None] * (n_slots + 1)
    selecionadas = []
    for prazo, lucro in tarefas:
        for s in range(prazo, 0, -1):  # slot mais tardio disponível
            if slots[s] is None:
                slots[s] = lucro
                selecionadas.append(lucro)
                break
    return selecionadas
