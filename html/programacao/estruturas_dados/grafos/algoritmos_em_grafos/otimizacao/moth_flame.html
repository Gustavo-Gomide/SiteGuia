Moth-Flame Optimization (MFO)

Moth-Flame Optimization MFO (2015)

Baseado na navegação transversal das mariposas: usam a lua como guia fixo; chamas artificiais as confundem, criando espirais em torno de cada chama.

MFO (Mirjalili, 2015) usa dois conjuntos: mariposas (soluções candidatas) e chamas (melhores soluções arquivadas). Cada mariposa se move em espiral logarítmica em torno de sua chama associada: X_i = D_e^(bt)cos(2πt) + Chama_j, onde D é a distância à chama e b define a forma da espiral. O número de chamas decresce iterativamente forçando convergência.

Mecanismo de atualização

Cada mariposa tem uma chama associada (a melhor que ela já viu ou a melhor global). A espiral logarítmica permite explorar ao redor da chama com diferentes raios dependendo de t [1, 1]. Quando t = 1, mariposa está na chama; quando t = 1, está no ponto mais distante da espiral.

Estrutura em Python

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework genérico para Moth-Flame Optimization (MFO).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # decresce de 2 a 0
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a  # vetor A
            C = 2 * r2          # vetor C
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicações

Otimização contínua

Benchmarks de otimização e problemas de engenharia com espaço de busca contínuo e multimodal.

Feature selection e ML

Versões binárias de MFO são usadas para selecionar características em problemas de classificação.

Metaheurística: Moth-Flame Optimization não garante ótimo global. Resultados variam com os hiperparâmetros e inicialização.
