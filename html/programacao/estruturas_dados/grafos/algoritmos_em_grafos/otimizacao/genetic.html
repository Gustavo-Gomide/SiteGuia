Algoritmo Genético (GA)

Algoritmo Genético: evolução como otimização

Seleção natural, crossover genético e mutação aleatória simulados como um processo de otimização sobrevivem as soluções mais aptas.

Algoritmos Genéticos (Holland, 1975) modelam a evolução biológica: uma população de soluções candidatas evolui ao longo de gerações. Soluções mais aptas têm maior chance de reprodução (seleção); novos indivíduos são criados combinando partes dos pais (crossover); mutações aleatórias introduzem diversidade. Convergência para boas soluções sem garantia de ótimo global.

Os três operadores

Seleção: indivíduos mais aptos são escolhidos para reprodução (torneio, roleta, rank).
Crossover: dois pais combinam seus genes (ponto único, dois pontos, uniforme).
Mutação: pequena perturbação aleatória mantém diversidade e evita estagnação.

Implementação em Python

genetic.py
import numpy as np

def ga(func, n_bits, pop_size=50, geracoes=200, p_cross=0.8, p_mut=0.01):
    """Minimiza func(x) com x representado por n_bits bits."""
    pop = np.random.randint(0, 2, (pop_size, n_bits))
    def decode(ind): return int(''.join(map(str, ind)), 2)
    def fit(ind): return -func(decode(ind))  # negativo para maximizar
    def select(pop, fits):
        i, j = np.random.choice(len(pop), 2, replace=False)
        return pop[i] if fits[i] > fits[j] else pop[j]

    melhor = None; melhor_fit = -np.inf
    for _ in range(geracoes):
        fits = np.array([fit(ind) for ind in pop])
        idx = np.argmax(fits)
        if fits[idx] > melhor_fit: melhor, melhor_fit = pop[idx].copy(), fits[idx]
        nova_pop = []
        while len(nova_pop) < pop_size:
            p1, p2 = select(pop, fits), select(pop, fits)
            if np.random.rand() < p_cross:
                pt = np.random.randint(1, n_bits)
                c = np.concatenate([p1[:pt], p2[pt:]])
            else: c = p1.copy()
            mask = np.random.rand(n_bits) < p_mut
            c[mask] ^= 1
            nova_pop.append(c)
        pop = np.array(nova_pop)
    return decode(melhor), -melhor_fit

Aplicações

Otimização de parâmetros

Ajuste automático de hiperparâmetros de modelos de ML GA é um dos métodos clássicos de AutoML.

Design de estruturas

Engenharia evolutiva: projeto de antenas, estruturas mecânicas e circuitos usando GA para explorar o espaço de design.

Problemas combinatórios

TSP, escalonamento, coloração de grafos GA com representações adequadas (permutações, grafos) obtém boas soluções.

Pré-maturidade: GA pode convergir prematuramente para ótimos locais se a pressão seletiva for alta demais ou a taxa de mutação muito baixa. Diversidade da população é essencial.
