Bee Colony Optimization

Bee Colony Optimization BCO

Abelhas buscadoras exploram o ambiente; as que encontram boas fontes dançam para recrutar outras abelhas para o mesmo local. Quanto melhor a fonte, mais abelhas recrutadas.

BCO (Teodorovic, 2003) modela abelhas Exploradoras (buscam novos locais aleatoriamente), Buscadoras (exploram vizinhança de bons locais) e Espectadoras (são recrutadas pelas melhores via dança). A divisão de trabalho produz balanceamento entre exploração e explotação.

Princípio fundamental

As abelhas buscadoras exploram vizinhança via modificações locais da solução. Espectadoras são recrutadas proporcionalmente à qualidade (fitness) da fonte. Buscadoras abandonadas se tornam novas exploradoras evitando estagnação.

Implementação em Python

bee_colony.py
import numpy as np

def abc_optimizer(func, bounds, pop=30, limit=20, iters=200):
    '''Artificial Bee Colony para funções contínuas.'''
    D = len(bounds)
    lo, hi = np.array([b[0] for b in bounds]), np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    trial = np.zeros(pop, dtype=int)
    best = X[np.argmin(fit)].copy()
    for _ in range(iters):
        # Fase empregada
        for i in range(pop):
            j = np.random.randint(D); k = np.random.choice([x for x in range(pop) if x != i])
            v = X[i].copy(); v[j] = X[i, j] + np.random.uniform(-1,1) * (X[i, j] - X[k, j])
            v = np.clip(v, lo, hi); fv = func(v)
            if fv < fit[i]: X[i] = v; fit[i] = fv; trial[i] = 0
            else: trial[i] += 1
        # Fase espectadora (proporcional ao fitness)
        probs = 1 / (1 + fit); probs /= probs.sum()
        for _ in range(pop):
            i = np.random.choice(pop, p=probs); j = np.random.randint(D)
            k = np.random.choice([x for x in range(pop) if x != i])
            v = X[i].copy(); v[j] = X[i,j] + np.random.uniform(-1,1) * (X[i,j] - X[k,j])
            v = np.clip(v, lo, hi); fv = func(v)
            if fv < fit[i]: X[i] = v; fit[i] = fv; trial[i] = 0
        # Fase exploradora
        for i in range(pop):
            if trial[i] > limit: X[i] = lo + np.random.rand(D) * (hi - lo); fit[i] = func(X[i]); trial[i] = 0
        if fit.min() < func(best): best = X[np.argmin(fit)].copy()
    return best, func(best)

Aplicações

Otimização contínua

BCO se destaca em funções contínuas multimodais o balanço explorador/espectador evita convergência prematura.

Problemas combinatórios

Versões discretas aplicam BCO a TSP, escalonamento e alocação de recursos.

Sem garantia de otimalidade: BCO é uma metaheurística sem garantia de ótimo global. Ajuste dos parâmetros limit (abandono) e tamanho da colônia é crítico para convergência.
