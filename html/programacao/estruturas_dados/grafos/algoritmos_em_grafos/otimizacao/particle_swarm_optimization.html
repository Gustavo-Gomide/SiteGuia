PSO Otimização por Enxame de Partículas

Particle Swarm Optimization

Partículas voam pelo espaço de busca, atraídas pela melhor posição que cada uma já visitou e pela melhor posição já vista pelo enxame.

PSO (Kennedy & Eberhart, 1995) simula o comportamento de um bando de pássaros. Cada partícula tem posição x e velocidade v. A velocidade é atualizada por: v wv + cr(pbestx) + cr(gbestx), onde pbest é o melhor pessoal e gbest é o melhor global. Simples de implementar e muito eficaz em otimização contínua.

Componentes da atualização

w (inércia): mantém a direção atual alto w favorece exploração, baixo w favorece convergência.
c (cognitivo): atração ao melhor pessoal.
c (social): atração ao melhor global do enxame.

Implementação em Python

particle_swarm_optimization.py
import numpy as np

def pso(func, bounds, n=30, iters=200, w=0.7, c1=1.5, c2=1.5):
    D = len(bounds)
    lo, hi = np.array([b[0] for b in bounds]), np.array([b[1] for b in bounds])
    X = lo + np.random.rand(n, D) * (hi - lo)
    V = np.zeros((n, D))
    pbest, pbest_f = X.copy(), np.array([func(x) for x in X])
    gbest = pbest[np.argmin(pbest_f)].copy()
    gbest_f = pbest_f.min()

    for _ in range(iters):
        r1, r2 = np.random.rand(n, D), np.random.rand(n, D)
        V = w * V + c1 * r1 * (pbest - X) + c2 * r2 * (gbest - X)
        X = np.clip(X + V, lo, hi)
        f = np.array([func(x) for x in X])
        melhora = f < pbest_f
        pbest[melhora] = X[melhora]; pbest_f[melhora] = f[melhora]
        idx = np.argmin(pbest_f)
        if pbest_f[idx] < gbest_f: gbest, gbest_f = pbest[idx].copy(), pbest_f[idx]
    return gbest, gbest_f

# Exemplo: mínimo de Rastrigin
def rastrigin(x): return 10*len(x) + sum(xi**2 - 10*__import__('math').cos(2*3.14159*xi) for xi in x)
sol, val = pso(rastrigin, [(-5.12, 5.12)] * 2)
print(round(val, 4))  #  0

Aplicações

Treinamento de redes neurais

PSO foi usado para treinar redes neurais antes do surgimento do backpropagation moderno ainda útil quando gradientes não estão disponíveis.

Engenharia e design

Otimização de parâmetros de controladores PID, design de antenas, calibração de modelos de simulação.

Convergência prematura: PSO pode convergir antes de explorar bem o espaço. Aumentar w ou usar variantes como IPSO (inércia adaptativa) ajuda a mitigar isso.

PSO é notável por sua simplicidade apenas duas equações de atualização e eficácia em funções contínuas multimodais. É uma das metaheurísticas mais citadas da literatura, com milhares de variantes.
