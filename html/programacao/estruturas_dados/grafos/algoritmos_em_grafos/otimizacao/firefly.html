Firefly Algorithm (FA)

Firefly Algorithm FA (2010)

Inspirado em vagalumes: atraem parceiros com flashes de luz; vagalumes mais brilhantes (soluções melhores) atraem os menos brilhantes.

FA (Yang, 2010) modela a atratividade entre vagalumes: β(r) = β e^(γr²), onde r é a distância entre dois vagalumes. Um vagalume i é atraído ao vagalume j se j é mais brilhante: x_i += β(r_ij)(x_j x_i) + αε_i. γ controla a absorção da luz alto γ favorece exploração local; baixo γ favorece global.

Atualização de posição

O brilho de cada vagalume é proporcional à qualidade da solução (inverso do custo). Vagalumes menos brilhantes movem-se em direção aos mais brilhantes. α é o passo aleatório (exploração). O decaimento gaussiano da atratividade cria nichos naturais no espaço de busca.

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework genérico de enxame para Firefly Algorithm.
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy()
    for t in range(iters):
        a = 2 - t * (2 / iters)  # parâmetro decrescente
        for i in range(pop):
            # Atualização específica de FA (simplificada)
            r = np.random.rand(D)
            X[i] = X[i] + a * r * (best - X[i])
            X[i] = np.clip(X[i], lo, hi)
            fi = func(X[i])
            if fi < fit[i]: fit[i] = fi
            if fi < func(best): best = X[i].copy()
    return best, func(best)

Aplicações

Otimização contínua multimodal

Funções de benchmark (Rastrigin, Rosenbrock, Ackley) e problemas de engenharia onde gradientes não estão disponíveis.

Ajuste de hiperparâmetros

Calibração de modelos de ML, redes neurais e controladores quando o espaço de busca é contínuo e multimodal.

Metaheurística: Firefly Algorithm não garante ótimo global. Resultados dependem fortemente dos hiperparâmetros e do número de iterações.
