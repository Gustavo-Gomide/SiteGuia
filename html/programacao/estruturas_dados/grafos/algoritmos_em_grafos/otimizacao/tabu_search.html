Tabu Search: busca local com memória

Escapa de ótimos locais proibindo revisitar soluções recentes a lista tabu é a memória de curto prazo que guia a busca para regiões inexploradas.

Tabu Search (Glover, 1986) aprimora a busca local adicionando memória: uma lista tabu registra as últimas k movimentações (ou soluções) e as proíbe por um número de iterações. Isso força o algoritmo a explorar novas regiões mesmo que piores no momento permitindo escapar de ótimos locais sistematicamente.

Lista tabu e critério de aspiração

A lista tabu proíbe reverter movimenta recentes. O critério de aspiração permite ignorar a proibição se o movimento produz uma solução melhor que o melhor já visto evitando que bons movimentos sejam bloqueados por culpa tabu.

Implementação em Python

tabu_search.py
from collections import deque

def tabu_search(func, vizinhos_fn, x0, tabu_size=10, iteracoes=200):
    """func = objetivo (minimizar).
    vizinhos_fn(x) = lista de (vizinho, move) candidatos.
    tabu_size = tamanho da lista tabu."""
    x, fx = x0, func(x0)
    melhor, melhor_f = x, fx
    tabu = deque(maxlen=tabu_size)

    for _ in lambda vm: func(vm[0]))
        for xn, move in candidatos:
            fxn = func(xn)
            # Aceita se não é tabu OU critério de aspiração satisfeito
            if move not in tabu or fxn < melhor_f:
                x, fx = xn, fxn; tabu.append(move)
                if fx < melhor_f: melhor, melhor_f = x, fx
                break
    return melhor, melhor_f

# Exemplo: minimizar f(x) = (x-3)² em inteiros
def f(x): return (x - 3) ** 2
def viz(x): return [(x + d, d) for d in [-2,-1,1,2]]
sol, val = tabu_search(f, viz, 10)
print(sol, val)  # 3, 0

Aplicações

Escalonamento

Job-shop scheduling, alocação de turnos Tabu Search é um dos melhores algoritmos para problemas de escalonamento.

TSP e VRP

Com estrutura de vizinhança baseada em 2-opt ou 3-opt, Tabu Search obtém soluções excelentes para problemas de roteamento.

Tabu size: Lista curta ciclos; lista longa exploração pobre. O tamanho ótimo depende do problema e típicamente exige ajuste experimental.
