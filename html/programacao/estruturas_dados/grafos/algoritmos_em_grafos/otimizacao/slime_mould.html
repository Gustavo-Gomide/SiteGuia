Slime Mould Algorithm (SMA)

Slime Mould Algorithm - SMA (2020)

Baseado na formacao de redes otimas do mofo-mucilaginoso Physarum polycephalum ao conectar fontes de alimento via oscilacoes biovenas.

SMA (Li et al., 2020) usa peso W para ponderar posicoes com base no rank de aptidao: W_i = 1 + r*log((best_f-fit_i)/(best_f-worst_f)+1) para melhores metades, 1 - r*log(-) para piores. Atualizacao: X_A(t)*X_B(t)-X_C(t) ou best(t) com probabilidades vb e vc que oscilam, simulando contracao e extensao das veias.

Mecanismo de atualizacao

O peso biovena W cria pressao positiva (melhores solucoes) e negativa (piores). vb = tanh|fitness_i - bestFitness| e vc oscillates de -1 a 1. Quando vb e grande, o individuo e puxado para a melhor solucao; quando vc domina, ha diversificacao. A oscilacao imita a biocontracao do mofo real.

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework generico para Slime Mould Algorithm (SMA).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # fator de escalonamento
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a; C = 2 * r2
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicacoes

Otimizacao continua

Benchmarks de otimizacao e problemas de engenharia com espaco de busca continuo e multimodal.

Feature selection e ML

Versoes binarias de SMA sao usadas para selecionar caracteristicas em problemas de classificacao.

Metaheuristica: Slime Mould Algorithm nao garante otimo global. Resultados variam com os hiperparametros e inicializacao.
