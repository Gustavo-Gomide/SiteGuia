Whale Optimization Algorithm (WOA)

Whale Optimization Algorithm - WOA (2016)

Baseado no comportamento de alimentacao em rede de bolhas das baleias jubarte: espiral ascendente para cercar cardumes de peixes pequenos.

WOA (Mirjalili e Lewis, 2016) modela tres estrategias: (1) Cercamento encolhente: X = X* - A*D, com A decrescendo; (2) Ataque em espiral: X = D'*e^(bl)*cos(2*pi*l) + X*, onde l in [-1,1] e b define a forma; (3) Busca aleatoria quando |A|>=1. A cada iteracao, p~Uniform(0,1): se p < 0.5 usa cercamento ou busca aleatoria; se p >= 0.5 usa espiral.

Mecanismo de atualizacao

A transicao entre espiral e cercamento e probabilistica (50/50 por iteracao), o que cria busca genuinamente diversa. O parametro a decresce de 2 a 0 ao longo das iteracoes, fazendo o balanco exploracao-explotacao ser automaticamente gerenciado. WOA e um dos algoritmos bio-inspirados mais citados e reimplementados.

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework generico para Whale Optimization Algorithm (WOA).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # fator de escalonamento
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a; C = 2 * r2
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicacoes

Otimizacao continua

Benchmarks de otimizacao e problemas de engenharia com espaco de busca continuo e multimodal.

Feature selection e ML

Versoes binarias de WOA sao usadas para selecionar caracteristicas em problemas de classificacao.

Metaheuristica: Whale Optimization Algorithm nao garante otimo global. Resultados variam com os hiperparametros e inicializacao.
