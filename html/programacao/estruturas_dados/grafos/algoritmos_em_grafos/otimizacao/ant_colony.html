Ant Colony Optimization

Formigas deixam feromônios nos melhores caminhos e o feromônio guia as próximas formigas. Inteligência coletiva aplicada à otimização.

ACO (Dorigo, 1992) é inspirado no comportamento de formigas reais: ao encontrar comida, depositam feromônio no caminho; caminhos mais curtos acumulam mais feromônio; formigas futuras os preferem. Matematicamente, cada aresta (i,j) tem feromônio τᵢⱼ e heurística ηᵢⱼ = 1/d(i,j). A probabilidade de escolher (i,j) é proporcional a τᵢⱼᵅ ηᵢⱼᵝ.

Feromônio e evaporação

Após cada iteração, o feromônio evapora: τᵢⱼ (1ρ)τᵢⱼ. As melhores formigas reforçam seus caminhos: τᵢⱼ τᵢⱼ + Δτ (proporcional à qualidade da solução). Evaporação evita convergência prematura para ótimos locais.

Implementação em Python (TSP)

ant_colony.py
import numpy as np

def aco_tsp(dist, n_formigas=20, iteracoes=100, alfa=1, beta=2, rho=0.5):
    n = len(dist)
    tau = np.ones((n, n))
    eta = 1.0 / (dist + np.eye(n))  # evita divisão por zero
    melhor, melhor_custo = None, float('inf')

    for _ in range(iteracoes):
        todos_caminhos, todos_custos = [], []
        for _ in range(n_formigas):
            caminho = [int(np.random.randint(n))]
            for __ in range(n - 1):
                i = caminho[-1]
                prob = tau[i] ** alfa * eta[i] ** beta
                prob[[c for c in caminho]] = 0
                prob /= prob.sum()
                caminho.append(int(np.random.choice(n, p=prob)))
            custo = sum(dist[caminho[k], caminho[k+1]] for k in range(n-1)) + dist[caminho[-1], caminho[0]]
            todos_caminhos.append(caminho); todos_custos.append(custo)
            if custo < melhor_custo: melhor, melhor_custo = caminho[:], custo
        # Evaporação
        tau *= (1 - rho)
        for caminho, custo in zip(todos_caminhos, todos_custos):
            for k in range(n): tau[caminho[k], caminho[(k+1) % n]] += 1 / custo
    return melhor, melhor_custo

Aplicações

TSP e VRP

ACO é historicamente um dos melhores algoritmos para o Problema do Caixeiro Viajante e variantes de roteamento.

Roteamento em redes

Usado em roteamento dinâmico de pacotes em redes de telecomunicações feromônio digital.

Sem garantia: ACO é uma metaheurística não garante o ótimo global. Ajuste cuidadoso de α, β e ρ é essencial para bons resultados.
