Dragonfly Algorithm (DA)

Dragonfly Algorithm DA (2016)

Inspirado no comportamento coletivo de libélulas: ataque a presas, fuga de predadores, atração a alimentos, desvio de colisão e coesão do grupo.

DA (Mirjalili, 2016) modela cinco comportamentos de libélulas representados por vetores: separação s, alinhamento a, coesão c, atração para alimento f e distância de inimigo e. A velocidade é: ΔX = wsS + waA + wcC + wfF + weE + wΔX_anterior. Equilibra exploração e explotação via pesos adaptativos.

Atualização de posição

Cada componente é calculada com base nas posições dos vizinhos (separação: evitar), da média do grupo (coesão, alinhamento), do melhor global (atração) e do pior (repulsão). Raio de vizinhança cresce iterativamente começa restrito (explotação) e aumenta (exploração).

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework genérico de enxame para Dragonfly Algorithm.
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy()
    for t in range(iters):
        a = 2 - t * (2 / iters)  # parâmetro decrescente
        for i in range(pop):
            # Atualização específica de DA (simplificada)
            r = np.random.rand(D)
            X[i] = X[i] + a * r * (best - X[i])
            X[i] = np.clip(X[i], lo, hi)
            fi = func(X[i])
            if fi < fit[i]: fit[i] = fi
            if fi < func(best): best = X[i].copy()
    return best, func(best)

Aplicações

Otimização contínua multimodal

Funções de benchmark (Rastrigin, Rosenbrock, Ackley) e problemas de engenharia onde gradientes não estão disponíveis.

Ajuste de hiperparâmetros

Calibração de modelos de ML, redes neurais e controladores quando o espaço de busca é contínuo e multimodal.

Metaheurística: Dragonfly Algorithm não garante ótimo global. Resultados dependem fortemente dos hiperparâmetros e do número de iterações.
