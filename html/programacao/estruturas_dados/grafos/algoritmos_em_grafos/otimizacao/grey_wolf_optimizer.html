Grey Wolf Optimizer (GWO)

Grey Wolf Optimizer GWO (2014)

Baseado na hierarquia de caça dos lobos-cinzentos: alfa, beta, delta e ômega. Os três melhores lobos guiam a posição dos demais.

GWO (Mirjalili et al., 2014) classifica as soluções em α (melhor), β (2º), δ (3º) e ω (restante). As posições α, β e δ estimam a localização da presa; cada lobo ω atualiza sua posição em relação a elas: X = (X+X+X)/3, onde X,X,X são posições calculadas a partir de α, β e δ. a decresce linearmente de 2 a 0 de exploração para explotação.

Mecanismo de atualização

Cada lobo calcula três posições candidatas usando α, β e δ como guias, e move-se para a média. Com a grande exploração; a pequeno convergência. A hierarquia natural do bando faz o algoritmo manter diversidade sem parâmetro explícito de diversidade.

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework genérico para Grey Wolf Optimizer (GWO).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # decresce de 2 a 0
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a  # vetor A
            C = 2 * r2          # vetor C
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicações

Otimização contínua

Benchmarks de otimização e problemas de engenharia com espaço de busca contínuo e multimodal.

Feature selection e ML

Versões binárias de GWO são usadas para selecionar características em problemas de classificação.

Metaheurística: Grey Wolf Optimizer não garante ótimo global. Resultados variam com os hiperparâmetros e inicialização.
