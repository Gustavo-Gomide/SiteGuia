Simulated Annealing (SA)

Simulated Annealing: resfriamento como otimização

Inspirado no resfriamento de metais, SA aceita soluções piores com probabilidade decrescente escapando de ótimos locais enquanto "esfria" para o ótimo global.

Simulated Annealing (Kirkpatrick et al., 1983) é motivado pelo processo físico de annealing: aquecer e resfriar lentamente um metal para atingir configuração de energia mínima. Em otimização: exploração ampla com temperatura alta (aceita soluções piores), convergência com temperatura baixa (rejeita quase tudo). A probabilidade de aceitar uma solução pior ΔE é e^(ΔE/T).

Critério de Metropolis

Soluções melhores são sempre aceitas (ΔE < 0). Soluções piores são aceitas com probabilidade P = e^(ΔE/T). Com T0, SA se torna descida de gradiente estocástica. Com T alto, aceita tudo exploração pura.

Implementação em Python

simulated_annealing.py
import math, random

def simulated_annealing(func, vizinho, x0, T0=1000, alpha=0.99, T_min=1e-4, iter_max=10000):
    """func = função objetivo (minimizar).
    vizinho = função que gera um vizinho de x.
    x0 = solução inicial. alpha = taxa de resfriamento."""
    x, fx = x0, func(x0)
    T = T0
    melhor, melhor_f = x, fx
    for i in range(iter_max):
        if T < T_min: break
        xn = vizinho(x)
        fxn = func(xn)
        delta = fxn - fx
        if delta < 0 or random.random() < math.exp(-delta / T):
            x, fx = xn, fxn
            if fx < melhor_f: melhor, melhor_f = x, fx
        if i % 100 == 0: T *= alpha
    return melhor, melhor_f

# Exemplo: mínimo de f(x) = x² no intervalo [-10, 10]
def f(x): return x**2
def viz(x): return x + random.uniform(-1, 1)
sol, val = simulated_annealing(f, viz, 8.0)
print(round(sol, 4), round(val, 4))  #  0, 0

Aplicações

Placement de circuitos

SA foi originalmente aplicado ao problema de alocação de componentes em chips ainda usado hoje em ferramentas EDA.

TSP e escalonamento

A flexibilidade da função vizinho torna SA facilmente adaptável a qualquer problema combinatório.

Schedule de resfriamento: O cronograma de temperatura (alpha, T0) é crítico. Muito rápido ótimo local; muito lento custo proibitivo. Ajuste empírico é necessário.
