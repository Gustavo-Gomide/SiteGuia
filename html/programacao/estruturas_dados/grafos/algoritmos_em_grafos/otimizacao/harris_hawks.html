Harris Hawks Optimization (HHO)

Harris Hawks Optimization HHO (2019)

Baseado no comportamento de caça cooperativa dos gaviões-asa-larga: emboscada, cercamento progressivo e ataque em mergulho.

HHO (Heidari et al., 2019) modela fases de exploração (gaviões pousam em árvores aleatórias à espera de presas) e explotação (padrões de ataque: cercamento suave/rígido e salto em mergulho com movimento de Lévy). A transição entre fases depende da 'energia de escape' E, que diminui com as iterações alta energia presa escapa (exploração), baixa presa capturada (explotação).

Mecanismo de atualização

Fase de exploração: X(t+1) = X_coelho(t) ε|X_coelho(t) 2rX(t)| ou baseada na média da população. Fase de explotação: cercamento suave se |E|0.5, cercamento rígido se |E|<0.5; salto Lévy quando E é baixo e r<0.5.

Estrutura em Python

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework genérico para Harris Hawks Optimization (HHO).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # decresce de 2 a 0
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a  # vetor A
            C = 2 * r2          # vetor C
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicações

Otimização contínua

Benchmarks de otimização e problemas de engenharia com espaço de busca contínuo e multimodal.

Feature selection e ML

Versões binárias de HHO são usadas para selecionar características em problemas de classificação.

Metaheurística: Harris Hawks Optimization não garante ótimo global. Resultados variam com os hiperparâmetros e inicialização.
