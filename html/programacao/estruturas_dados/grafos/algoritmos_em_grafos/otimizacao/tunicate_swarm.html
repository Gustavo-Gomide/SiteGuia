Tunicate Swarm Algorithm (TSA)

Tunicate Swarm Algorithm - TSA (2020)

Baseado no comportamento de locomocao por propulsao a jato e formacao de cardume das tunicatas marinhas.

TSA (Kaur et al., 2020) combina dois mecanismos: propulsao a jato (busca individual) e comportamento de cardume (busca social). Na fase de propulsao, o individuo se move em direcao ao melhor com forca determinada por um vetor A. Na fase de cardume, a posicao e media ponderada entre a atual e a do vizinho mais proximo. Cada iteracao aplica ambos: X = (|A*best - D*X_t| + best) / 2.

Mecanismo de atualizacao

O fator A = 2*r - 1 e D = |best - X_t|. Se r < 0.5, o individuo usa propulsao para se aproximar do lider; caso contrario, usa media de cardume. A combinacao das duas fases garante tanto exploracao (diversidade) quanto explotacao (convergencia).

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework generico para Tunicate Swarm Algorithm (TSA).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # fator de escalonamento
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a; C = 2 * r2
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicacoes

Otimizacao continua

Benchmarks de otimizacao e problemas de engenharia com espaco de busca continuo e multimodal.

Feature selection e ML

Versoes binarias de TSA sao usadas para selecionar caracteristicas em problemas de classificacao.

Metaheuristica: Tunicate Swarm Algorithm nao garante otimo global. Resultados variam com os hiperparametros e inicializacao.
