Salp Swarm Algorithm (SSA)

Salp Swarm Algorithm - SSA (2017)

Baseado no comportamento de cadeias de salpa no oceano: a lider acompanha a fonte de alimento e os seguidores se deslocam em fila.

SSA (Mirjalili et al., 2017) divide a populacao em lider (salpa fronteira) e seguidores. A lider atualiza posicao relativa ao melhor alimento: x1_j = F_j +/- c1*((ub-lb)*c2+lb), onde c1 = 2*exp(-(4t/T)^2) equilibra exploracao e explotacao. Seguidores usam media aritmetica com o anterior: x_i = (x_i + x_{i-1})/2.

Mecanismo de atualizacao

c1 decresce com o tempo inicio com grande c1 (exploracao ampla), final com c1 pequeno (explotacao fina). A estrutura de cadeia propaga informacao do lider para os seguidores de forma gradual. A fonte de alimento e atualizada sempre que um individuo encontra posicao melhor.

import numpy as np

def optimize(func, bounds, pop=30, iters=200):
    """Framework generico para Salp Swarm Algorithm (SSA).
    func = objetivo (minimizar), bounds = [(lo,hi)]*D."""
    D = len(bounds)
    lo = np.array([b[0] for b in bounds])
    hi = np.array([b[1] for b in bounds])
    X = lo + np.random.rand(pop, D) * (hi - lo)
    fit = np.array([func(x) for x in X])
    best = X[np.argmin(fit)].copy(); best_f = fit.min()
    for t in range(iters):
        a = 2 * (1 - t / iters)  # fator de escalonamento
        for i in range(pop):
            r1, r2 = np.random.rand(D), np.random.rand(D)
            A = 2 * a * r1 - a; C = 2 * r2
            D_vec = abs(C * best - X[i])
            X[i] = np.clip(best - A * D_vec, lo, hi)
            fi = func(X[i]); fit[i] = fi
            if fi < best_f: best, best_f = X[i].copy(), fi
    return best, best_f

Aplicacoes

Otimizacao continua

Benchmarks de otimizacao e problemas de engenharia com espaco de busca continuo e multimodal.

Feature selection e ML

Versoes binarias de SSA sao usadas para selecionar caracteristicas em problemas de classificacao.

Metaheuristica: Salp Swarm Algorithm nao garante otimo global. Resultados variam com os hiperparametros e inicializacao.
