Algoritmo de Johnson

Johnson: todos os pares com pesos negativos

O melhor algoritmo para todos os pares em grafos esparsos com pesos negativos combina Bellman-Ford uma vez com Dijkstra V vezes.

O algoritmo de Johnson resolve todos os pares de caminhos mínimos em grafos com (possivelmente) pesos negativos usando o truque de reponderação (reweighting): executa Bellman-Ford uma vez para calcular uma função de potencial h que elimina todos os pesos negativos, depois executa Dijkstra de cada vértice. Complexidade: O(V² log V + VE) melhor que Floyd-Warshall em grafos esparsos.

A ideia da reponderação

Passo 1: Adicionar vértice extra q com arestas de peso 0 para todos os vértices. Executar Bellman-Ford de q obter h[v] = menor distância de q a v.
Passo 2: Repoderar cada aresta (u,v,w) para w' = w + h[u] - h[v] 0.
Passo 3: Executar Dijkstra de cada vértice com os novos pesos. Corrigir o resultado: dist(u,v) = dijkstra_dist(u,v) - h[u] + h[v].

Implementação em Python

johnson.py
import heapq

def johnson(grafo, vertices):
    """grafo = {u: [(v, w)]}. O(V² log V + VE).
    Retorna matriz dist[u][v] = caminho mínimo."""
    # Adicionar vértice extra q
    q = '__q__'
    grafo[q] = [(v, 0) for v in vertices]
    todos = vertices + [q]

    # Bellman-Ford de q
    h = {v: float('inf') for v in todos}
    h[q] = 0
    for _ in range(len(todos) - 1):
        for u in grafo:
            for v, w in grafo[u]:
                if h[u] + w < h[v]: h[v] = h[u] + w
    del grafo[q]

    # Dijkstra de cada vértice com pesos ajustados
    def dijkstra(src):
        d = {v: float('inf') for v in vertices}
        d[src] = 0; heap = [(0, src)]
        while heap:
            dd, u = heapq.heappop(heap)
            if dd > d[u]: continue
            for v, w in grafo.get(u, []):
                w2 = w + h[u] - h[v]   # peso ajustado  0
                if d[u] + w2 < d[v]:
                    d[v] = d[u] + w2; heapq.heappush(heap, (d[v], v))
        return {v: d[v] - h[src] + h[v] for v in vertices}

    return {src: dijkstra(src) for src in vertices}

Aplicações

Redes esparsas complexas

Grafos grandes e esparsos (E O(V)) com alguns pesos negativos onde Floyd-Warshall seria lento demais.

Análise de fluxo financeiro

Redes de transações onde alguns fluxos têm "custo negativo" (benefício) Johnson permite análise global.

Atenção: Johnson, como todos os algoritmos de caminhos mínimos, falha na presença de ciclos negativos acessíveis. O Bellman-Ford inicial serve como detector desse caso se houver ciclo negativo, não há caminho mínimo bem definido.
