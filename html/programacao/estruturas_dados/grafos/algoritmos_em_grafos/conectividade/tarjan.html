Tarjan SCC

Algoritmo de Tarjan

Tarjan: SCCs em um único DFS

Encontrar todos os componentes fortemente conexos com um só passe de busca em profundidade usando discovery time e low-link para identificar raízes de SCC.

O algoritmo de Tarjan encontra todos os Componentes Fortemente Conexos (SCCs) de um dígrafo em O(V + E) com um único DFS. Cada vértice recebe um discovery time (quando foi visitado) e um low-link (o menor discovery time alcançável a partir de sua subárvore). Quando disc[v] == low[v], v é a raiz de um SCC e todos os vértices até ele na pilha formam esse componente.

Conceito: discovery time e low-link

disc[v]: timestamp em que v foi visitado pelo DFS.
low[v]: menor disc acessível de v usando arestas da árvore DFS + uma back edge.
Raiz do SCC: quando low[v] == disc[v], nenhuma back edge alcança um ancestral de v v é raiz do seu SCC.

Implementação em Python

tarjan.py
def tarjan(grafo, vertices):
    disc  = {}           # discovery time
    low   = {}           # low-link
    em_pilha = set()
    pilha = []
    timer = [0]          # mutável via lista
    sccs  = []

    def dfs(v):
        disc[v] = low[v] = timer[0]; timer[0] += 1
        pilha.append(v); em_pilha.add(v)

        for u in grafo.get(v, []):
            if u not in disc:
                dfs(u)
                low[v] = min(low[v], low[u])
            elif u in em_pilha:
                low[v] = min(low[v], disc[u])

        if low[v] == disc[v]:        # v é raiz do SCC
            scc = []
            while True:
                w = pilha.pop(); em_pilha.discard(w); scc.append(w)
                if w == v: break
            sccs.append(scc)

    for v in vertices:
        if v not in disc: dfs(v)

    return sccs

# Exemplo
g = {0:[1], 1:[2], 2:[0], 3:[2,4], 4:[3]}
print(tarjan(g, range(5)))
# [[2,1,0], [4,3]]  dois SCCs

Aplicações

Condensação de gráfos

Condensar SCCs em super-nós transforma qualquer dígrafo em um DAG habilitando ordenação topológica.

Detecção de dependências circulares

Build systems usam Tarjan para detectar ciclos de dependência e reportar grupos problemáticos.

Jogos e IA

Análise de estados de jogos identificar estados equivalentes (mutuamente alcançáveis) em grafos de estado.

Atenção: A versão recursiva de Tarjan pode causar stack overflow em Python para grafos com profundidade de DFS maior que ~1000. Para grafos grandes, converta para versão iterativa com pilha explícita simulando o frame de chamada.
