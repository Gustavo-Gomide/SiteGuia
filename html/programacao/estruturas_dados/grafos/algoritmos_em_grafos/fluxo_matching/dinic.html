Dinic: fluxo bloqueante em grafo de níveis

O algoritmo mais usado na prática para fluxo máximo: combina BFS para construir o grafo de níveis e DFS eficiente para encontrar o fluxo bloqueante.

O algoritmo de Dinic (1970) supera Edmonds-Karp ao enviar fluxo bloqueante de uma vez - em vez de um caminho por iteração. Constrói um grafo de níveis via BFS, depois satura-o com DFS usando ponteiros de arestas ("link-cut" style). Complexidade: O(V²E) geral e O(EV) para grafos unitários (bipartite matching).

Grafo de níveis

O grafo de níveis contém apenas arestas que avançam exatamente um nível (distância BFS) de s para t. O fluxo bloqueante satura pelo menos uma aresta de cada caminho de s a t nesse grafo, aumentando a distância mínima no próximo round garantindo no máximo O(V) rounds.

Implementação em Python

dinic.py
from collections import deque

class Dinic:
    def __init__(self, n):
        self.n = n; self.grafo = [[] for _ in range(n)]

    def add_edge(self, u, v, c):
        self.grafo[u].append([v, c, len(self.grafo[v])])
        self.grafo[v].append([u, 0, len(self.grafo[u]) - 1])

    def _bfs(self, s, t):
        self.nivel = [-1] * self.n; self.nivel[s] = 0
        fila = deque([s])
        while fila:
            u = fila.popleft()
            for v, c, _ in self.grafo[u]:
                if c > 0 and self.nivel[v] < 0:
                    self.nivel[v] = self.nivel[u] + 1; fila.append(v)
        return self.nivel[t] >= 0

    def _dfs(self, u, t, empurrado):
        if u == t: return empurrado
        while self.ptr[u] < len(self.grafo[u]):
            v, c, rev = self.grafo[u][self.ptr[u]]
            if c > 0 and self.nivel[v] == self.nivel[u] + 1:
                d = self._dfs(v, t, min(empurrado, c))
                if d > 0:
                    self.grafo[u][self.ptr[u]][1] -= d
                    self.grafo[v][rev][1] += d
                    return d
            self.ptr[u] += 1
        return 0

    def max_flow(self, s, t):
        total = 0
        while self._bfs(s, t):
            self.ptr = [0] * self.n
            while True:
                f = self._dfs(s, t, float('inf'))
                if f == 0: break
                total += f
        return total

# Exemplo
g = Dinic(4)
g.add_edge(0,1,10); g.add_edge(0,2,10)
g.add_edge(1,3,10); g.add_edge(2,3,10); g.add_edge(1,2,1)
print(g.max_flow(0, 3))  # 20

Complexidade

CasoComplexidade
GeralO(V²E)

Aplicações

Matching bipartido máximo

Dinic resolve matching bipartido em O(EV) melhor que Edmonds-Karp para grafos densos.

Problema de projeto

Dado um conjunto de projetos e recursos, escolher projetos lucrativos com corte mínimo de recursos reduzido a fluxo máximo.

Competições algorítmicas

Dinic é o algoritmo de fluxo padrão em competições por sua implementação simples e eficiência prática excepcional.

Detalhe crítico: Os ponteiros ptr[] (current arc) são o que tornam Dinic eficiente sem eles, o DFS pode re-explorar arestas saturadas, degradando para O(VE²).
