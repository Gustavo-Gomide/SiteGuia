Hopcroft-Karp: matching bipartido em O(EV)

O algoritmo ótimo para emparelhamento máximo em grafos bipartidos processa múltiplos caminhos aumentantes em paralelo a cada fase.

Hopcroft-Karp (1973) melhora o emparelhamento bipartido ingênuo de O(VE) para O(EV). A ideia: uma fase BFS encontra todos os caminhos aumentantes de menor comprimento simultaneamente; depois uma fase DFS aumenta todos eles de uma vez. Após O(V) fases, o emparelhamento é máximo.

Caminho aumentante

Um caminho aumentante alterna entre arestas não-casadas e arestas casadas, começando e terminando em vértices livres. Ao invertê-lo (trocar casadas não-casadas), o emparelhamento cresce em 1. Hopcroft-Karp encontra um conjunto máximo (disjunto) de caminhos mais curtos de uma vez.

Implementação em Python

hopcroft_karp.py
from collections import deque

def hopcroft_karp(U, V, adj):
    """U, V = conjuntos de vértices (0..U-1, 0..V-1).
    adj[u] = lista de vizinhos v."""
    INF = float('inf')
    mu = [-1] * U   # empar. de u para v
    mv = [-1] * V   # empar. de v para u
    dist = [0] * U

    def bfs():
        fila = deque()
        for u in range(U):
            if mu[u] == -1: dist[u] = 0; fila.append(u)
            else: dist[u] = INF
        encontrou = False
        while fila:
            u = fila.popleft()
            for v in adj[u]:
                w = mv[v]
                if w == -1: encontrou = True
                elif dist[w] == INF:
                    dist[w] = dist[u] + 1; fila.append(w)
        return encontrou

    def dfs(u):
        for v in adj[u]:
            w = mv[v]
            if w == -1 or (dist[w] == dist[u] + 1 and dfs(w)):
                mu[u] = v; mv[v] = u; return True
        dist[u] = INF; return False

    match = 0
    while bfs():
        for u in range(U):
            if mu[u] == -1 and dfs(u): match += 1
    return match, mu

# Exemplo: 3 trabalhadores, 3 tarefas
adj = [[0,1],[1,2],[0,2]]
m, empar = hopcroft_karp(3, 3, adj)
print(m)  # 3 (emparelhamento perfeito)

Aplicações

Atribuição de tarefas

Alocar trabalhadores a tarefas de forma que o número de atribuições seja máximo clássico problema de matching bipartido.

Cobertura de vértices

Pelo teorema de König, o emparelhamento máximo em grafos bipartidos tem tamanho igual ao da cobertura mínima de vértices.

Escopo: Hopcroft-Karp resolve emparelhamento máximo em grafos bipartidos. Para grafos gerais, use o algoritmo de Edmonds (Blossom) mais complexo.

Hopcroft-Karp é um dos algoritmos mais elegantes em teoria dos grafos: a combinação de BFS para encontrar todos os caminhos mínimos e DFS para aumentá-los simultaneamente é a síntese perfeita entre estrutura e eficiência.

"Encontre todos os caminhos mais curtos de uma vez e reduza O(V) fases a O(V)." Hopcroft & Karp, 1973
