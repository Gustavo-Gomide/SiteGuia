Ford-Fulkerson: fluxo máximo por caminhos aumentantes

O método fundamental de fluxo máximo: encontre um caminho de fonte a destino no grafo residual, aumente o fluxo, repita.

O método de Ford-Fulkerson (1956) é a base de todos os algoritmos de fluxo máximo. A ideia: enquanto existir um caminho de s a t no grafo residual, aumente o fluxo ao longo desse caminho pelo mínimo de capacidade disponível. O teorema max-flow min-cut garante que ao término, o fluxo é máximo.

Grafo residual

Para cada aresta (u,v) com capacidade c e fluxo f, o grafo residual contém: a aresta direta com capacidade residual c-f e uma aresta reversa (v,u) com capacidade f. A aresta reversa permite "desfazer" fluxo essencial para corrigir caminhos subótimos.

Implementação em Python (DFS)

ford_fulkerson.py
from collections import defaultdict

class FordFulkerson:
    def __init__(self, n):
        self.n = n
        self.cap = defaultdict(lambda: defaultdict(int))

    def add_edge(self, u, v, c):
        self.cap[u][v] += c  # aresta reversa começa em 0

    def _dfs(self, u, t, visitado, fluxo):
        if u == t: return fluxo
        visitado.add(u)
        for v in range(self.n):
            if v not in visitado and self.cap[u][v] > 0:
                f = self._dfs(v, t, visitado, min(fluxo, self.cap[u][v]))
                if f > 0:
                    self.cap[u][v] -= f; self.cap[v][u] += f
                    return f
        return 0

    def max_flow(self, s, t):
        total = 0
        while True:
            f = self._dfs(s, t, set(), float('inf'))
            if f == 0: break
            total += f
        return total

# Exemplo
g = FordFulkerson(4)
g.add_edge(0,1,10); g.add_edge(0,2,10)
g.add_edge(1,3,10); g.add_edge(2,3,10); g.add_edge(1,2,1)
print(g.max_flow(0, 3))  # 20

Aplicações

Redes de transporte

Fluxo máximo de mercadorias em redes com capacidades limitadas em cada trecho.

Bipartite matching

Todo problema de emparelhamento bipartido é reduzível a fluxo máximo adiciona-se fonte e destino com arestas de capacidade 1.

Corte mínimo

Pelo teorema max-flow min-cut, o valor do fluxo máximo equivale à capacidade do corte mínimo que separa s de t.

Atenção: Com capacidades irracionais, Ford-Fulkerson com DFS pode não convergir. Use sempre Edmonds-Karp (BFS) para garantir terminação polinomial.

Ford-Fulkerson não é apenas um algoritmo é um método. O teorema max-flow min-cut que ele prova conecta fluxo em redes a cortes de grafo, fundando uma das mais ricas teorias da otimização combinatória.
