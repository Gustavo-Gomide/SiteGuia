Componentes Conexos

Componentes Conexos: grupos de vértices alcançáveis

Identificar quais vértices fazem parte de quais grupos a operação fundamental para detectar sub-redes isoladas.

Em um grafo não-direcionado, um componente conexo é um subconjunto maximal de vértices onde existe caminho entre qualquer par. Se o grafo for completamente conectado, ele tem exatamente 1 componente. Se houver ilhas isoladas, cada ilha é um componente separado. Encontrar componentes conexos é O(V + E) via BFS ou DFS ou O(α(V)) por operação com Union-Find.

Visualização

Grafos direcionados têm componentes fortemente conexos (SCC) grupos onde qualquer vértice pode alcançar qualquer outro. Para isso use Kosaraju ou Tarjan.

Implementação em Python

componentes_conexos.py
from collections import deque, defaultdict

def componentes_conexos(grafo):
    """Retorna lista de componentes conexos (cada um é um set de vértices)."""
    visitado = set()
    componentes = []

    def bfs(inicio):
        comp, fila = set(), deque([inicio])
        visitado.add(inicio)
        while fila:
            v = fila.popleft()
            comp.add(v)
            for viz in grafo[v]:
                if viz not in visitado:
                    visitado.add(viz); fila.append(viz)
        return comp

    for v in grafo:
        if v not in visitado:
            componentes.append(bfs(v))

    return componentes

# Exemplo: dois componentes + vértice isolado
g = defaultdict(list)
for u, v in [('A','B'),('B','C'),('D','E')]:
    g[u].append(v); g[v].append(u)
g['F']  # vértice isolado
print(componentes_conexos(g))
# [{'A','B','C'}, {'D','E'}, {'F'}]

Aplicações

Detecção de ilhas

Em mapas de pixels, identificar regiões contíguas de mesma cor (flood fill).

Redes de computadores

Verificar se todos os hosts estão alcançáveis detectar sub-redes isoladas.

Análise de grafos sociais

Identificar grupos desconectados em redes sociais ou de colaboração.

Atenção: Para grafos direcionados, componentes conexos (alcançabilidade unilateral) são diferentes de componentes fortemente conexos (alcançabilidade bilateral). Não confunda use Kosaraju ou Tarjan para SCCs em dígrafos.
