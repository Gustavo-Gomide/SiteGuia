Rede de Hopfield

Hopfield: memória associativa como energia de grafo

Uma rede neural recorrente modelada como grafo pesado minimizar a energia da rede é equivalente a recuperar padrões armazenados.

A rede de Hopfield (1982) é um grafo completo não-direcionado onde cada nó é um neurônio (1) e as arestas têm pesos aprendidos. A função de energia de Lyapunov E = Σᵢⱼ wᵢⱼ sᵢ sⱼ nunca aumenta durante atualizações assíncronas garantindo convergência para um mínimo local (padrão armazenado).

Grafo de energia

Os pesos wᵢⱼ são aprendidos pela regra de Hebb: wᵢⱼ = (1/n) Σ sᵢᵏ sⱼᵏ (soma sobre padrões). A rede é então um grafo cujos mínimos de energia correspondem aos padrões aprendidos apresente um padrão corrompido e a rede converge para o original.

Implementação em Python

hopfield.py
import numpy as np

class Hopfield:
    def __init__(self, n):
        self.n = n; self.W = np.zeros((n, n))

    def treinar(self, padroes):
        """padroes: lista de arrays 1 de tamanho n."""
        for p in padroes:
            self.W += np.outer(p, p)
        np.fill_diagonal(self.W, 0)
        self.W /= self.n

    def recuperar(self, estado, passos=20):
        """Atualização assíncrona  converge para mínimo local."""
        s = np.array(estado, dtype=float)
        for _ in range(passos):
            for i in np.random.permutation(self.n):
                h = self.W[i] @ s
                s[i] = 1.0 if h >= 0 else -1.0
        return s

    def energia(self, s):
        return -0.5 * s @ self.W @ s

# Exemplo
h = Hopfield(5)
p1 = np.array([1, -1, 1, -1, 1])
h.treinar([p1])
corrompido = np.array([1, -1, -1, -1, 1])  # 1 bit errado
print(h.recuperar(corrompido))  #  p1

Aplicações

Correção de dados corrompidos

Recuperar imagens ou padrões parcialmente corrompidos a aplicação clássica de memória associativa.

Otimização combinatória

TSP e outros problemas NP foram mapeados para minimização de energia em redes de Hopfield na década de 1980.

Mínimos locais: A rede pode convergir para padrões espúrios (mínimos locais que não são padrões treinados). A capacidade prática é bem menor que o limite teórico.
