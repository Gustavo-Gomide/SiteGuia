Clarke-Wright: heurística de economia para VRP

A heurística de economia mais influente para o Problema de Roteamento de Veículos simples, eficiente e amplamente usada em logística.

Clarke-Wright (1964) resolve o VRP (Vehicle Routing Problem) de forma heurística. Começa com cada cliente em sua própria rota (depósitoclientedepósito) e combina rotas com base na economia: s(i,j) = d(depósito,i) + d(depósito,j) d(i,j). Quanto maior a economia, mais vale mesclar as rotas de i e j.

Fórmula de economia

A economia s(i,j) mede quanto se ganha ao atender i e j na mesma rota em vez de rotas separadas. Ordenar as economias e aplicar merges válidos (sem violar capacidade) é o núcleo do algoritmo.

Implementação em Python

clarke_wright.py
def clarke_wright(n, dist, demanda, capacidade):
    """n = clientes (1..n), nó 0 = depósito.
    dist[i][j] = distância entre i e j.
    Retorna lista de rotas."""
    # Calcular economias
    economias = []
    for i in range(1, n + 1):
        for j in range(i + 1, n + 1):
            s = dist[0][i] + dist[0][j] - dist[i][j]
            economias.append((s, i, j))
    economias.sort(reverse=True)

    rota_de = {i: [i] for i in range(1, n + 1)}
    carga = dict(demanda)

    for s, i, j in economias:
        ri, rj = rota_de[i], rota_de[j]
        if ri is rj: continue
        if i != ri[-1] or j != rj[0]: continue  # extremidades válidas
        nova_carga = sum(demanda[v] for v in ri + rj)
        if nova_carga > capacidade: continue
        nova_rota = ri + rj
        for v in nova_rota: rota_de[v] = nova_rota

    rotas_unicas = {id(r): r for r in rota_de.values()}
    return list(rotas_unicas.values())

Aplicações

Logística de entrega

Rotear frotas de veículos para minimizar distância total respeitando capacidades por veículo.

Coleta de lixo

Otimizar rotas de caminhões de coleta em cidades aplicação histórica original de Clarke-Wright.

Heurística, não ótima: Clarke-Wright não garante a solução ótima. O VRP é NP-difícil; para instâncias grandes, use meta-heurísticas como ALNS ou VNS.

Referências

Publicada em 1964, a heurística de Clarke-Wright é usada até hoje em sistemas de logística reais. Simplicity is elegant e uma boa heurística sobrevive décadas.

"Quanto mais você economiza unindo duas rotas, mais vale uni-las. Ordene e mescle a frota se organiza sozinha." Clarke & Wright, 1964
