Gale-Shapley: emparelhamento estável

O algoritmo que venceu o Prêmio Nobel de Economia (2012) encontra o emparelhamento estável sempre, em O(n²).

O problema do emparelhamento estável: dado um conjunto de candidatos com preferências ordenadas sobre empregos (e vice-versa), encontrar um emparelhamento onde não existe par (candidato, emprego) que se prefira mutuamente ao seu par atual. Gale-Shapley (1962) resolve isso em O(n²) usando aceitação diferida (deferred acceptance).

Estabilidade

Um emparelhamento é instável se existem um candidato a e um emprego b tal que a prefere b ao seu emprego atual e b prefere a ao seu candidato atual. Gale-Shapley garante que isso nunca aconteça ao final.

Implementação em Python

gale_shapley.py
def gale_shapley(pref_c, pref_e):
    """pref_c[c] = lista de empregos em ordem de preferência de c.
    pref_e[e] = ranking de candidatos (dict {c: rank}).
    Retorna {candidato: emprego}."""
    n = len(pref_c)
    livre = list(range(n))   # candidatos ainda não emparelhados
    prox = [0] * n           # próximo emprego a propor para cada c
    match_e = [-1] * n      # match_e[e] = candidato atual
    match_c = [-1] * n      # match_c[c] = emprego atual

    while livre:
        c = livre.pop()
        e = pref_c[c][prox[c]]; prox[c] += 1
        if match_e[e] == -1:
            match_e[e] = c; match_c[c] = e
        else:
            c2 = match_e[e]
            if pref_e[e][c] < pref_e[e][c2]:
                # e prefere c ao atual c2
                match_e[e] = c; match_c[c] = e
                match_c[c2] = -1; livre.append(c2)
            else:
                livre.append(c)  # rejeitado, tenta próximo
    return match_c

# Exemplo
pref_c = [[0,1,2],[0,2,1],[1,0,2]]
pref_e = [{0:0,1:1,2:2},{0:1,1:0,2:2},{0:0,1:2,2:1}]
print(gale_shapley(pref_c, pref_e))  # emparelhamento estável

Aplicações

Residências médicas (NRMP)

O programa de atribuição de residentes a hospitais nos EUA usa Gale-Shapley desde 1952 literalmente o problema original.

Admissões universitárias

Sistemas de ingresso como o SISU brasileiro usam variantes de Gale-Shapley para alocar estudantes a universidades.

Viés: Gale-Shapley é ótimo para o lado que propõe (candidatos) e pior possível para o lado que aceita (empregos). A ordem importa.

Referências

Gale-Shapley provou que um emparelhamento estável sempre existe e deu um algoritmo simples para encontrá-lo. Shapley recebeu o Nobel de Economia em 2012 por este trabalho (Gale havia falecido em 2008).

"Sempre que dois lados têm preferências, é possível encontrar um equilíbrio estável basta propor e diferir aceite." Gale & Shapley, 1962
