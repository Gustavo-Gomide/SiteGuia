Kruskal: MST por arestas em ordem crescente

Ordenar todas as arestas e adicionar cada uma que não forme ciclo a abordagem gulosa mais intuitiva para a árvore geradora mínima.

Kruskal constrói a Árvore Geradora Mínima (MST) de um grafo ponderado não-direcionado. A estratégia: ordene todas as arestas por peso crescente e adicione cada aresta ao MST se e somente se ela não criar um ciclo. O Union-Find garante a verificação de ciclos em tempo praticamente constante. Complexidade: O(E log E).

Ideia: crescimento por arestas

Kruskal processa arestas globalmente, enquanto Prim cresce localmente a partir de um vértice. Para grafos muito esparsos (E V), Kruskal pode ser mais eficiente pois ordena E arestas; para grafos densos, Prim com heap pode ser preferível.

Implementação em Python

kruskal.py
def kruskal(vertices, arestas):
    """arestas = [(peso, u, v)]. O(E log E).
    Retorna lista de arestas do MST e custo total."""
    # Union-Find
    pai = {v: v for v in vertices}
    rank = {v: 0 for v in vertices}

    def find(x):
        while pai[x] != x: pai[x] = pai[pai[x]]; x = pai[x]
        return x

    def union(a, b):
        a, b = find(a), find(b)
        if a == b: return False
        if rank[a] < rank[b]: a, b = b, a
        pai[b] = a
        if rank[a] == rank[b]: rank[a] += 1
        return True

    mst, custo = [], 0
    for w, u, v in sorted(arestas):
        if union(u, v):
            mst.append((u, v, w)); custo += w
    return mst, custo

# Exemplo
V = ['A','B','C','D']
E = [(1,'A','B'),(4,'A','C'),(3,'B','C'),(2,'B','D'),(5,'C','D')]
mst, total = kruskal(V, E)
print(mst)    # [('A','B',1), ('B','D',2), ('B','C',3)]
print(total)  # 6

Aplicações

Redes de telecomunicação

Conectar N cidades com o menor custo total de cabos a MST é exatamente a solução mínima.

Clustering

Single-link clustering usa a MST: remover as k arestas mais caras da MST separa o grafo em k+1 clusters.

Circuitos elétricos

Minimizar o comprimento total de fios numa placa de circuito respeitando todas as conexões necessárias.

Atenção: Kruskal exige que o grafo seja não-direcionado e conexo para produzir uma MST. Para grafos direcionados, use Prim (adaptado) ou Chu-Liu/Edmonds. Para grafos desconexos, Kruskal produz uma floresta geradora mínima.
