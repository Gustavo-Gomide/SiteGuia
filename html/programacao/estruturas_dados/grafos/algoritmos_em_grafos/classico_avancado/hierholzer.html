Hierholzer: circuito euleriano em O(E)

O problema das pontes de Königsberg resolvido de forma eficiente: percorra todas as arestas genexatamente uma vez e volte ao início se possível.

O algoritmo de Hierholzer (1873) encontra um circuito euleriano (que usa cada aresta exatamente uma vez) em O(E). A condição necessária e suficiente: o grafo é conexo e todo vértice tem grau par. Para caminho euleriano (sem voltar ao início): exatamente 2 vértices têm grau ímpar (início e fim).

Condições de Euler

Circuito euleriano: existe grafo conexo + todos os graus pares.
Caminho euleriano: existe grafo conexo + exatamente 2 vértices de grau ímpar (são o início e o fim).

Implementação em Python

hierholzer.py
from collections import defaultdict

def circuito_euleriano(n, arestas):
    """n = vértices, arestas = [(u, v)].
    Retorna lista de vértices do circuito, ou None se impossível."""
    grau = [0] * n
    adj = defaultdict(list)
    for u, v in arestas:
        adj[u].append(v); adj[v].append(u)
        grau[u] += 1; grau[v] += 1

    if any(g % 2 != 0 for g in grau): return None

    ptr = {u: 0 for u in range(n)}
    pilha, circuito = [0], []
    while pilha:
        u = pilha[-1]
        if ptr[u] < len(adj[u]):
            v = adj[u][ptr[u]]; ptr[u] += 1
            adj[v].remove(u)  # remove aresta usada (grafo não-dir)
            pilha.append(v)
        else:
            circuito.append(pilha.pop())
    return circuito[::-1]

# Exemplo: ciclo K4
arestas = [(0,1),(1,2),(2,3),(3,0),(0,2),(1,3)]
print(circuito_euleriano(4, arestas))  # alguma permutação do circuito

Aplicações

Impressão de circuitos

Trajar um caminho de impressão que use cada trilha de PCB exatamente uma vez minimiza o levantamento da caneta.

Roteamento de rotas

Verificar se uma rede permite que um veiculo percorra todas as ruas sem repetir ou encontrar o mínimo de re-visitações.

Algoritmos de sequenciamento

Montagem de genomas por De Bruijn caminhos eulerianos em grafos de k-mers.

Atenção com grafos direcionados: A condição muda todo vértice deve ter grau de entrada igual ao grau de saída para um circuito euleriano.

O problema das pontes de Königsberg (Euler, 1736) inaugurou a teoria dos grafos. Hierholzer deu a ele um algoritmo eficiente 137 anos depois. A ideia sobreviveu ao tempo e hoje é central na bioinformática moderna.

"Se cada vértice tem grau par, você pode começar em qualquer lugar e sempre voltar. Natureza e ponto de entrada são indiferentes." Euler / Hierholzer
