Vizing

Vizing Coloração de Arestas

Teorema de Vizing: Δ ou Δ+1 cores bastam

Um dos teoremas mais elegantes da teoria dos grafos: toda aresta pode ser colorida com no máximo Δ+1 cores, onde Δ é o grau máximo do grafo.

O teorema de Vizing (1964) afirma que o índice cromático (número mínimo de cores para colorir arestas sem que duas arestas incidentes no mesmo vértice compartilhem cor) é sempre Δ ou Δ+1. Grafos de classe 1 usam Δ cores; grafos de classe 2 precisam de Δ+1. Determinar a classe é NP-completo em geral.

Por que colorir arestas?

Coloração de arestas modela escalonamento livre de conflitos: cada cor é um "slot de tempo", cada aresta é uma tarefa que requer os dois vértices extremos. Com cores adjacentes distintas, nenhum recurso é usado duas vezes ao mesmo tempo.

Implementação em Python (Vizing com Δ+1 cores)

vizing.py
def vizing_coloring(n, arestas):
    """Colore arestas com no máximo Δ+1 cores.
    Retorna dict {(u,v): cor}."""
    from collections import defaultdict

    adj_cores = [defaultdict(lambda: -1) for _ in range(n)]
    delta = max(sum(1 for u, v in arestas if u == i or v == i) for i in range(n))
    color = {}

    def livre(v, c):
        return adj_cores[v][c] == -1

    def augmentar(u, v):
        # Encontra cor livre em u, não livre em v; inverte cadeia Kempe
        c_livre_u = next(c for c in range(delta + 1) if livre(u, c))
        if livre(v, c_livre_u):
            return c_livre_u
        c_livre_v = next(c for c in range(delta + 1) if livre(v, c))
        # Inverte cadeia Kempe (c_livre_u, c_livre_v) a partir de v
        w, c_cur = adj_cores[u][c_livre_v], c_livre_v
        while w != -1:
            c_next = c_livre_u if c_cur == c_livre_v else c_livre_v
            w_next = adj_cores[w][c_next]
            color[(u, w)] = c_next  # atualiza
            adj_cores[u][c_cur] = -1; adj_cores[u][c_next] = w
            adj_cores[w][c_cur] = -1; adj_cores[w][c_next] = u
            u, c_cur, w = w, c_next, w_next
        return c_livre_u

    for u, v in arestas:
        c = augmentar(u, v)
        color[(u, v)] = c
        adj_cores[u][c] = v; adj_cores[v][c] = u
    return color

Aplicações

Escalonamento de exames

Cada prova são dois alunos compartilhando recursos. Coloração de arestas com Δ+1 slots garante que nenhum aluno tenha duas provas ao mesmo tempo.

Alocação de frequências

Em redes de rádio, arestas representam interferência; coloração atribui frequências sem conflito.

Classe 1 vs Classe 2: Determinar se um grafo geral precisa de Δ ou Δ+1 cores é NP-completo. Para grafos bipartidos, o mínimo é sempre Δ (König, 1916).
