Busca Binária: dividir para conquistar

Corta o espaço de busca ao meio a cada passo de O(n) para O(log n) com uma condição: o array deve estar ordenado.

A busca binária é um dos algoritmos mais elegantes da ciência da computação. Ao invés de verificar cada elemento, ela examina o elemento do meio e decide descartar metade inteira do array. Em um array de 1 milhão de itens ordenados, a busca binária encontra qualquer valor em no máximo 20 comparações.

O que é a Busca Binária?

A busca binária opera sobre arrays ordenados. Ela mantém dois ponteiros inicio e fim que definem o intervalo atual de busca. A cada iteração, calcula o índice do meio, compara com o alvo e elimina metade do intervalo: se o alvo é menor que o meio, descarta a metade direita; se é maior, descarta a esquerda. Repete até encontrar o alvo ou o intervalo ficar vazio.

Analogia: Procurar uma palavra no dicionário abrindo sempre no meio. Se a página aberta está depois da palavra, você vai à metade esquerda. Se está antes, vai à metade direita. Você nunca lê o dicionário do início ao fim.

Implementação em Python

busca_binaria.py
def busca_binaria(arr, alvo):
    """Retorna o índice do alvo em arr (ordenado) ou -1."""
    inicio = 0
    fim = len(arr) - 1

    while inicio <= fim:
        meio = (inicio + fim) // 2   # evita overflow vs (i+f)/2

        if arr[meio] == alvo:
            return meio               # encontrado
        elif arr[meio] < alvo:
            inicio = meio + 1          # descarta metade esquerda
        else:
            fim = meio - 1            # descarta metade direita

    return -1                      # não encontrado


# --- Exemplo de uso ---
numeros = [10, 20, 30, 40, 50, 60, 70]
idx = busca_binaria(numeros, 70)
print(f"Índice: {idx}")  # Índice: 6

Pontos importantes:

  • O cálculo meio = (inicio + fim) // 2 usa divisão inteira; em Python não há overflow de inteiro, mas a forma é idiomática.
  • A condição do loop é inicio <= fim quando cruzam, o intervalo fica vazio e o alvo não existe.
  • O array precisa estar ordenado antes da chamada; a função não valida isso.
  • Python oferece o módulo bisect da biblioteca padrão com busca binária otimizada.

Quando usar

Array grande e estático

Quando o array é ordenado uma vez e consultado muitas vezes, a busca binária amortiza o custo de ordenação em buscas O(log n).

Busca em intervalos numéricos

Encontrar o ponto de inserção, o menor elemento maior que X, ou o maior elemento menor que X todos padrões nativos de busca binária.

Bases de dados ordenadas

Índices B-Tree em bancos de dados são essencialmente busca binária generalizada para estruturas em disco.

Atenção: Aplicar busca binária em um array não ordenado pode retornar resultados silenciosamente errados o algoritmo termina sem erro mas reporta "não encontrado" mesmo quando o alvo está lá, ou retorna um índice incorreto.
