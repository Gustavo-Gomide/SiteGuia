MQTT, serial e coleta de sensores

Combinar MQTT com leitura serial e um padrao comum para integrar dispositivos legados e sensores modernos.

Loop simplificado: serial para MQTT
local socket = require("socket")
local mqtt = require("mqtt_library")

local client = mqtt.client.create("broker.local", 1883, function() end)
client:connect("edge-gw-01")

while true do
  local payload = "{\"temp\":23.4,\"umid\":51}"
  client:publish("fabrica/linha1/sensores", payload)
  socket.sleep(1)
end

Boas praticas

Use QoS adequado, timestamp monotonic para ordenacao e validacao de payload no consumidor.

Dispositivo em campo reinicia, perde link e sofre ruido eletrico. Por isso, o software precisa ser resiliente por design.
