Python — Introdução

Python

A linguagem de programação mais popular do mundo — legível, expressiva e com o ecossistema mais vasto da história da computação.

Python é a linguagem preferida para ciência de dados, automação, web, IA e ensino de programação. Criada por Guido van Rossum em 1991 com a filosofia de que código deve ser lido mais do que escrito, Python domina rankings de popularidade como TIOBE, Stack Overflow e GitHub desde 2021.

História e Origem

Python nasceu no Natal de 1989, quando Guido van Rossum — trabalhando no CWI (Centrum Wiskunde & Informatica) em Amsterdã — começou a escrever a linguagem como projeto de fim de semana. O nome foi inspirado em Monty Python's Flying Circus, não pela serpente tropical.

O Zen do Python

Os princípios de design do Python estão codificados no PEP 20, disponível executando import this no interpretador. Esses princípios guiam toda decisão de design da linguagem e da comunidade.

PEP 20 — The Zen of Python (Tim Peters, 1999)

PrincípioSignificado
Beautiful is better than uglyCódigo bonito > código que "apenas funciona"
Explicit is better than implicitNão oculte intenções; torne o código legível
Simple is better than complexEvite complexidade desnecessária
Readability countsCódigo é lido mais do que escrito
Errors should never pass silentlyNunca silencie exceções sem intenção explícita
There should be one obvious way to do itContrasta com Perl: "There is more than one way"

Python — programa completo com input, loops e funções
#!/usr/bin/env python3
"""Calculadora de fatorial com tratamento de erro."""

def fatorial(n: int) -> int:
    """Calcula o fatorial de n de forma iterativa."""
    if n < 0:
        raise ValueError(f"n deve ser >= 0, recebeu {n}")
    resultado = 1
    for i in range(2, n + 1):
        resultado *= i
    return resultado


def main() -> None:
    while True:
        entrada = input("Digite um inteiro não-negativo (ou 'sair'): ")
        if entrada.lower() == "sair":
            break
        try:
            n = int(entrada)
            print(f"{n}! = {fatorial(n):,}")
        except ValueError as e:
            print(f"Erro: {e}")


if __name__ == "__main__":
    main()

Por que aprender Python?

Ciência de Dados e IA

NumPy, Pandas, Scikit-learn, TensorFlow, PyTorch, Hugging Face — o ecossistema de ML mais completo do mundo.

Web Backend

Django (Instagram, Pinterest, Spotify), FastAPI (Microsoft, Netflix APIs) e Flask. Python alimenta serviços de bilhões de usuários.

Automação e DevOps

Ansible, Fabric, scripting de CI/CD, automação de tarefas repetitivas. Python substituiu bash para a maioria das automações complexas.

Velocidade de Desenvolvimento

Código Python é 5–10x mais curto que Java/C++ equivalente. O REPL interativo e a tipagem dinâmica aceleram prototipagem.
