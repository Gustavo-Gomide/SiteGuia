Pascal — Introdução

Pascal

A linguagem que ensinou programação estruturada a gerações de programadores — criada por Niklaus Wirth com clareza pedagógica como objetivo principal.

Pascal foi projetada deliberadamente como ferramenta de ensino, mas sua influência vai muito além da academia. De compiladores a sistemas operacionais, de aplicações desktop a microcontroladores, a família Pascal continua viva através do Free Pascal Compiler e do Delphi, mais de 50 anos após sua criação.

História — Niklaus Wirth e a Linguagem Pedagógica

Niklaus Wirth (1934–2024), cientista da computação suíço e Prêmio Turing 1984, criou Pascal para ser a ferramenta perfeita para ensinar programação estruturada — o paradigma que substituiu o caótico uso de GOTO dos anos 1960.

1970 — Pascal 1.0

Wirth publica a definição de Pascal no ETH Zürich. O nome homenageia o matemático francês Blaise Pascal (1623–1662), inventor de uma calculadora mecânica pioneira. A linguagem é projetada para que programas corretos sejam fáceis de escrever e programas incorretos difíceis de compilar.

1974 — Popularização

O compilador CDC 6000 Pascal é lançado amplamente. Universidades ao redor do mundo adotam Pascal como linguagem de ensino, substituindo Fortran e ALGOL.

1978 — Jensen & Wirth

Publicação do livro Pascal User Manual and Report (Jensen & Wirth), que se torna a referência definitiva da linguagem por décadas.

1983 — Turbo Pascal

A Borland lança o Turbo Pascal por apenas US$ 49 — uma fração do custo dos compiladores da época. Inclui IDE integrado, compilação ultra-rápida e acesso a hardware. Revoluciona o mercado de PCs e torna Pascal a linguagem de escolha para desenvolvimento MS-DOS.

1995 — Delphi

Borland lança Delphi, estendendo Pascal com Orientação a Objetos robusta (Object Pascal) e RAD (Rapid Application Development) para Windows. Torna-se dominante no desenvolvimento de aplicações de negócio.

1993–hoje — Free Pascal

Free Pascal Compiler (FPC) surge como alternativa open-source total. Compatível com Turbo Pascal e Delphi, suporta dezenas de plataformas: Linux, Windows, macOS, iOS, Android, *BSD, DOS, e microcontroladores como AVR e ARM.

Filosofia de Design

Wirth projetou Pascal com princípios que ainda são relevantes:

Tipagem Forte

Cada variável tem um tipo declarado. Conversões entre tipos são explícitas. Isso elimina uma classe enorme de bugs comuns em C, onde um inteiro pode ser tratado como ponteiro silenciosamente.

Legibilidade

Palavras-chave em inglês natural (begin, end, for, to, do, while) tornam o código auto-documentado. Um iniciante pode ler Pascal e entender a intenção do código.

Estrutura Obrigatória

Todo programa tem uma estrutura bem definida: declarações antes do código executável, blocos delimitados por begin/end. Não é possível misturar declarações e instruções aleatoriamente.

Sem Ponteiros Implícitos

No Pascal original, ponteiros existem mas são explícitos e verificados. Não há aritmética de ponteiros como em C. A segurança é prioritária.

Instalação — Free Pascal e Lazarus

bash — Instalação do Free Pascal Compiler
# Linux (Ubuntu/Debian)
sudo apt install fpc lazarus

# Arch Linux
sudo pacman -S fpc lazarus

# macOS (via Homebrew)
brew install fpc

# Windows
# Baixar instalador: https://www.lazarus-ide.org/index.php?page=downloads
# Lazarus inclui FPC automaticamente

# Verificar versão
fpc -v
# Free Pascal Compiler version 3.2.2

# Compilar um arquivo
fpc hello.pas
./hello

# Cross-compilar para Windows no Linux
fpc -Tcrossos=win64 hello.pas

Hello World e Estrutura Básica

Todo programa Pascal começa com a palavra program, seguida pelo nome do programa. O bloco principal é delimitado por begin e end. (com ponto final). Seções de declaração (var, const, type) vêm antes do begin.

hello.pas — Estrutura completa de programa Pascal
program HelloMundo;

{ Comentários com chaves }
(* Ou com parênteses-asterisco *)
// Ou comentário de linha (extensão moderna)

uses
  SysUtils;   { unidade de utilitários do sistema }

const
  VERSAO = '1.0';
  ANO    = 1970;

var
  nome    : String;
  numero  : Integer;

begin
  { Saída básica }
  WriteLn('Olá, Mundo!');
  WriteLn('Pascal versão: ', VERSAO);
  WriteLn('Criado em: ', ANO);

  { Write sem newline }
  Write('Digite seu nome: ');
  ReadLn(nome);
  WriteLn('Olá, ', nome, '!');

  { Formatação numérica }
  numero := 42;
  WriteLn('O número é: ', numero);
  WriteLn('Formatado: ', numero:5);    { campo de largura 5 }
  WriteLn('Pi = ', 3.14159:8:5);   { 8 total, 5 decimais }

  { Aguardar tecla antes de fechar }
  ReadLn;
end.
Compilar e executar
# Compilar
fpc hello.pas
# Saída: hello (Linux/macOS) ou hello.exe (Windows)

# Executar
./hello

# Com otimização
fpc -O3 hello.pas

# Gerar debug info (para GDB)
fpc -g hello.pas

Pascal no Mundo Atual

Educação

Muitas universidades europeias e latino-americanas ainda usam Pascal (via FPC) para introduzir programação estruturada. A sintaxe clara e o compilador rigoroso forçam boas práticas desde o início.

Sistemas Legados

Grandes sistemas corporativos escritos em Delphi/Pascal estão em produção nos setores financeiro, governamental e de saúde. Manutenção e extensão desses sistemas mantêm desenvolvedores Pascal empregados.

Sistemas Embarcados

Free Pascal suporta cross-compilation para AVR, ARM Cortex-M e outros microcontroladores. Projetos como µPas e MicroPascal são usados em sistemas industriais.

Lazarus/LCL

Para desenvolvedores que precisam de GUI nativa multiplataforma sem custo de licença, Lazarus é uma alternativa viável ao Delphi. Uma base fiel de usuários mantém o projeto ativo.
