Julia — Introdução

Julia — Do Zero ao Avançado

Julia é uma linguagem de programação dinâmica de alto desempenho projetada para computação científica, análise de dados e machine learning. Combina a facilidade do Python com velocidade próxima a C, usando compilação JIT com LLVM.

Julia resolve o problema das duas linguagens: não é mais necessário prototipar em Python e reescrever em C/Fortran para ter desempenho. Com multiple dispatch e geração de código especializado por tipo, Julia chega a velocidades de C enquanto mantém sintaxe expressiva de alto nível.

Características Principais

Multiple Dispatch

Funções se especializam em todos os tipos dos argumentos, não apenas no primeiro. O método mais específico é selecionado em tempo de execução.

JIT com LLVM

Código Julia é compilado para código nativo em tempo de execução. Sem interpretação: primeiro chamada compila, subsequentes rodam código nativo.

Tipagem Dinâmica Opcional

Tipos são opcionais mas presentes. Anotações de tipo melhoram desempenho e documentação, mas não são obrigatórias para funcionar.

Matemática Nativa

Suporte nativo a inteiros arbitrários, racionais, complexos, frações. Unicode em identificadores: α = 0.5, ∑(x).

Metaprogramação

Macros e sistema de expressões completo. Código Julia como dado: @time, @generated, transformações de AST.

Interoperabilidade

Chama C, Fortran sem overhead. Interfaces Python (PyCall), R (RCall), Java. Convivência com ecossistemas existentes de ciência.

Instalação e REPL

juliaup — gerenciador de versões (recomendado)
# Linux/macOS
curl -fsSL https://install.julialang.org | sh

# Windows (PowerShell)
winget install julia -s msstore
# ou
winget install Julialang.Juliaup

# Instalar versão específica
juliaup add 1.11
juliaup default 1.11

# Verificar instalação
julia --version
REPL — Julia interativo
# Iniciar REPL
julia

# Modos do REPL:
julia> 1 + 1          # modo Julia (padrão)
2

julia> ?sin           # ? para modo Help
julia> ]               # ] para modo Pkg
(@v1.11) pkg> add Plots
(@v1.11) pkg> status

julia> ;               # ; para modo Shell
shell> ls

# Variáveis persistem na sessão
julia> x = 42
julia> x^2             # 1764

# ans é o último resultado
julia> 2 + 2
julia> ans * 10        # 40

Primeiro Programa

hello.jl — conceitos fundamentais
# Primeiro programa Julia
println("Olá, Mundo!")

# Strings com interpolação
nome = "Julia"
versao = 1.11
println("Linguagem: $nome $versao")
println("1 + 1 = $(1 + 1)")

# Funções
function saudar(nome::String)
    return "Olá, $nome!"
end

# Sintaxe curta para funções simples
saudar_curto(nome) = "Olá, $nome!"

# Funções anônimas
dobrar = x -> x * 2

println(saudar("Mundo"))
println(dobrar(21))

# Unicode em código
α = 0.5
β = 2.0
resultado = α * β
println("α × β = $resultado")

# Executar: julia hello.jl

Quando Usar Julia?

Use Julia quando:

  • Computação científica com alto desempenho
  • Machine learning e data science
  • Simulações numéricas (física, engenharia, finanças)
  • Prototipagem que depois precisa de performance
  • Processamento de dados em larga escala
  • Pesquisa acadêmica em computação

Evite Julia quando:

  • Aplicações web CRUD simples
  • Projetos sem cálculos intensivos
  • Equipe sem afinidade com matemática/ciência
  • Startup time crítico (latência de compilação JIT na 1ª chamada)
  • Ecosistema dominado por outro stack (Node para frontend)
