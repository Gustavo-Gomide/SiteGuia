Julia — Dados e IA

Julia — Dados e Inteligência Artificial

Julia foi criada com computação numérica em mente. Por isso, o ecossistema de dados e IA é uma continuação natural da linguagem: ingestão tabular com CSV.jl, transformação com DataFrames.jl, modelagem com MLJ.jl e deep learning com Flux.jl.

O grande ganho de Julia em dados e IA é a unificação: leitura, limpeza, estatística, treinamento, profiling e deploy podem viver no mesmo código, com o mesmo sistema de tipos e com desempenho suficiente para eliminar boa parte da reescrita em linguagens de baixo nível.

ETL com CSV.jl e DataFrames.jl

Leitura, limpeza e agregação tabular
using CSV
using DataFrames
using Statistics

df = CSV.read("vendas.csv", DataFrame)

# Visão inicial
first(df, 5)
describe(df)

# Limpeza
dropmissing!(df)
rename!(df, :valor_total => :valor)
transform!(df, :valor => ByRow(x -> round(x, digits = 2)) => :valor)

# Filtragem e seleção
df_validos = subset(df, :valor => ByRow(>(0)), :categoria => ByRow(!ismissing))
df_reduzido = select(df_validos, :data, :categoria, :valor)

# Agrupamento
resumo = combine(groupby(df_reduzido, :categoria),
		:valor => mean => :media,
		:valor => std => :desvio,
		:valor => sum => :total,
		nrow => :quantidade)

CSV.write("resumo.csv", resumo)

# CSV.File também permite streaming/iteração sem DataFrame imediato.

Estatística e Machine Learning com MLJ

Pipeline supervisionado com split, machine e métricas
using MLJ
using DataFrames
using Statistics

iris = DataFrame(load_iris())
y, X = unpack(iris, ==(:target), rng = 123)

train, test = partition(eachindex(y), 0.7, shuffle = true, rng = 42)

Tree = @load DecisionTreeClassifier pkg = DecisionTree verbosity = 0
modelo = Tree(max_depth = 4)

mach = machine(modelo, X, y)
fit!(mach, rows = train)

pred = predict_mode(mach, rows = test)
acc = accuracy(pred, y[test])
println("Acurácia = $(round(acc, digits = 4))")

# Tuning automático
ranges = [range(modelo, :max_depth, lower = 2, upper = 8)]
tm = TunedModel(
		model = modelo,
		tuning = Grid(resolution = 7),
		resampling = CV(nfolds = 5),
		range = ranges,
		measure = accuracy,
)

tuned = machine(tm, X, y)
fit!(tuned)

# MLJ oferece interface única para muitos algoritmos.

Deep Learning com Flux.jl

Rede neural simples e treinamento por gradiente
using Flux
using Statistics

modelo = Chain(
		Dense(10 => 32, relu),
		Dense(32 => 16, relu),
		Dense(16 => 1)
)

x = rand(Float32, 10, 256)
y = rand(Float32, 1, 256)

loss(m, x, y) = Flux.mse(m(x), y)
opt = Flux.setup(Adam(0.001), modelo)

for epoca in 1:50
		grads = Flux.gradient(modelo) do m
				loss(m, x, y)
		end
		Flux.update!(opt, modelo, grads[1])

		if epoca % 10 == 0
				println("época $epoca - loss = $(loss(modelo, x, y))")
		end
end

# Flux expõe a lógica do treinamento com pouca mágica.
# Isso facilita entender tensores, gradientes e atualização de pesos.
GPU com CUDA.jl
using CUDA
using Flux

CUDA.functional() || error("GPU CUDA não disponível")

modelo_gpu = Chain(
		Dense(784 => 128, relu),
		Dense(128 => 10)
) |> gpu

x_gpu = CUDA.rand(Float32, 784, 512)
y_gpu = CUDA.rand(Float32, 10, 512)

loss_gpu(m, x, y) = Flux.mse(m(x), y)
opt_gpu = Flux.setup(Adam(), modelo_gpu)

grads = Flux.gradient(modelo_gpu) do m
		loss_gpu(m, x_gpu, y_gpu)
end

Flux.update!(opt_gpu, modelo_gpu, grads[1])

# Em muitos casos basta mover arrays/modelo para gpu.

Cenários em que Julia se destaca

  • Modelagem científica e física computacional
  • Pipelines numéricos pesados com pouca tolerância a overhead
  • Otimização matemática integrada com ML
  • Pesquisa onde protótipo e produção precisam convergir

Trade-offs reais

  • Ecossistema menor que Python em quantidade de pacotes
  • Warm-up JIT pode afetar a primeira execução
  • Algumas integrações empresariais são menos maduras
