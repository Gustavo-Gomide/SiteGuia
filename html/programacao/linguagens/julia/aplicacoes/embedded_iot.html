Julia — Embedded e IoT

Julia — Embedded e Internet das Coisas

Julia não é uma linguagem de firmware bare-metal como C, mas é excelente para o lado de borda e gateway IoT: aquisição serial, pré-processamento local, inferência leve e publicação para nuvem via MQTT.

Em IoT real, Julia costuma atuar como camada inteligente entre sensores e cloud: recebe telemetria, limpa ruído, agrega janelas de tempo, detecta anomalias e envia dados já enriquecidos.

Leitura de Sensores por Serial

SerialPorts.jl para comunicação com microcontroladores
using SerialPorts
using Dates

porta = SerialPort("COM3", 115200)

function ler_linha_segura(sp::SerialPort)
		try
				return readline(sp)
		catch err
				@warn "falha na leitura serial" erro = err
				return nothing
		end
end

for _ in 1:10
		linha = ler_linha_segura(porta)
		isnothing(linha) && continue

		# Exemplo esperado do Arduino: "temp=24.5,hum=56.2"
		println("$(now()) | $linha")
end

close(porta)

# Dica prática: use timeout e reconexão para evitar travas.

MQTT para Telemetria

Publicar e consumir mensagens IoT
using MQTTClient
using JSON3

client = Client("tcp://broker.hivemq.com:1883", "julia-iot-node")
connect(client)

payload = JSON3.write(Dict(
		"device_id" => "sensor-01",
		"temp" => 24.6,
		"hum" => 55.9,
		"ts" => time(),
))

publish(client, "siteguia/iot/telemetria", payload)

subscribe(client, "siteguia/iot/comandos") do msg
		println("Comando recebido: ", String(msg.payload))
end

disconnect(client)

# QoS, retained message e reconexão automática são essenciais em produção.

Edge Analytics no Gateway

Filtragem, janelas e detecção simples de anomalias
using Statistics

function media_movel(v::Vector{Float64}, janela::Int)
		n = length(v)
		n < janela && return Float64[]
		saida = Float64[]
		for i in janela:n
				push!(saida, mean(@view v[i-janela+1:i]))
		end
		saida
end

function detectar_anomalia(amostra::Float64, historico::Vector{Float64}; zlim = 3.0)
		length(historico) < 10 && return false
		mu = mean(historico)
		sigma = std(historico)
		sigma == 0 && return false
		z = abs((amostra - mu) / sigma)
		z > zlim
end

leituras = [24.0, 24.2, 24.1, 24.3, 24.2, 24.1, 24.0, 24.2, 24.3, 24.2]
nova = 30.8

if detectar_anomalia(nova, leituras)
		println("ALERTA: valor fora do padrão")
end

# Esse tipo de lógica local reduz tráfego e latência de alertas.

Boa arquitetura IoT com Julia

  • Microcontrolador coleta sinal bruto
  • Julia no gateway faz limpeza e agregação
  • Broker MQTT distribui para serviços
  • Nuvem armazena e gera dashboards

Pontos críticos

  • Firmware ainda costuma ser C/C++/Rust
  • Conectividade intermitente exige buffer local
  • Segurança (TLS, certificados) não pode ser opcional
