Julia — GUI Desktop

Julia — Interfaces Desktop

Embora Julia seja associada a ciência de dados, ela também suporta aplicações desktop para ferramentas internas, painéis operacionais e utilitários técnicos, principalmente com Gtk.jl, QML.jl e Tk.jl.

Desktop em Julia funciona muito bem quando a interface é a camada de operação de um núcleo analítico. Você mantém UI, cálculo e visualização no mesmo stack.

Gtk.jl — GUI Tradicional e Robusta

Janela, botão e eventos
using Gtk

win = GtkWindow("Calculadora Julia", 420, 240)
box = GtkBox(:v)
entrada = GtkEntry()
saida = GtkLabel("Resultado: -")
botao = GtkButton("Calcular dobro")

push!(box, entrada)
push!(box, botao)
push!(box, saida)
push!(win, box)

signal_connect(botao, "clicked") do _
		txt = Gtk.bytestring(entrada)
		valor = tryparse(Float64, txt)
		if isnothing(valor)
				set_gtk_property!(saida, :label, "Entrada inválida")
		else
				set_gtk_property!(saida, :label, "Resultado: $(2 * valor)")
		end
end

showall(win)

# main() implícito ao executar no REPL/event loop GTK.

QML.jl — Interface Moderna com Qt

Camada visual em QML e lógica em Julia
using QML

mutable struct Estado
		contador::Int
end

estado = Estado(0)

function incrementar()
		estado.contador += 1
		return estado.contador
end

qml = """
import QtQuick 2.15
import QtQuick.Controls 2.15

ApplicationWindow {
		visible: true
		width: 320
		height: 200
		title: "QML + Julia"

		Column {
				anchors.centerIn: parent
				spacing: 12
				Text { id: lbl; text: "Contador: 0" }
				Button {
						text: "Incrementar"
						onClicked: lbl.text = "Contador: " + julia_incrementar()
				}
		}
}
"""

loadqml(qml)
setcontextproperty("julia_incrementar", incrementar)
exec()

# QML separa design visual da lógica computacional.

Dashboards Locais e Visualização

Integração GUI + gráficos científicos
using CairoMakie
using Statistics

dados = cumsum(randn(300))

fig = Figure(size = (800, 500))
ax = Axis(fig[1, 1], title = "Sinal de Processo")
lines!(ax, dados, color = :blue)
hline!(ax, [mean(dados)], color = :red, linestyle = :dash)

save("dashboard_local.png", fig)

# Em apps desktop técnicos, esse gráfico costuma aparecer embutido na UI.
# Fluxo comum:
# 1. UI recebe parâmetros
# 2. Julia executa cálculo pesado
# 3. resultado é plotado e exportado
