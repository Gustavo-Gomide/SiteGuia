Dart — Introducao

Introducao ao Dart

Dart e uma linguagem compilada, fortemente tipada e orientada a objetos, criada pelo Google em 2011. Seu principal caso de uso e o desenvolvimento de aplicacoes Flutter (mobile, web e desktop), mas tambem e utilizada como linguagem de scripts, APIs REST com o framework Shelf e ferramentas de linha de comando. Dart compila tanto via JIT (Just-In-Time) — para desenvolvimento rapido com hot reload — quanto via AOT (Ahead-Of-Time) gerando binarios nativos altamente otimizados.

Dart 3 (lançado em 2023) trouxe records, patterns, sealed classes e garantia de null safety em 100% da linguagem — mudancas que tornam o codigo mais expressivo e seguro sem sacrificar performance.

Historia e Versoes

2011 — Lancamento
Lars Bak e Kasper Lund apresentam Dart na conferencia GOTO como alternativa estruturada ao JavaScript para o ecossistema web do Google.
2013 — Dart 1.0
Primeira versao estavel. Suporte a compilacao JavaScript (dart2js), VM propria para servidores e plugin para o editor Dartium.
2018 — Dart 2.0
Reescrita completa com sistema de tipos obrigatorio (sem dynamic implicito), nova toolchain pub e lancamento do Flutter como framework oficial Dart.
2021 — Dart 2.12 Null Safety
Sound null safety: o compilador garante em tempo de compilacao que valores null nao causam erros em runtime. Toda variavel e nao-nula por padrao.
2023 — Dart 3.0
Records, patterns e exhaustive matching com sealed classes. Toda base de codigo deve ter null safety; Dart 2 sem null safety nao e mais suportado.
2024/2025 — Dart 3.x
Macros (experimental), inline classes, extension types mais poderosos e melhorias continuas na compilacao AOT para WASM (WebAssembly) via Flutter Web.

Instalacao e Primeiros Passos

Dart vem integrado ao Flutter SDK. Se voce ja tem Flutter, o executavel dart ja esta no seu PATH. Para instalar so o Dart SDK standalone use o gerenciador de pacotes da sua plataforma.
Shell — Instalacao do Dart SDK
# Windows (via winget)
winget install Google.DartSDK

# macOS (via Homebrew)
brew install dart

# Ubuntu/Debian
sudo apt-get update
sudo apt-get install dart

# Verificar versao
dart --version
# Dart SDK version: 3.5.0 (stable)
Shell — Criar projeto Dart e estrutura de diretorios
# Novo projeto console
dart create meu_app
cd meu_app

# Estrutura gerada
# meu_app/
#   bin/meu_app.dart   — ponto de entrada
#   lib/               — codigo reutilizavel
#   test/              — testes
#   pubspec.yaml       — manifesto e dependencias
#   analysis_options.yaml

# Executar
dart run

# Compilar para executavel nativo AOT
dart compile exe bin/meu_app.dart -o meu_app
./meu_app

# Compilar para JavaScript
dart compile js bin/meu_app.dart -o meu_app.js

# Analisar erros estaticos
dart analyze

# Formatar codigo
dart format .
Dart — Primeiro programa com tipos e null safety
// bin/meu_app.dart

void main(List<String> args) {
  // Tipos basicos
  String nome    = 'Dart';
  int    versao  = 3;
  double pi      = 3.14159;
  bool   ativo   = true;

  // Inferencia de tipo com var
  var mensagem = 'Ola, $nome $versao!'; // String

  // Tipo nullable — pode ser null
  String? nomeOpcional;
  nomeOpcional = args.isNotEmpty ? args[0] : null;

  // Operador ?.
  int? tamanho = nomeOpcional?.length;

  // Operador ??
  String efectivo = nomeOpcional ?? 'anonimo';

  print(mensagem);
  print('Pi = $pi, ativo = $ativo');
  print('Nome: $efectivo, tamanho: ${tamanho ?? "desconhecido"}');

  // Collections
  List<int>          numeros = [1, 2, 3];
  Set<String>        tags    = {'dart', 'flutter'};
  Map<String, int>  notas   = {'alice': 90, 'bob': 85};

  print(numeros.map((n) => n * 2).toList()); // [2, 4, 6]
  print(notas['alice']);                        // 90
}

Modelos de Compilacao: JIT e AOT

JIT vs AOT

AspectoJIT (dart run)AOT (dart compile exe)
Uso principalDesenvolvimento, testes, hot reload FlutterDistribuicao de producao
Tempo de startupMaior (VM precisa carregar)Extremamente rapido (<10ms)
Performance runtimeBoa (otimizacao especulativa)Melhor (inlining estatico)
Tamanho binarioN/A (precisa do SDK)~5–15 MB self-contained
Hot reloadSim (Flutter)Nao
ReflectionSuportadaRestrita (tree-shaking)
