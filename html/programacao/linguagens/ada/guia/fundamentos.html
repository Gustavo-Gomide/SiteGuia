Fundamentos de Ada

Tipos, Variáveis, Controle de Fluxo e Subprogramas

A base da linguagem: um sistema de tipos rigoroso que elimina classes inteiras de bugs.

O coração do Ada é seu sistema de tipos. Enquanto em C ou Python você pode misturar valores de tipos distintos com conversões implícitas, Ada proíbe isso por padrão. Ao dominar os tipos primitivos, os tipos derivados, os subtipos com restrições de intervalo e as estruturas de controle, você terá a base sólida para escrever ada correto e legível.

1. Tipos Primitivos

Ada possui tipos primitivos predefinidos no pacote Standard, importado automaticamente em todo programa. Os principais são:

Tipos Numéricos Inteiros

TipoTamanhoIntervalo
Integer32 bits (típico)−2³¹ a 2³¹−1
Long_Integer64 bits−2⁶³ a 2⁶³−1
Short_Integer16 bits−32768 a 32767
Naturalsubtipo de Integer0 a Integer'Last
Positivesubtipo de Integer1 a Integer'Last
ada — declaração de variáveis
procedure Tipos_Demo is
   -- Declarações de variáveis na seção declarativa (antes do begin)
   Idade     : Natural  := 25;
   Preco     : Float    := 19.99;
   Nome      : String   := "Alice";
   Ativo     : Boolean  := True;
   Letra     : Character:= 'A';

   -- Constante: valor fixo em tempo de compilação
   Pi        : constant Float := 3.14159_26535;

   -- Ada permite _ em literais numéricos para legibilidade
   Populacao : constant Long_Integer := 7_900_000_000;
begin
   null;  -- corpo vazio, válido em Ada
end Tipos_Demo;

Ada não inicializa variáveis automaticamente com zero ou null. Uma variável não inicializada tem um valor indeterminado. O compilador GNAT emite aviso se você usar uma variável antes de inicializá-la, e o pragma Normalize_Scalars força inicialização com valor predefinido.

2. Tipos Definidos pelo Usuário

A característica mais poderosa de Ada é a capacidade de criar novos tipos que não são compatíveis com seus tipos base mesmo que tenham a mesma representação. Isso evita a confusão clássica de misturar, por exemplo, metros com quilômetros.

ada — tipos distintos e subtipos
package Sistema_Medidas is

   -- Tipo derivado: incompatível com Float por design
   type Metros      is new Float;
   type Quilometros is new Float;
   type Segundos    is new Float;

   -- Subtipo: COMPATÍVEL com o tipo base, mas com restrição de range
   subtype Percentual   is Float range 0.0 .. 100.0;
   subtype Temperatura   is Float range -270.0 .. 5000.0;
   subtype Indice_Positivo is Integer range 1 .. Integer'Last;

end Sistema_Medidas;

-- Uso: Ada detecta o erro em tempo de compilação
with Sistema_Medidas;
procedure Teste is
   use Sistema_Medidas;
   D : Metros     := 100.0;
   K : Quilometros := 5.0;
begin
   -- D := K;  ← ERRO de compilação: tipos incompatíveis!
   -- Conversão explícita é possível e intencional:
   D := Metros (Quilometros'(1.0) * 1000.0);
end Teste;

3. Tipos Enumerados

Ada tem suporte nativo e robusto para enumerações. O compilador pode verificar que todos os casos são tratados em estruturas case — sem "fall-through" silencioso como em C.

ada — enumerações e case
type Dia_Semana is
   (Segunda, Terca, Quarta, Quinta, Sexta, Sabado, Domingo);

type Cor_Semaforo is (Vermelho, Amarelo, Verde);

procedure Verificar_Dia (Dia : Dia_Semana) is
begin
   case Dia is
      when Segunda .. Sexta =>
         Ada.Text_IO.Put_Line ("Dia útil");
      when Sabado | Domingo =>
         Ada.Text_IO.Put_Line ("Final de semana");
      -- Ada exige que todos os casos sejam cobertos!
      -- Sem "when others" necessário pois os 7 dias foram cobertos.
   end case;
end Verificar_Dia;

-- Atributos de enumeração
with Ada.Text_IO; use Ada.Text_IO;
procedure Demo_Enum is
   D : Dia_Semana := Segunda;
begin
   Put_Line (D'Image);              -- "SEGUNDA"
   Put_Line (Dia_Semana'First'Image); -- "SEGUNDA"
   Put_Line (Dia_Semana'Last'Image);  -- "DOMINGO"
   D := Dia_Semana'Succ (D);        -- Terca
end Demo_Enum;

4. Estruturas de Controle

Ada oferece as estruturas clássicas de controle com sintaxe explícita e sem armadilhas de precedência.

ada — if, loop, while, for
procedure Controle_Demo is
   X : Integer := 10;
begin
   -- IF com elsif (sem else-if)
   if X > 0 then
      Ada.Text_IO.Put_Line ("Positivo");
   elsif X < 0 then
      Ada.Text_IO.Put_Line ("Negativo");
   else
      Ada.Text_IO.Put_Line ("Zero");
   end if;

   -- FOR range loop (inclusivo em ambas as extremidades)
   for I in 1 .. 10 loop
      Ada.Integer_Text_IO.Put (I);
   end loop;

   -- FOR reverso
   for I in reverse 1 .. 5 loop
      Ada.Integer_Text_IO.Put (I);  -- 5 4 3 2 1
   end loop;

   -- WHILE loop
   while X > 0 loop
      X := X - 1;
   end loop;

   -- LOOP básico com EXIT
   loop
      X := X + 1;
      exit when X >= 100;
   end loop;
end Controle_Demo;

5. Procedures e Functions

Ada distingue rigorosamente entre procedures (subprogramas sem valor de retorno, podem modificar parâmetros) e functions (retornam um valor, não podem ter parâmetros out em Ada 95+; Ada 2012 relaxa parcialmente isso).

ada — modes de parâmetros: in, out, in out
package Matematica is

   -- FUNCTION: retorna valor, parâmetros são "in" por padrão
   function Quadrado (N : Integer) return Integer;

   -- PROCEDURE: sem retorno
   -- "in"     = apenas leitura (padrão)
   -- "out"    = apenas escrita (caller passa variável não inicializada)
   -- "in out" = leitura e escrita (passagem por referência)
   procedure Dividir
      (Dividendo  :     Integer;
       Divisor    :     Integer;
       Quociente  : out Integer;
       Resto      : out Integer);

   -- "in out": modifica o valor original
   procedure Incrementar (N : in out Integer; Passo : Integer := 1);

end Matematica;

package body Matematica is

   function Quadrado (N : Integer) return Integer is
   begin
      return N * N;
   end Quadrado;

   procedure Dividir
      (Dividendo : Integer; Divisor : Integer;
       Quociente : out Integer; Resto : out Integer) is
   begin
      if Divisor = 0 then
         raise Constraint_Error with "Divisor não pode ser zero";
      end if;
      Quociente := Dividendo / Divisor;
      Resto     := Dividendo rem Divisor;
   end Dividir;

   procedure Incrementar (N : in out Integer; Passo : Integer := 1) is
   begin
      N := N + Passo;
   end Incrementar;

end Matematica;

6. Arrays e Strings

Arrays em Ada têm limites verificados e podem usar qualquer tipo discreto como índice, inclusive enumerações. O tipo String é um array de Character com índice do tipo Positive.

ada — arrays constrained e unconstrained
procedure Arrays_Demo is

   -- Array com bounds fixos
   type Vetor_10    is array (1 .. 10) of Integer;
   type Matriz_3x3  is array (1 .. 3, 1 .. 3) of Float;

   -- Array indexado por enumeração
   type Dia_Semana   is (Seg, Ter, Qua, Qui, Sex);
   type Horas_Trabalho is array (Dia_Semana) of Natural;

   Horas : Horas_Trabalho := (Seg => 8, Ter => 8, Qua => 6,
                               Qui => 8, Sex => 4);
   V     : Vetor_10       := (others => 0);  -- todos zeros
   M     : Matriz_3x3     := (others => (others => 0.0));

begin
   V(1) := 42;
   M(2, 3) := 1.5;

   -- Atributos de array
   Ada.Integer_Text_IO.Put (V'First);   -- 1
   Ada.Integer_Text_IO.Put (V'Last);    -- 10
   Ada.Integer_Text_IO.Put (V'Length);  -- 10

   -- Iterar com for-of (Ada 2012+)
   for Elemento of V loop
      Ada.Integer_Text_IO.Put (Elemento);
   end loop;
end Arrays_Demo;
ada — strings e Ada.Strings.Unbounded
with Ada.Text_IO;             use Ada.Text_IO;
with Ada.Strings.Unbounded;  use Ada.Strings.Unbounded;

procedure Strings_Demo is
   -- String de tamanho fixo (como array)
   S1 : String (1 .. 5) := "Hello";

   -- Unbounded_String: tamanho variável (como std::string em C++)
   S2 : Unbounded_String := To_Unbounded_String ("Olá, Mundo");
begin
   Put_Line (S1);
   Put_Line (To_String (S2));

   -- Concatenação
   S2 := S2 & To_Unbounded_String ("!");

   -- Slicing
   Put_Line (S1 (1 .. 3));  -- "Hel"

   -- Length
   Ada.Integer_Text_IO.Put (Length (S2));
end Strings_Demo;

7. Tratamento de Exceções

Ada tem suporte nativo a exceções desde 1983. O mecanismo é estritamente estruturado: exceções são levantadas com raise e capturadas com exception no final de um bloco.

ada — declare blocks e exception handlers
with Ada.Text_IO;   use Ada.Text_IO;
with Ada.Exceptions; use Ada.Exceptions;

procedure Excecoes_Demo is

   -- Exceção personalizada (declarada como constante de tipo Exception)
   Valor_Negativo : exception;

   function Raiz_Quadrada (X : Float) return Float is
   begin
      if X < 0.0 then
         raise Valor_Negativo with "Raiz de negativo: " & Float'Image (X);
      end if;
      return Float'Sqrt (X);  -- Nota: usa Ada.Numerics.Elementary_Functions
   end Raiz_Quadrada;

begin
   declare  -- bloco local com seu próprio handler
      R : Float := Raiz_Quadrada (-4.0);
   begin
      Put_Line ("Raiz: " & Float'Image (R));
   exception
      when Valor_Negativo =>
         Put_Line ("Erro: número negativo");
      when Constraint_Error =>
         Put_Line ("Erro de restrição de tipo");
      when E : others =>
         Put_Line ("Erro inesperado: " & Exception_Message (E));
   end;
end Excecoes_Demo;

8. Records (Estruturas)

O equivalente Ada de struct em C é o record. Ada suporta discriminantes (campos que determinam o "formato" do record) e variant records.

ada — record e record com discriminante
package Geometria is

   -- Record simples
   type Ponto is record
      X, Y : Float := 0.0;
   end record;

   -- Variant record: forma depende do discriminante
   type Forma is (Circulo, Retangulo, Triangulo);

   type Figura (Tipo : Forma) is record
      Centro : Ponto;
      case Tipo is
         when Circulo    => Raio : Float;
         when Retangulo  => Largura, Altura : Float;
         when Triangulo  => Base, Altura_Tri : Float;
      end case;
   end record;

   function Area (F : Figura) return Float;

end Geometria;

-- Uso
with Geometria; use Geometria;
procedure Demo_Record is
   C : Figura := (Tipo => Circulo, Centro => (X => 0.0, Y => 0.0), Raio => 5.0);
   R : Figura := (Tipo => Retangulo, Centro => (X => 1.0, Y => 1.0),
                  Largura => 4.0, Altura => 3.0);
begin
   Ada.Float_Text_IO.Put (Area (C));  -- 78.5398...
   Ada.Float_Text_IO.Put (Area (R));  -- 12.0
end Demo_Record;
