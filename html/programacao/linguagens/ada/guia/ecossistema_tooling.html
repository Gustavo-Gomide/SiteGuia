Ada — Ecossistema e Tooling

Ecossistema Ada

GNAT, Alire, GPS IDE e as Ferramentas do Mundo Ada

Compiladores, gerenciadores de pacotes, IDEs e ferramentas de análise estática.

O ecossistema Ada gira em torno do compilador GNAT (GNU Ada Translator), mantido pela AdaCore e pela comunidade GCC/LLVM. O gerenciador de pacotes Alire modernizou o workflow de dependências. O IDE GNAT Studio (GPS) oferece integração com GNATprove para verificação formal. Ferramentas como GNATcheck e Polyspace completam o pipeline de qualidade exigido por certificações aeronáuticas e ferroviárias.

1. Compiladores Ada

GNAT FSF (GCC)

Compilador open-source integrado ao GCC. Disponível em todas as distribuições Linux via apt install gnat. Suporta até Ada 2012 completamente; Ada 2022 em progresso. Ideal para aprendizado e projetos open-source.

GNAT Pro (AdaCore)

Versão comercial com suporte profissional. Essencial para projetos certificados (DO-178C, EN 50128). Inclui GNAT Studio IDE, GNATprove, GNATcheck, GNATcoverage e suporte a alvos embarcados (VxWorks, QNX, LynxOS).

GNAT LLVM

Backend LLVM para GNAT — em desenvolvimento ativo pela AdaCore. Permite usar as otimizações e alvos do LLVM (WebAssembly, ARM, RISC-V) com Ada.

Janus/Ada, ObjectAda

Compiladores comerciais alternativos. Janus/Ada (RR Software) com foco em conformidade máxima ao padrão. ObjectAda (PTC) para alvos embarcados industriais.

terminal — instalar GNAT e compilar
# Ubuntu/Debian
sudo apt install gnat gprbuild

# macOS com Homebrew
brew install gnat

# Windows: MSYS2
pacman -S mingw-w64-x86_64-gcc-ada

# Compilar arquivo único
gnatmake ola_mundo.adb

# Compilar projeto com gprbuild (recomendado para projetos maiores)
gprbuild -P meu_projeto.gpr

2. Alire — Gerenciador de Pacotes

Alire (Ada Library Repository) é o gerenciador de pacotes moderno do ecossistema Ada, lançado em 2021. Funciona de forma similar ao Cargo (Rust) ou pip (Python): gerencia dependências, builds e ambientes de trabalho.

terminal — workflow com Alire
# Criar novo projeto
alr init --bin meu_app
cd meu_app

# Buscar bibliotecas disponíveis
alr search json
alr search --full http

# Adicionar dependência ao projeto
alr with gnatcoll
alr with simple_components

# Build e execução
alr build
alr run

# Ver dependências do projeto
alr with --graph

# Publicar no índice Alire (Alire.Index no GitHub)
alr publish
toml — alire.toml (manifesto do projeto)
name = "meu_app"
description = "Meu aplicativo Ada"
version = "0.1.0"

[[depends-on]]
gnatcoll = "^22.0"

[[depends-on]]
simple_components = "*"

[build-switches]
release.optimization = ["-O2"]

3. GNAT Studio (GPS IDE)

GNAT Studio é o IDE oficial mantido pela AdaCore. Gratuito e open-source (desde 2020). Integra o compilador, depurador GDB, GNATprove (verificação formal) e GNATcheck (análise de código).

Funcionalidades GPS

  • Navegação de código cross-reference (ir para declaração/uso)
  • Completion com reconhecimento de tipos Ada
  • Integração direta com GNATprove: ver provas inline no editor
  • Visualizador de call graph e dependências de pacotes
  • Suporte a `.gpr` (GNAT Project Files) para targets cross-platform
  • Integração Git/SVN/Mercurial nativa

Extensões VS Code para Ada

  • Ada Language Server (AdaCore) — extension oficial para VS Code com LSP completo
  • Completion, navegação, formatação, diagnósticos em tempo real
  • Instalar: extensão ada-lang-io.ada no VS Code Marketplace
  • Requer GNAT e ada_language_server instalados

4. Bibliotecas Essenciais

GNATColl

Coleção de componentes da AdaCore: acesso a banco de dados (SQLite, PostgreSQL), manipulação de JSON/XML, sistema de arquivos, logging. Disponível via Alire.

Ada.Containers

Biblioteca padrão (ISO) com Vectors, Maps, Sets, Queues, Trees. Análoga à STL de C++, porém com verificação de bounds por padrão.

Simple Components

Vasta coleção de componentes por Dmitry Kazakov: comunicação em rede, protocolos industriais (Modbus, OPC-UA), parsers e estruturas de dados avançadas.

Florist (POSIX)

Binding Ada para a API POSIX: processos, sinais, threads, I/O assíncrono. Fundamental para software de sistema em Unix/Linux escrito em Ada.
