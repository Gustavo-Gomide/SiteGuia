Ruby em GUI Desktop

Aplicacoes Desktop com Ruby

Construcao de interfaces graficas multiplataforma com DSLs e bindings nativos.

Ruby em desktop e menos comum que web, mas continua relevante para ferramentas internas, prototipos ricos e apps utilitarios com baixa friccao de desenvolvimento.

Shoes, FXRuby e wxRuby3

Shoes foi um toolkit historico focado em simplicidade. FXRuby e wxRuby3 oferecem bindings para frameworks GUI maduros, com maior controle de componentes nativos.

Glimmer, Tk e GTK3

Glimmer DSL abstrai detalhes de Tk/SWT/LibUI com uma API Ruby declarativa. O Ruby tambem possui bindings para Tk na stdlib (em alguns ambientes) e GTK3 via ruby-gnome2.

glimmer.rb
require "glimmer-dsl-libui"

include Glimmer

window("Ola", 300, 120) {
  label("Ruby Desktop")
}.show

Empacotamento e Distribuicao

Para distribuir apps, avalie Traveling Ruby (empacotar runtime e dependencias) ou abordagem JVM com JRuby em artefato JAR quando fizer sentido de infraestrutura.

Boas praticas

  • Congelar versoes de gems no Gemfile.lock.
  • Automatizar build por plataforma em CI.
  • Medir startup e consumo de memoria em hardware alvo.
