Terraform e HCL: Avancado

Escalando Terraform em Ambientes Reais

Praticas avancadas para operacao confiavel

Workspaces, state remoto, refatoracao segura e construcao dinamica de recursos.

No nivel avancado, o desafio deixa de ser "fazer funcionar" e passa a ser "evoluir sem quebrar ambiente".

Workspaces e Backends Remotos

Workspaces separam estados para diferentes contextos, como dev e prod. Backends remotos armazenam state com controle de concorrencia e auditoria.

hcl - backend remoto + workspace por sufixo
terraform {
	backend "s3" {
		bucket         = "org-tfstate"
		key            = "network/terraform.tfstate"
		region         = "us-east-1"
		dynamodb_table = "org-tf-locks"
		encrypt        = true
	}
}

locals {
	env = terraform.workspace
}

Lifecycle e Meta-Arguments

Meta-arguments controlam comportamento de recursos: criacao, destruicao, iteracao e dependencias.

hcl - lifecycle, for_each e count
resource "aws_security_group" "svc" {
	name = "svc-${terraform.workspace}"

	lifecycle {
		create_before_destroy = true
		ignore_changes        = [tags]
	}
}

resource "aws_instance" "nodes" {
	for_each      = toset(["api", "worker"])
	ami           = data.aws_ami.ubuntu.id
	instance_type = "t3.micro"
	tags = { Role = each.key }
}

for_each vs count

Prefira for_each quando os itens possuem identidade estavel (chaves). Use count para repeticoes simples baseadas em numero.

Dynamic Blocks, Import e Moved

Esses recursos ajudam a reduzir duplicacao, adotar infraestrutura legada e refatorar sem destruir.

hcl - dynamic block
resource "aws_security_group" "web" {
	name = "web-sg"

	dynamic "ingress" {
		for_each = [80, 443]
		content {
			from_port   = ingress.value
			to_port     = ingress.value
			protocol    = "tcp"
			cidr_blocks = ["0.0.0.0/0"]
		}
	}
}
hcl - import e moved
# Terraform 1.5+: bloco import declarativo
import {
	to = aws_s3_bucket.logs
	id = "bucket-de-logs-legado"
}

moved {
	from = aws_instance.web
	to   = module.compute.aws_instance.web
}

Refatoracao sem moved pode provocar destruicao/recriacao inesperada. Sempre valide com plan.
