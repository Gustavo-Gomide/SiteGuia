Fortran: Bibliotecas e Frameworks

Ecossistema Numerico de Alto Desempenho

As Ferramentas que Sustentam o HPC

Fortran brilha quando combinado com bibliotecas matematicas, paralelismo e toolchains cientificas maduras.

Mais que linguagem, Fortran e um ecossistema de computacao numerica confiavel, previsivel e otimizado.

Bibliotecas Fundamentais

BLAS / LAPACK

Operacoes de algebra linear densas, base de milhares de codigos cientificos.

ScaLAPACK

Extensao distribuida para clusters com MPI.

FFTW e MKL

Transformadas rapidas e kernels otimizados para CPU moderna.

netCDF/HDF5

Persistencia cientifica de alta escala para dados multidimensionais.

Paralelismo no Dia a Dia

OpenMP em loop vetorizavel
! paralelismo compartilhado com OpenMP
!$omp parallel do
do i = 1, n
  c(i) = a(i) + b(i)
end do
!$omp end parallel do

A estrategia classica combina MPI entre nos e OpenMP dentro de cada no de processamento.

Ferramentas Complementares

  • fpm (Fortran Package Manager) para organizacao de dependencias.
  • CMake + Ninja para build reproduzivel multiplataforma.
  • FORD para documentacao automatica de codigo Fortran.
  • pFUnit para testes unitarios cientificos.
