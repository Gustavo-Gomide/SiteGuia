Fortran — HPC e Supercomputação

HPC (High-Performance Computing) em Fortran combina três camadas de paralelismo: MPI para memória distribuída entre nós de cluster, OpenMP para threads dentro de um nó multicore, e vetorização SIMD automática dentro de cada thread. Esse modelo MPI+OpenMP é o padrão na computação científica de alto desempenho.

O benchmark LINPACK — que classifica o TOP500 de supercomputadores — é escrito em Fortran usando BLAS/LAPACK. Em 2024, o supercomputador mais rápido do mundo (Frontier, Oak Ridge, EUA) executa 1.206 exaFLOPS usando código Fortran+MPI+OpenMP como kernel de referência.

MPI — Message Passing Interface

MPI ponto-a-ponto, broadcast, reduce e scatter/gather
program mpi_demo
    use mpi_f08   ! interface moderna Fortran 2008 (vs use mpi)
    implicit none
    integer, parameter :: dp = kind(1.0d0)
    integer :: rank, nprocs, ierr
    real(dp) :: local_sum, global_sum
    real(dp), allocatable :: dados(:), local_dados(:)
    integer :: N, local_N, i

    call MPI_Init(ierr)
    call MPI_Comm_rank(MPI_COMM_WORLD, rank, ierr)   ! rank = id do processo
    call MPI_Comm_size(MPI_COMM_WORLD, nprocs, ierr) ! nprocs = total

    N = 1000
    local_N = N / nprocs

    ! Processo 0 gera dados e distribui (scatter)
    if (rank == 0) then
        allocate(dados(N))
        dados = [(real(i, dp), i=1,N)]
    end if
    allocate(local_dados(local_N))

    call MPI_Scatter(dados, local_N, MPI_DOUBLE_PRECISION, &
                     local_dados, local_N, MPI_DOUBLE_PRECISION, &
                     0, MPI_COMM_WORLD, ierr)

    ! Cada processo calcula soma local
    local_sum = sum(local_dados)

    ! Reduzir (somar) em todos → rank 0
    call MPI_Reduce(local_sum, global_sum, 1, MPI_DOUBLE_PRECISION, &
                    MPI_SUM, 0, MPI_COMM_WORLD, ierr)

    if (rank == 0) then
        print '(A, F12.0)', "Soma total: ", global_sum   ! = 500500
    end if

    ! Broadcast de scalar de rank 0 para todos
    real(dp) :: param = 0.0_dp
    if (rank == 0) param = 3.14_dp
    call MPI_Bcast(param, 1, MPI_DOUBLE_PRECISION, 0, MPI_COMM_WORLD, ierr)
    ! Agora todos os processos têm param = 3.14

    ! Operação coletiva com MPI_Allreduce (resultado em todos os processos)
    call MPI_Allreduce(local_sum, global_sum, 1, MPI_DOUBLE_PRECISION, &
                       MPI_SUM, MPI_COMM_WORLD, ierr)

    call MPI_Finalize(ierr)

end program mpi_demo

! Compilar: mpif90 -O3 mpi_demo.f90 -o mpi_demo
! Executar:  mpirun -np 8 ./mpi_demo

Coarrays — Paralelismo Nativo do Fortran

Coarrays SPMD: imagens, [] e sync all
program coarray_demo
    implicit none
    integer, parameter :: dp = kind(1.0d0)

    ! Coarray: cada "imagem" tem sua cópia de x
    real(dp) :: x[*]          ! escalar coarray
    real(dp) :: v(100)[*]     ! array coarray
    real(dp) :: soma
    integer  :: me, n_imgs, i

    me     = this_image()     ! número da imagem atual (inicia em 1)
    n_imgs = num_images()     ! total de imagens

    ! Cada imagem inicializa seu valor local
    x = real(me, dp) * 10.0_dp

    ! Sincronizar antes de ler valores de outras imagens
    sync all

    ! Imagem 1 lê de todas as outras
    if (me == 1) then
        soma = 0.0_dp
        do i = 1, n_imgs
            soma = soma + x[i]   ! x[i] = x na imagem i
        end do
        print *, "Soma total:", soma
    end if

    ! Comunicação one-sided: imagem 1 escreve em imagem 2
    if (me == 1) x[2] = 999.0_dp
    sync all

    if (me == 2) print *, "x na imagem 2:", x   ! 999.0

    ! Compilar com gfortran: gfortran -fcoarray=single prog.f90  (1 imagem)
    ! Para múltiplas: gfortran -fcoarray=lib prog.f90 -lcaf_mpi
    ! Executar: cafrun -n 4 ./prog (OpenCoarrays)

end program coarray_demo

Modelo Híbrido MPI+OpenMP

Paralelismo em dois níveis: distribuído + threads
program hibrido_mpi_omp
    use mpi_f08
    use omp_lib
    implicit none
    integer :: rank, nprocs, nthreads, tid
    integer :: ierr
    real :: local_result

    ! MPI_Init_thread para suporte a threads dentro de processos MPI
    integer :: provided
    call MPI_Init_thread(MPI_THREAD_FUNNELED, provided, ierr)
    call MPI_Comm_rank(MPI_COMM_WORLD, rank, ierr)
    call MPI_Comm_size(MPI_COMM_WORLD, nprocs, ierr)

    ! Cada processo MPI usa OpenMP internamente
    !$omp parallel private(tid, nthreads)
    tid      = omp_get_thread_num()
    nthreads = omp_get_num_threads()

    !$omp single
    print '(A,I3,A,I3,A,I3)', &
        "MPI rank: ", rank, " de ", nprocs, &
        " | Threads OMP: ", nthreads
    !$omp end single

    ! Trabalho computacional dividido por tid
    !$omp do reduction(+:local_result)
    ! ... processamento intensivo ...
    !$omp end do

    !$omp end parallel

    ! MPI reduz resultados de todos os ranks
    call MPI_Finalize(ierr)

end program hibrido_mpi_omp

O modelo MPI+OpenMP usa um processo MPI por nó NUMA e uma thread OpenMP por core — isso evita overhead de comunicação entre threads de memórias NUMA diferentes via MPI, ao mesmo tempo que maximiza o uso de todos os cores do nó com OpenMP. É o modelo padrão dos 10 supercomputadores mais rápidos do mundo em 2024.
