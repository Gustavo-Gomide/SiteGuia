Exemplo de Programa CLI

Recebendo parametro e executando simulacao
! programa simplificado com argumento de entrada
program sim_cli
  implicit none
  character(len=32) :: arg
  call get_command_argument(1, arg)
  print *, "Parametro:", arg
end program sim_cli

Praticas Avancadas

Checklist de engenharia

  • Registrar versao do compilador e flags no log.
  • Padronizar formato de saida (CSV/JSON) para pos-processamento.
  • Definir codigos de retorno para falhas e convergencia.
