Scala - Introducao

Scala

Uma linguagem JVM que combina orientacao a objetos e programacao funcional com tipagem avancada.

Scala nasceu em 2004, liderada por Martin Odersky, para unir produtividade de alto nivel com robustez de tipos para sistemas de grande escala.

Historia e Contexto

O nome Scala vem de scalable language. Odersky e a equipe do EPFL projetaram a linguagem para ser escalavel em dois sentidos: no tamanho do sistema e no nivel de abstracao, do script curto ao backend distribuido.

Linha do tempo: 2004 (primeira versao), 2011 (consolidacao empresarial), 2016+ (ecossistema FP com Cats/ZIO), 2021 (Scala 3, com Dotty).

JVM + Modelo Hibrido FP/OO

Scala compila para bytecode JVM, aproveitando GC, tooling de producao e interoperabilidade com Java. Ao mesmo tempo, traz recursos funcionais nativos: funcoes de alta ordem, imutabilidade por padrao em colecoes e ADTs com sealed trait.

scala - objeto, funcao e imutabilidade
case class Pedido(id: Int, valor: BigDecimal)

val pedidos = List(Pedido(1, BigDecimal(120)), Pedido(2, BigDecimal(80)))
val total = pedidos.map(_.valor).sum

println(s"Total: $total") // sem mutacao de estado

Essa composicao permite usar heranca e polimorfismo quando faz sentido, mas com forte incentivo para modelagem algebrica e efeitos controlados.

Projetos novos tendem a adotar Scala 3, mas muitas empresas ainda mantem partes em Scala 2.13 por maturidade de dependencias.

Onde Scala e Mais Usada

Backends de alta concorrencia, pipelines de dados com Spark, plataformas financeiras e sistemas onde consistencia de dominio e modelagem tipada sao requisitos centrais.

Plataformas Web

Play Framework e http4s para APIs robustas com tipagem forte.

Dados e IA

Spark e ecossistema Big Data em JVM, com boa integracao em clusters.

Streaming

Akka/Pekko Streams, FS2 e ZIO Streams para processamento continuo.
