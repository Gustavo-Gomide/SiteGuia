ScalaFX e JavaFX

ScalaFX simplifica a API JavaFX com wrappers idiomaticos. Bom para ferramentas internas, dashboards offline e utilitarios de engenharia.

scala - janela minima com ScalaFX
object AppDesktop extends JFXApp3:
	override def start(): Unit =
		stage = new JFXApp3.PrimaryStage:
			title = "ScalaFX"

Interop com JavaFX e Swing

Se uma dependencia existe apenas em Java, use direto. Isso preserva investimentos legados e facilita migracao incremental.

Legado: Swing ainda aparece em sistemas corporativos antigos. Em novos projetos, prefira JavaFX/ScalaFX.
