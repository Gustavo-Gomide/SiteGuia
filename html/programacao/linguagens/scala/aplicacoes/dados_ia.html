Em dados e IA, Scala se destaca por integrar fortemente com Spark e por rodar com eficiencia em plataformas JVM de larga escala.

Spark + MLlib

Spark em Scala oferece API completa de ETL, feature engineering e treino distribuido.

scala - pipeline de classificacao no Spark
val df = spark.read.option("header", "true").csv("treino.csv")
val assembler = new VectorAssembler().setInputCols(Array("x1", "x2")).setOutputCol("features")
val lr = new LogisticRegression().setLabelCol("label")
val model = new Pipeline().setStages(Array(assembler, lr)).fit(df)

Breeze e Smile

Breeze e util para algebra linear, estatistica e otimizacao local. Smile oferece algoritmos classicos de machine learning fora do contexto Spark.

  • Breeze: operacoes vetoriais e matriciais em JVM.
  • Smile: classificacao, regressao, clustering e avaliacao.
