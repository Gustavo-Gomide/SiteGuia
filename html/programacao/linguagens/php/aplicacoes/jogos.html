PHP no Desenvolvimento de Jogos

Backend de jogos online, leaderboards, multiplayer com Ratchet e Swoole

PHP não é uma engine de jogos e nunca deve renderizar gráficos. Mas jogos online precisam de servidores: gerenciamento de estado, leaderboards, loja de itens, autenticação, matchmaking e comunicação em tempo real via WebSockets. Nesse domínio — o backend de jogos — PHP com Ratchet ou Swoole tem papel legítimo.

O Papel do PHP em Jogos

PHP não serve para motores de renderização, engines de física, desenvolvimento de jogos em tempo real com 60 fps, nem para lógica game-loop tight. Para isso: Unity (C#), Unreal (C++), Godot (GDScript/C#).

API REST de Jogo com Laravel

A maioria dos sistemas de jogo online precisam de endpoints para autenticação, submissão de scores, consulta de leaderboards e compra de itens. Laravel é ideal para essa camada.

php — LeaderboardController
class LeaderboardController extends Controller
{
    public function top(): JsonResponse
    {
        // Cache agressivo — leaderboard muda raramente
        $ranking = Cache::remember('leaderboard.top100', 60, fn() =>
            Score::with('jogador:id,nome,avatar')
                ->orderByDesc('pontos')
                ->limit(100)
                ->get()
        );
        return response()->json($ranking);
    }

    public function submit(Request $req): JsonResponse
    {
        $req->validate([
            'pontos'   => 'required|integer|min:0|max:9999999',
            'nivel'    => 'required|integer|min:1',
            'assinatura' => 'required|string', // anti-cheat HMAC
        ]);

        // Validar assinatura HMAC para resistir a manipulação de score
        $esperado = hash_hmac('sha256',
            $req->user()->id . '|' . $req->pontos . '|' . $req->nivel,
            config('game.hmac_secret')
        );

        if (!hash_equals($esperado, $req->assinatura)) {
            return response()->json(['error' => 'Score inválido'], 422);
        }

        $score = Score::updateOrCreate(
            ['jogador_id' => $req->user()->id, 'nivel' => $req->nivel],
            ['pontos' => max($req->pontos, Score::where/* recorde atual */)]
        );

        Cache::forget('leaderboard.top100'); // invalida cache
        return response()->json($score, 201);
    }
}

Multiplayer em Tempo Real com Ratchet

Ratchet é uma biblioteca de WebSockets para PHP baseada em ReactPHP (event loop não bloqueante). Permite criar servidores que mantêm conexões persistentes — essencial para jogos multiplayer.

bash — instalar Ratchet
composer require cboden/ratchet
php — servidor de sala de jogo com Ratchet
use Ratchet\MessageComponentInterface;
use Ratchet\ConnectionInterface;

class SalaDeJogo implements MessageComponentInterface
{
    /** @var SplObjectStorage<ConnectionInterface, array> */
    private SplObjectStorage $clientes;

    public function __construct() {
        $this->clientes = new SplObjectStorage();
    }

    public function onOpen(ConnectionInterface $conn): void
    {
        $this->clientes->attach($conn);
        echo "Novo jogador: {$conn->resourceId}\n";
    }

    public function onMessage(ConnectionInterface $de, $msg): void
    {
        $dados = json_decode($msg, true);

        // Broadcast para todos exceto emissor
        foreach ($this->clientes as $cliente) {
            if ($de !== $cliente) {
                $cliente->send(json_encode([
                    'tipo'     => $dados['tipo'] ?? 'acao',
                    'jogador'  => $de->resourceId,
                    'payload'  => $dados['payload'] ?? [],
                ]));
            }
        }
    }

    public function onClose(ConnectionInterface $conn): void
    {
        $this->clientes->detach($conn);
    }

    public function onError(ConnectionInterface $conn, \Exception $e): void
    {
        error_log($e->getMessage());
        $conn->close();
    }
}
php — inicializar servidor WebSocket na porta 8080
use Ratchet\Server\IoServer;
use Ratchet\Http\HttpServer;
use Ratchet\WebSocket\WsServer;

$server = IoServer::factory(
    new HttpServer(new WsServer(new SalaDeJogo())),
    8080
);

echo "Servidor WebSocket rodando em ws://localhost:8080\n";
$server->run();
