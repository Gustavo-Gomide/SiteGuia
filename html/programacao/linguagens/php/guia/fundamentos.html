PHP — Fundamentos

Fundamentos do PHP

Tipos, variáveis, operadores, controle de fluxo, funções e arrays — a base sólida que toda aplicação PHP exige

Dominar os fundamentos do PHP significa entender como a linguagem trata dados, toma decisões e organiza código em funções reutilizáveis. Esta página cobre todos os elementos básicos com exemplos práticos e comentários sobre as diferenças entre PHP 5, 7 e 8.

Sistema de Tipos

PHP possui oito tipos primitivos. Ao contrário de Java ou C#, os tipos em PHP são determinados dinamicamente — a variável assume o tipo do valor atribuído. Com PHP 7+, é possível declarar tipos explicitamente.

Escalares

  • int — inteiro de 32 ou 64 bits (plataforma-dependente). Ex: 42, -7, 0xFF
  • float — ponto flutuante IEEE 754 de dupla precisão. Ex: 3.14, 1.5e3
  • string — sequência de bytes (não Unicode por padrão; use mbstring para multibyte). Ex: "olá"
  • bool — true ou false

Compostos

  • array — mapa ordenado chave→valor; pode ser indexado ou associativo
  • object — instância de uma classe

Especiais

  • null — ausência de valor
  • resource — referência a recurso externo (conexão DB, arquivo aberto)

Verificação de tipo

php
gettype(42);       // "integer"
gettype("hello"); // "string"
is_int(42);       // true
is_string(42);   // false
var_dump(3.14);  // float(3.14)

Coerção e strict_types

PHP converte tipos automaticamente (coerção). Para desabilitar isso num arquivo e exigir comparação estrita, declare:

php — strict_types
<?php
declare(strict_types=1); // deve ser a primeira declaração do arquivo

function somar(int $a, int $b): int
{
    return $a + $b;
}

somar("5", 3); // TypeError: Argument #1 must be of type int, string given

Variáveis e Constantes

Variáveis

Toda variável PHP começa com $ seguido de um identificador. Não há declaração de tipo obrigatória — o tipo é inferido do valor.

php — variáveis
$nome   = "Alice";         // string
$idade  = 30;              // int
$altura = 1.72;           // float
$ativo  = true;           // bool
$nada   = null;           // null

// Interpolação de string com aspas duplas
echo "Olá, {$nome}! Você tem {$idade} anos.\n";

// Strings com aspas simples NÃO interpolam variáveis
echo 'Olá, $nome'; // imprime literalmente $nome

Constantes

Constantes são identificadores cujo valor não muda após definição. PHP oferece duas formas:

php — constantes
// define(): funciona em qualquer escopo, inclusive dentro de if
define('VERSAO', '2.0.1');
echo VERSAO; // 2.0.1

// const: declarada em tempo de compilação (só no escopo global ou dentro de classe)
const MAX_TENTATIVAS = 5;

// Constantes mágicas (predefinidas pelo PHP)
echo __FILE__;   // caminho absoluto do arquivo atual
echo __LINE__;   // número da linha atual
echo __CLASS__;  // nome da classe atual
echo __DIR__;    // diretório do arquivo atual

php — spaceship e null coalescing
// Spaceship: retorna -1, 0 ou 1
echo 1 <=> 2; // -1  (left < right)
echo 2 <=> 2; //  0  (igual)
echo 3 <=> 2; //  1  (left > right)

// Útil em usort/uasort
usort($arr, fn($a, $b) => $a <=> $b);

// Null coalescing encadeado
$valor = $_GET['param'] ?? $_POST['param'] ?? 'padrão';

Controle de Fluxo

if / elseif / else

php — condicionais
$nota = 75;

if ($nota >= 90) {
    echo 'A';
} elseif ($nota >= 70) {
    echo 'B';
} elseif ($nota >= 50) {
    echo 'C';
} else {
    echo 'Reprovado';
}

match (PHP 8.0)

Prefira match ao switch quando quiser comparação estrita e um valor de retorno:

php — match
$cor = 'vermelho';
$hex = match($cor) {
    'vermelho' => '#FF0000',
    'verde'    => '#00FF00',
    'azul'     => '#0000FF',
    default    => '#000000',
};
echo $hex; // #FF0000

Loops

php — for, foreach, while, do-while
// for — quando se conhece o número de iterações
for ($i = 0; $i < 5; $i++) {
    echo $i . ' ';
}
// 0 1 2 3 4

// foreach — ideal para arrays e iteráveis
$frutas = ['maçã', 'banana', 'laranja'];
foreach ($frutas as $indice => $fruta) {
    echo "{$indice}: {$fruta}\n";
}

// while — enquanto condição for verdadeira
$n = 1;
while ($n <= 3) {
    echo $n++;
}

// do-while — executa ao menos uma vez
do {
    echo "executado!";
} while (false);

Funções

Funções em PHP são cidadãos de primeira classe a partir do PHP 5.3. Suportam type hints, valores padrão, variáveis variádicas e podem ser anônimas.

php — funções com type hints
// Função com type hints e return type (PHP 7+)
function calcularImc(float $peso, float $altura): float
{
    return $peso / ($altura ** 2);
}

// Tipo nullable: aceita float OU null
function encontrar(string $busca): ?string
{
    return $busca !== '' ? $busca : null;
}

// Union types (PHP 8.0): int|string
function processar(int|string $id): string
{
    return "ID: " . $id;
}

// Funções variádicas (...$args)
function somar(int ...$numeros): int
{
    return array_sum($numeros);
}
echo somar(1, 2, 3, 4); // 10
php — closures e arrow functions
// Closure: captura variáveis do escopo externo com 'use'
$fator = 3;
$multiplicar = function(int $n) use ($fator): int {
    return $n * $fator;
};
echo $multiplicar(7); // 21

// Arrow Function (PHP 7.4): captura automaticamente por valor
$dobrar = fn(int $n) => $n * 2;
echo $dobrar(5); // 10

// Uso com array_map
$numeros = [1, 2, 3, 4];
$dobrados = array_map(fn($n) => $n * 2, $numeros);
// [2, 4, 6, 8]

Arrays

Em PHP, arrays são mapas ordenados de chave→valor. A chave pode ser int ou string. Isso significa que arrays indexados, associativos e multidimensionais são todos o mesmo tipo de dado internamente.

php — tipos de array
// Array indexado (chaves inteiras automáticas)
$cores = ['vermelho', 'verde', 'azul'];
echo $cores[0]; // vermelho

// Array associativo (chaves string)
$pessoa = [
    'nome'  => 'Alice',
    'idade' => 30,
    'ativo' => true,
];
echo $pessoa['nome']; // Alice

// Array multidimensional
$tabela = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
];
echo $tabela[1][2]; // 6

Funções de Array mais usadas

php — funções de array
$nums = [3, 1, 4, 1, 5, 9];

// Filtrar
$pares = array_filter($nums, fn($n) => $n % 2 === 0); // [4]

// Transformar (map)
$dobrados = array_map(fn($n) => $n * 2, $nums);

// Reduzir (fold)
$soma = array_reduce($nums, fn($carry, $item) => $carry + $item, 0);

// Ordenar
sort($nums);        // re-indexa: [1, 1, 3, 4, 5, 9]
asort($nums);       // mantém chaves originais
ksort($pessoa);     // ordena por chave

// Busca
in_array(5, $nums);             // true
array_search(4, $nums);        // índice do valor

// Combinar arrays
array_merge($a, $b);           // concatena
array_combine($chaves, $vals); // cria array associativo

// Spread operator (PHP 7.4+)
$fusao = [...$a, ...$b];

array_map não preserva chaves associativas quando recebe apenas um array. Use array_walk ou uma arrow function com array_combine se precisar preservá-las.
