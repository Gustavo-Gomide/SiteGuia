Bibliotecas e Frameworks TypeScript

Zod para validacao runtime com inferencia de tipos, tRPC para APIs type-safe end-to-end, Hono como framework ultra-rapido e React com tipagem avancada.

O diferencial do ecossistema TypeScript sao bibliotecas que expoe APIs totalmente tipadas. tRPC elimina a necessidade de OpenAPI/GraphQL ao compartilhar o tipo do router entre cliente e servidor. Zod infere o tipo TS automaticamente do schema de validacao.

Zod: Validacao e Inferencia

TypeScript + Zod — Schema, inferencia, refinements e transform
import { z } from "zod";

// Definir schema e inferir tipo automaticamente
const UsuarioSchema = z.object({
  id:       z.string().uuid(),
  nome:     z.string().min(2).max(80),
  email:    z.string().email(),
  idade:    z.number().int().min(0).max(120),
  papel:    z.enum(["admin", "editor", "leitor"]).default("leitor"),
  criadoEm: z.coerce.date(), // converte string para Date
});

// Inferir tipo sem duplicar
type Usuario = z.infer<typeof UsuarioSchema>;

// Para criar: omitir id (gerado pelo servidor)
const CriarUsuarioSchema = UsuarioSchema.omit({ id: true, criadoEm: true });
type CriarUsuario = z.infer<typeof CriarUsuarioSchema>;

// Refinement customizado (logica de negocio)
const SenhaSchema = z.string()
  .min(8, "Minimo 8 caracteres")
  .refine(v => /[A-Z]/.test(v), "Precisa de maiuscula")
  .refine(v => /[0-9]/.test(v), "Precisa de numero")
  .refine(v => /[!@#$%]/.test(v), "Precisa de simbolo");

// Transform: validar e transformar em uma etapa
const SlugSchema = z.string()
  .transform(s => s.toLowerCase().replace(/\s+/g, "-").replace(/[^a-z0-9-]/g, ""))
  .pipe(z.string().min(1));

// safeParse retorna { success, data } ou { success, error }
const resultado = UsuarioSchema.safeParse({
  id: "123e4567-e89b-12d3-a456-426614174000",
  nome: "Alice", email: "alice@exemplo.com",
  idade: 30, criadoEm: "2024-01-15",
});
if (resultado.success) {
  const usuario: Usuario = resultado.data; // totalmente tipado
} else {
  console.error(resultado.error.flatten());
}

tRPC: API Type-Safe End-to-End

TypeScript + tRPC — Router tipado com Zod e cliente React Query
// server/router.ts — router do servidor
import { initTRPC } from "@trpc/server";
import { z }       from "zod";

const t = initTRPC.context<{ userId?: string }>().create();

const middleware = t.middleware(async ({ ctx, next }) => {
  if (!ctx.userId) throw new t.TRPCError({ code: "UNAUTHORIZED" });
  return next({ ctx: { ...ctx, userId: ctx.userId } });
});

const procedurePublica    = t.procedure;
const procedureProtegida  = t.procedure.use(middleware);

export const appRouter = t.router({
  usuarios: t.router({
    listar: procedurePublica.query(async () => {
      return [{ id: "1", nome: "Alice", email: "alice@ex.com" }];
    }),
    porId: procedurePublica
      .input(z.object({ id: z.string().uuid() }))
      .query(async ({ input }) => {
        return { id: input.id, nome: "Alice" };
      }),
    criar: procedureProtegida
      .input(z.object({
        nome:  z.string().min(2),
        email: z.string().email(),
      }))
      .mutation(async ({ input, ctx }) => {
        console.log(`Criado por ${ctx.userId}`);
        return { id: crypto.randomUUID(), ...input };
      }),
  }),
});

export type AppRouter = typeof appRouter;

Hono: Framework Web Ultra Rapido

TypeScript + Hono — API tipada com validators e middleware
import { Hono }                from "hono";
import { zValidator }         from "@hono/zod-validator";
import { jwt, sign, verify } from "hono/jwt";
import { z }                  from "zod";

type Variaveis = { userId: string };
const app = new Hono<{ Variables: Variaveis }>();

const CriarPostSchema = z.object({
  titulo:  z.string().min(3),
  conteudo: z.string().min(10),
});

// Middleware JWT
app.use("/api/*", jwt({ secret: "meu-segredo" }));

// Rota publica
app.get("/", (c) => c.json({ ok: true }));

// Criar post com validacao Zod automatica
app.post("/api/posts",
  zValidator("json", CriarPostSchema),
  (c) => {
    const body = c.req.valid("json"); // totalmente tipado
    return c.json({ id: crypto.randomUUID(), ...body }, 201);
  }
);

// Funciona no Node.js, Bun, Deno, Cloudflare Workers, AWS Lambda
export default app;

// Node.js: serve(app, { port: 3000 })
// Bun:     export default app
// Cloudflare Workers: export default app
