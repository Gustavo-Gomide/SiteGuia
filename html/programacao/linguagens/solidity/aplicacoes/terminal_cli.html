Solidity em Terminal CLI

Automacao por linha de comando

Deploy e operacao com Foundry e Hardhat

CLI e a forma mais reproduzivel de compilar, testar, publicar e interagir com contratos.

Em times profissionais, o terminal vira "fonte da verdade" do pipeline tecnico.

Foundry CLI

Comandos centrais: `forge build`, `forge test`, `forge script` e `cast` para consultas/chamadas. Excelente para automacao CI/CD.

Bash
# Pipeline minimo de build e teste
forge build
forge test
forge script script/Deploy.s.sol:Deploy --rpc-url $RPC --broadcast

Hardhat CLI

Hardhat oferece tasks customizadas e deploy scripts em JavaScript/TypeScript.

Bash
# Rotina equivalente com ecossistema Node.js
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.ts --network sepolia

Boas praticas de automacao

Segredo e ambiente

Use variaveis de ambiente, nunca chave privada hardcoded.

Reprodutibilidade

Versao fixa do compilador, lockfile e scripts idempotentes.

Sempre teste em testnet e rode simulacao local antes de `--broadcast` no mainnet.
