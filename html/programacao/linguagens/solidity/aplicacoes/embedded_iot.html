Identidade de dispositivos

Cada dispositivo pode ter chave criptografica e identificador on-chain. O contrato registra status (ativo, revogado, atualizado), permitindo verificacao publica.

Registro imutavel de eventos

Em vez de armazenar telemetria completa on-chain, publica-se hash de lotes de dados. Isso prova integridade sem custo excessivo.

Solidity
// Registro de lote para auditoria posterior
event LoteTelemetria(bytes32 loteHash, uint256 timestamp);

function registrarLote(bytes32 loteHash) external {
	require(loteHash != bytes32(0), "lote invalido");
	emit LoteTelemetria(loteHash, block.timestamp);
}
