BASIC — Ecossistema e Tooling

O ecossistema BASIC é fragmentado por dialetos, mas cada dialeto tem ferramentas maduras: o QB64 Phoenix Edition inclui IDE, compilador e emulador DOS integrados; o FreeBASIC usa GCC como back-end e funciona com qualquer editor de texto; o Visual Basic .NET tem o ecossistema completo do Visual Studio com IntelliSense, depurador, NuGet e .NET SDK.

A escolha do dialeto dita o ecossistema. Para projetos legados dos anos 90, QB64 é a ponte ideal. Para software empresarial Windows novo, VB.NET é a escolha natural. Para programação de sistemas ou games indie no Linux, FreeBASIC compila binários nativos rápidos sem dependências de runtime.

Instalação e Setup

Instalação — FreeBASIC, QB64 e VB.NET
# ── FreeBASIC no Linux (Ubuntu/Debian) ─────────────────
sudo apt update
sudo apt install freebasic          # instala fbc (FreeBASIC Compiler)
fbc --version                       # verificar: FreeBASIC Compiler 1.x.x

# Compilar e rodar
fbc hello.bas                       # gera executável ./hello
./hello

# Com otimização
fbc -O 2 meu_programa.bas

# Gerar biblioteca compartilhada
fbc -lib minha_lib.bas              # gera libminha_lib.a

# ── FreeBASIC no Windows ────────────────────────────────
# Baixar installer de https://www.freebasic.net/downloads
# Após extrair:
fbc.exe hello.bas
hello.exe

# ── QB64 Phoenix Edition ────────────────────────────────
# https://github.com/QB64-Phoenix-Edition/QB64pe/releases
# Extrair ZIP, rodar qb64pe.exe (IDE integrada)
# Ou usar o compilador de linha de comando:
qb64pe -c hello.bas                 # compilar para C++ e linkar

# ── VB.NET via .NET SDK ─────────────────────────────────
# https://dotnet.microsoft.com/download
dotnet new console -lang "VB" -o MeuProjeto
cd MeuProjeto
dotnet run                          # compilar e executar
dotnet build -c Release             # build de release
dotnet publish -r win-x64 --self-contained   # binário standalone

Opções do Compilador FreeBASIC

fbc — opções de compilação
# Compilação básica
fbc programa.bas                    # gera ./programa (Linux) ou programa.exe

# Especificar saída
fbc -o bin/app programa.bas

# Dialeto
fbc -lang qb programa.bas           # compatibilidade QB/QuickBASIC
fbc -lang fblite programa.bas       # modo intermediário
fbc -lang fb programa.bas           # padrão FreeBASIC moderno

# Otimizações
fbc -O 0 programa.bas               # sem otimização (debug)
fbc -O 2 programa.bas               # otimização padrão
fbc -O 3 programa.bas               # máxima otimização

# Gerar código de 64 bits
fbc -gen gcc -arch 64bit programa.bas

# Incluir diretórios
fbc -i include/ programa.bas

# Linkar com biblioteca
fbc -l SDL2 jogo.bas                # linka -lSDL2
fbc -p /usr/local/lib programa.bas  # caminho de busca de bibliotecas

# Debug Info (GDB)
fbc -g programa.bas
gdb ./programa

Desenvolvimento VBA no Office

VBA — Editor e ferramentas do Office
' Abrir editor VBA: Alt+F11 (Excel/Word/Access)
' ── Estrutura de um módulo VBA ─────────────────────────

Option Explicit    ' exige declaração de todas as variáveis (recomendado)

' Macro acessível via Alt+F8
Sub OrganizarDados()
    Dim ws As Worksheet
    Dim ultima_linha As Long

    ws = ThisWorkbook.Sheets("Dados")
    ultima_linha = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row

    Dim i As Long
    For i = 2 To ultima_linha
        If ws.Cells(i, 1).Value = "" Then
            ws.Rows(i).Delete
            i = i - 1
            ultima_linha = ultima_linha - 1
        End If
    Next i

    MsgBox "Dados organizados! " & (ultima_linha - 1) & " linhas."
End Sub

' ── Ferramentas de depuração VBA ───────────────────────
' F9         → Breakpoint
' F8         → Step Into (linha a linha)
' Shi+F8     → Step Over (pular sub)
' Ctrl+G     → Immediate Window (digitar ? variavel = ver valor)
' Watches    → View > Watch Window (monitorar expressões)
' Debug.Print variavel  → imprime na Immediate Window
