BASIC: Banco de Dados

Persistencia no Ecossistema BASIC

Do VBA ao VB.NET com SQL Confiavel

BASIC possui longa historia com bancos de dados, desde automacoes Access/Excel ate APIs corporativas em .NET.

A boa pratica e universal: dados devem ser tratados com transacao, parametros SQL e observabilidade, independentemente do dialeto BASIC usado.

Conceitos Essenciais

Em VBA, o foco costuma ser automacao operacional com Access/Excel e conexoes ADO/DAO. Em VB.NET, o padrao principal e ADO.NET com providers modernos.

Para sistemas maiores, adote repositorios e separacao de camadas para reduzir acoplamento e facilitar testes.

Exemplo com Parametros em VB.NET

vbnet — insert seguro com SqlParameter
Imports System.Data.SqlClient

Public Sub SalvarCliente(nome As String)
    Using conn As New SqlConnection("Server=.;Database=ERP;Trusted_Connection=True;")
        conn.Open()
        Using cmd As New SqlCommand("INSERT INTO clientes(nome) VALUES (@nome)", conn)
            cmd.Parameters.AddWithValue("@nome", nome)
            cmd.ExecuteNonQuery()
        End Using
    End Using
End Sub

' Parametros reduzem risco de SQL injection e erros de parsing.

Topicos Avancados

Praticas para producao

  • Transacoes explicitas em operacoes compostas.
  • Indexacao baseada em plano de consulta real.
  • Retries controlados para falhas transientes.
  • Migracoes versionadas com rollback.

Nao concatene SQL com dados de usuario. Esse erro ainda e comum em scripts VBA antigos.
