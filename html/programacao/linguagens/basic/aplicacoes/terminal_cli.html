BASIC — Terminal e CLI

O FreeBASIC produz executáveis de linha de comando nativos — sem runtime, sem VM. Um programa FreeBASIC pode ler argumentos da linha de comando, processar arquivos de texto e binários, interagir com pipes do sistema operacional, e chamar programas externos via Shell ou Exec. É uma escolha prática para scripts de automação compilados mais rápidos que Python ou Bash.

O QB64 também gera executáveis nativos (via conversão para C++) que rodam no terminal, mantendo total compatibilidade com programas originais do MS-DOS que usavam INPUT, PRINT e operações de arquivo.

Argumentos de Linha de Comando

FreeBASIC — Command, Command$, e parsing de argumentos
' FreeBASIC: ler argumentos da linha de comando
' Executar: ./programa arg1 arg2 "texto com espaços"

Dim num_args As Integer

' __FB_ARGC__ e __FB_ARGV__ são macros que espelham argc/argv do C
' Mas a forma idiomática usa Command() e Command$

' Command(n) retorna o n-ésimo argumento (0 = nome do programa)
num_args = 0
Do While Command(num_args + 1) <> ""
    num_args = num_args + 1
Loop

Print "Programa: " & Command(0)
Print "Argumentos: " & num_args

Dim i As Integer
For i = 1 To num_args
    Print "  arg[" & i & "] = " & Command(i)
Next i

' Command$() — retorna TODOS os argumentos como uma única string
Print "Todos: " & Command$()

' ── Programa utilitário: contar linhas de um arquivo ──
' Uso: ./contar_linhas arquivo.txt
Dim arquivo As String = Command(1)
If arquivo = "" Then
    Print "Uso: contar_linhas <arquivo>"
    End 1    ' exit code 1
End If

Dim arq As Integer = FreeFile()
Open arquivo For Input As #arq
If Err <> 0 Then
    Print "Erro ao abrir: " & arquivo
    End 1
End If

Dim linhas As Long = 0
Dim buf    As String
Do While Not EOF(arq)
    Line Input #arq, buf
    linhas = linhas + 1
Loop
Close #arq
Print linhas & " linhas em " & arquivo

Processamento de Texto e CSV

Parser de CSV simples em FreeBASIC
' Ler e parsear CSV
' Formato: id,nome,salario

Type TFuncionario
    id      As Integer
    nome    As String
    salario As Double
End Type

' Função para extrair campo de uma linha CSV
Function CampoCSV(linha As String, indice As Integer) As String
    Dim campo_atual As Integer = 0
    Dim inicio      As Integer = 1
    Dim pos         As Integer

    Do While campo_atual < indice
        pos = InStr(inicio, linha, ",")
        If pos = 0 Then
            CampoCSV = ""
            Exit Function
        End If
        inicio = pos + 1
        campo_atual = campo_atual + 1
    Loop

    pos = InStr(inicio, linha, ",")
    If pos = 0 Then
        CampoCSV = Mid(linha, inicio)
    Else
        CampoCSV = Mid(linha, inicio, pos - inicio)
    End If
End Function

' Ler arquivo CSV
Dim arq As Integer = FreeFile()
Open "funcionarios.csv" For Input As #arq

Dim buf  As String
Dim func As TFuncionario
Dim total_salario As Double = 0
Dim count As Integer = 0
Dim linha_num As Integer = 0

Do While Not EOF(arq)
    Line Input #arq, buf
    linha_num = linha_num + 1
    If linha_num = 1 Then GoTo ProximaLinha  ' pular cabeçalho

    func.id      = CInt(CampoCSV(buf, 0))
    func.nome    = CampoCSV(buf, 1)
    func.salario = CDbl(CampoCSV(buf, 2))

    total_salario = total_salario + func.salario
    count = count + 1
    Print func.id & ": " & func.nome & " — R$" & func.salario

    ProximaLinha:
Loop
Close #arq

If count > 0 Then
    Print "Média salarial: R$" & Format(total_salario / count, "0.00")
End If

Executar Comandos do Sistema

Shell — executar programas externos
' FreeBASIC — Shell() para executar comandos do SO

' Executar e aguardar (retorna exit code)
Dim ret As Integer
ret = Shell("ls -la")                    ' Linux
ret = Shell("dir /b")                   ' Windows
Print "Exit code: " & ret

' Com redirecionamento de saída
ret = Shell("find /etc -name '*.conf' > result.txt 2>&1")

' Capturar saída de um comando via arquivo temporário
Function CapturarComando(cmd As String) As String
    Dim tmp_file As String = "/tmp/fb_out_" & Timer & ".txt"
    Shell cmd & " > " & tmp_file

    Dim arq As Integer = FreeFile()
    Dim resultado As String = ""
    Dim buf As String

    Open tmp_file For Input As #arq
    Do While Not EOF(arq)
        Line Input #arq, buf
        resultado = resultado & buf & Chr(10)
    Loop
    Close #arq
    Kill tmp_file    ' deletar arquivo temporário
    CapturarComando = resultado
End Function

Dim saida As String = CapturarComando("uname -a")
Print saida

FreeFile() é uma função crucial no BASIC que retorna o menor número de handle de arquivo disponível. Sempre use FreeFile() em vez de números hardcoded como #1 — isso evita conflitos quando múltiplos arquivos são abertos simultaneamente no mesmo programa.
