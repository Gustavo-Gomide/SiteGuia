Java para GUI Desktop

JavaFX 21 com FXML, injecao de controlador via @FXML, propriedades observaveis, TableView com editable cells, Task e Service para operacoes em background, CSS para estilizacao, e Swing para manutencao de sistemas legados.

JavaFX e o toolkit moderno para aplicacoes desktop Java. Usa arquitetura MVC natural com FXML como view, separando completamente a logica da apresentacao. Rodavel standalone com JavaFX runtime empacotado via jpackage.

JavaFX — Application, Stage e Scene

Java + JavaFX — Application com FXML e controlador injetado
// Ponto de entrada da aplicacao
public class LojaApp extends Application {

  @Override
  public void start(Stage stage) throws Exception {
    var fxmlLoader = new FXMLLoader(
      getClass().getResource("/views/principal.fxml"));
    Parent root = fxmlLoader.load();

    var scene = new Scene(root, 1280, 720);
    scene.getStylesheets().add(
      getClass().getResource("/css/tema.css").toExternalForm());

    stage.setTitle("Loja Desktop");
    stage.setScene(scene);
    stage.setMinWidth(800);
    stage.setMinHeight(600);
    stage.show();
  }

  public static void main(String[] args) {
    launch(args);
  }
}

Controlador FXML e ObservableList

Java + JavaFX — controlador com TableView, bindings e propriedades
public class PrincipalController implements Initializable {

  @FXML private TableView<Produto>          tabelaProdutos;
  @FXML private TableColumn<Produto, String>     colNome;
  @FXML private TableColumn<Produto, BigDecimal> colPreco;
  @FXML private TableColumn<Produto, Integer>   colEstoque;
  @FXML private TextField                     campoBusca;
  @FXML private Label                         lblStatus;

  private final ObservableList<Produto> produtos =
    FXCollections.observableArrayList();
  private FilteredList<Produto> produtosFiltrados;

  @Override
  public void initialize(URL url, ResourceBundle rb) {
    // Mapeia propriedades para as colunas
    colNome   .setCellValueFactory(new PropertyValueFactory<>("nome"));
    colPreco  .setCellValueFactory(new PropertyValueFactory<>("preco"));
    colEstoque.setCellValueFactory(new PropertyValueFactory<>("estoque"));

    // Coluna de preco: celula customizada com formatacao BR
    colPreco.setCellFactory(col -> new TableCell<>() {
      private final NumberFormat fmt =
        NumberFormat.getCurrencyInstance(new Locale("pt", "BR"));
      @Override
      protected void updateItem(BigDecimal item, boolean empty) {
        super.updateItem(item, empty);
        setText(empty || item == null ? null : fmt.format(item));
      }
    });

    // Filtro em tempo real enquanto digita
    produtosFiltrados = new FilteredList<>(produtos, p -> true);
    campoBusca.textProperty().addListener((obs, old, novo) -> {
      produtosFiltrados.setPredicate(p ->
        novo == null || novo.isBlank()
          ? true
          : p.getNome().toLowerCase().contains(novo.toLowerCase()));
    });

    var sortedList = new SortedList<>(produtosFiltrados);
    sortedList.comparatorProperty().bind(tabelaProdutos.comparatorProperty());
    tabelaProdutos.setItems(sortedList);

    carregarProdutos();
  }

  @FXML
  private void carregarProdutos() {
    // Task executa em thread separada, atualiza UI na JavaFX thread
    var task = new Task<List<Produto>>() {
      @Override
      protected List<Produto> call() {
        updateMessage("Carregando produtos...");
        return ProdutoService.listarTodos();
      }
    };

    task.setOnSucceeded(e -> {
      produtos.setAll(task.getValue());
      lblStatus.setText(produtos.size() + " produtos carregados");
    });
    task.setOnFailed(e ->
      lblStatus.setText("Erro: " + task.getException().getMessage()));

    lblStatus.textProperty().bind(task.messageProperty());
    new Thread(task, "carregar-produtos").start();
  }
}

Swing — Manutencao de Sistemas Legados

Java + Swing — JFrame, SwingWorker e look-and-feel
public class TelaPrincipalSwing {

  public static void main(String[] args) {
    // SEMPRE iniciar Swing na EDT (Event Dispatch Thread)
    SwingUtilities.invokeLater(TelaPrincipalSwing::criarUI);
  }

  private static void criarUI() {
    try {
      // FlatLaf: look-and-feel moderno (import com.formdev.flatlaf.*)
      FlatDarkLaf.setup();
    } catch (Exception e) {
      UIManager.put("Button.arc", 8);   // fallback
    }

    var frame = new JFrame("Loja Desktop");
    frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
    frame.setSize(900, 600);

    // Layout principal
    var painel = new JPanel(new BorderLayout(8, 8));
    painel.setBorder(BorderFactory.createEmptyBorder(8, 8, 8, 8));

    // Tabela
    var modelo = new ProdutoTableModel();
    var tabela = new JTable(modelo);
    tabela.setRowHeight(28);
    tabela.getTableHeader().setReorderingAllowed(false);
    painel.add(new JScrollPane(tabela), BorderLayout.CENTER);

    // Barra de botoes
    var barraBtn = new JPanel(new FlowLayout(FlowLayout.LEFT));
    var btnCarregar = new JButton("Carregar");
    var progressBar = new JProgressBar();
    progressBar.setIndeterminate(false);
    barraBtn.add(btnCarregar);
    barraBtn.add(progressBar);
    painel.add(barraBtn, BorderLayout.SOUTH);

    btnCarregar.addActionListener(e -> {
      progressBar.setIndeterminate(true);
      btnCarregar.setEnabled(false);

      // SwingWorker — tarefa pesada em background, atualiza EDT ao concluir
      new SwingWorker<List<Produto>, Void>() {
        @Override
        protected List<Produto> doInBackground() {
          return ProdutoService.listarTodos();   // fora da EDT
        }
        @Override
        protected void done() {
          try {
            modelo.setProdutos(get());   // de volta na EDT
          } catch (Exception ex) {
            JOptionPane.showMessageDialog(frame, "Erro: " + ex.getMessage(),
              "Erro", JOptionPane.ERROR_MESSAGE);
          } finally {
            progressBar.setIndeterminate(false);
            btnCarregar.setEnabled(true);
          }
        }
      }.execute();
    });

    frame.add(painel);
    frame.setLocationRelativeTo(null);
    frame.setVisible(true);
  }
}

JavaFX — Quando usar

  • Novos projetos desktop Java
  • Precisa de CSS e animacoes suaves
  • Binding bidirecional de propriedades
  • Charts e graficos nativos (LineChart, BarChart...)
  • WebView com Webkit embutido

Swing — Quando ainda usar

  • Manutencao de sistema legado em Swing
  • IDE plugins (NetBeans, IntelliJ usam Swing)
  • Dependencias corporativas ja em Swing
  • Look-and-feel do SO mais integrado (via FlatLaf)

Empacotamento com jpackage

Shell — criar instalador nativo com jpackage (Java 14+)
# Windows: gera .exe ou .msi com JRE embutido
jpackage \
  --input target/ \
  --name "Loja Desktop" \
  --main-jar loja-desktop.jar \
  --main-class com.loja.LojaApp \
  --type msi \
  --win-dir-chooser \
  --win-shortcut \
  --win-menu \
  --app-version 1.0.0 \
  --vendor "Minha Empresa" \
  --icon src/main/resources/icon.ico \
  --java-options "--add-opens javafx.graphics/com.sun.javafx.css=ALL-UNNAMED"

# Linux: gera .deb ou .rpm
jpackage --type deb --linux-shortcut ...

# macOS: gera .dmg ou .pkg
jpackage --type dmg --mac-package-name "Loja Desktop" ...
