Clojure — Dados e IA

Clojure está emergindo como linguagem de ciência de dados séria: tech.ml.dataset oferece DataFrames imutáveis; Tablecloth fornece uma API fluente em cima deles; Fastmath dá estatística e álgebra linear; e Neanderthal conecta-se ao BLAS/LAPACK nativo para operações vetorizadas de alto desempenho.

A imutabilidade de Clojure é uma vantagem em ciência de dados: pipelines de transformação de dados são sequências de funções puras — fáceis de testar, reproduzir e depurar no REPL linha a linha.

Tablecloth — DataFrame com API fluente
(ns analise
  (:require [tablecloth.api :as tc]))

; ── Carregar dados ────────────────────────────────────────────
(def ds
  (tc/dataset "dados/vendas.csv" {:key-fn keyword}))

(tc/info ds)           ; shape, dtypes, missing
(tc/head ds 5)         ; primeiras 5 linhas
(tc/shape ds)          ; [n-linhas n-colunas]

; ── Selecionar e filtrar ──────────────────────────────────────
(-> ds
    (tc/select-columns [:produto :valor :regiao])
    (tc/select-rows #(> (get % :valor) 1000))
    (tc/order-by :valor :desc))

; ── Agregar por grupo ─────────────────────────────────────────
(-> ds
    (tc/group-by :regiao)
    (tc/aggregate {:total-vendas #(reduce + (:valor %))
                   :n-vendas     #(count   (:valor %))
                   :ticket-medio #(/ (reduce + (:valor %))
                                       (count   (:valor %)))}))

; ── Adicionar coluna derivada ─────────────────────────────────
(-> ds
    (tc/add-column :valor-com-imposto
                   (fn [row] (* (get row :valor) 1.12)))
    (tc/rename-columns {:produto :nome-produto}))

; ── Join entre datasets ───────────────────────────────────────
(tc/left-join ds clientes-ds :cliente-id)

Fastmath — Estatística e Álgebra Linear

Estatísticas descritivas e distribuições
(ns stats
  (:require [fastmath.stats  :as st]
            [fastmath.random :as rng]))

(def dados [23.1 25.6 22.9 28.4 24.7 26.0 21.5 29.3])

; ── Estatísticas descritivas ─────────────────────────────────
(st/mean     dados)   ; média aritmética
(st/median   dados)   ; mediana
(st/variance dados)   ; variância amostral
(st/stddev   dados)   ; desvio padrão
(st/skewness dados)   ; assimetria
(st/kurtosis dados)   ; curtose

; Quantis e box-plot stats
(st/percentile dados 25)    ; Q1
(st/percentile dados 75)    ; Q3

; ── Correlação e covariância ──────────────────────────────────
(st/correlation  x-data y-data)   ; Pearson r
(st/spearman     x-data y-data)   ; Spearman ρ
(st/covariance   x-data y-data)

; ── Distribuições ─────────────────────────────────────────────
(def normal (rng/distribution :normal {:mu 0 :sigma 1}))
(rng/sample  normal)           ; uma amostra
(rng/sample  normal 1000)     ; 1000 amostras
(rng/pdf     normal 1.5)      ; densidade em x=1.5
(rng/cdf     normal 1.96)     ; P(X ≤ 1.96)

; ── Teste t ──────────────────────────────────────────────────
(st/t-test grupo-a grupo-b)   ; retorna {:t :p-value :df ...}

Machine Learning com scicloj/clay

Pipeline de ML com tech.ml
(ns ml-pipeline
  (:require [tech.v3.dataset         :as ds]
            [tech.v3.dataset.modelling :as ds-mod]
            [tech.v3.ml              :as ml]
            [tech.v3.ml.metrics      :as metrics]
            [tech.v3.ml.gridsearch   :as gs]))

; Carregar e preparar dataset Iris
(def iris
  (-> (ds/->dataset "iris.csv" {:key-fn keyword})
      (ds-mod/set-inference-target :species)))

; Train/test split 80/20
(def splits (ds-mod/train-test-split iris {:test-fraction 0.2}))
(def treino  (:train-ds splits))
(def teste   (:test-ds  splits))

; Treinar — Random Forest
(def modelo
  (ml/train treino {:model-type :smile.classification/random-forest
                     :n-trees    100
                     :max-depth  10}))

; Predizer
(def predicoes (ml/predict modelo teste))

; Métricas de avaliação
(metrics/accuracy           teste predicoes)
(metrics/classification-report teste predicoes)

Oz — visualizações Vega/Vega-Lite no REPL
(ns viz
  (:require [oz.core :as oz]))

(oz/start-server!)   ; abre browser com live reload

; Histograma de uma coluna
(def histograma
  {:data   {:values (map #({:valor %}) dados)}
   :mark   "bar"
   :encoding
   {:x {:bin  true :field "valor" :type "quantitative"}
    :y {:aggregate "count" :type "quantitative"}}})

(oz/view! histograma)

; Scatter plot — correlação entre duas variáveis
(def scatter

(oz/view! scatter)
