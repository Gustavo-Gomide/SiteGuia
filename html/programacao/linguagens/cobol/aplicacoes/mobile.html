COBOL: Mobile

Mobile com Core COBOL

Aplicativos Moveis apoiados em Mainframe

Milhoes de operacoes moveis sao confirmadas por sistemas COBOL no backend.

No mobile, COBOL nao roda no dispositivo: ele valida e liquida transacoes no servidor.

Fluxo de Operacao

App envia requisicao autenticada, API integra com servicos COBOL, resultado retorna ao usuario com trilha de auditoria.

Trecho de decisao de limite transacional
*> regra de autorizacao para operacao vinda do app mobile
  IF WS-VALOR-TRANSACAO <= WS-LIMITE-DIARIO
   PERFORM AUTORIZAR-TRANSACAO
   MOVE "APROVADA" TO WS-STATUS
ELSE
   MOVE "NEGADA" TO WS-STATUS
END-IF.

Banco digital

Seguradoras

Sinistro e consulta de apolice com regras historicas em COBOL.

Setor publico

Consulta cadastral e beneficios com compliance forte.

Topicos Avancados

Design para escala

  • Cache de leitura para reduzir carga no host.
  • Rate limit e antifraude em borda.
  • Versionamento de API para evolucao sem quebrar app instalado.

Nao acople o app mobile ao layout interno de copybooks; use contratos estaveis e documentados.
