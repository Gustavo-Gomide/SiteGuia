Perl — Automação

Automação com Perl

Manipulação de arquivos em massa, processamento de logs, scraping, envio de email, parsers de CSV e automação SSH.

Perl nasceu para automação: Larry Wall o criou para processar relatórios e manipular o sistema de arquivos Unix de forma que awk e shell não conseguiam. Décadas depois, ainda é uma das ferramentas mais produtivas para tarefas de sysadmin, processamento de dados e integração de sistemas.

Renomear Arquivos em Massa

A combinação de opendir/readdir com expressões regulares no operador s/// torna a renomeação em massa de arquivos trivial. Para recursividade, usa-se File::Find.

renomear_fotos.pl — adicionar prefixo de data
#!/usr/bin/env perl
use strict;
use warnings;
use autodie;
use POSIX qw(strftime);

my $diretorio = $ARGV[0] // ".";    # diretório como argumento ou atual

opendir my $dh, $diretorio;
my @arquivos = grep { /\.(jpg|jpeg|png|gif)$/i } readdir($dh);
closedir $dh;

say "Encontrados " . scalar(@arquivos) . " arquivos de imagem";

my $data_prefix = strftime("%Y%m%d_", localtime);
my $renomeados = 0;

foreach my $arquivo (sort @arquivos) {
    my $novo_nome = $data_prefix . lc($arquivo);
    $novo_nome =~ s/\s+/_/g;              # substituir espaços por _
    
    my $origem  = "$diretorio/$arquivo";
    my $destino = "$diretorio/$novo_nome";
    
    if ($origem ne $destino) {
        rename $origem, $destino;
        printf "%-40s  ==>  %s\n", $arquivo, $novo_nome;
        $renomeados++;
    }
}

say "Renomeados: $renomeados arquivos";
renomear_recursivo.pl — File::Find
use strict;
use warnings;
use File::Find;
use File::Basename;

# Converter todos os .txt em .md recursivamente
find({
    wanted => sub {
        return unless -f and /\.txt$/i;   # apenas arquivos .txt
        my $antigo = $_;
        (my $novo = $_) =~ s/\.txt$/.md/i;
        rename $antigo, $novo;
        say "$antigo → $novo";
    },
    no_chdir => 0,
}, ".");

Processar Logs com Regex

Processamento de logs é onde Perl reina. Combinando leitura de arquivo linha por linha com expressões regulares, é possível extrair métricas, detectar anomalias e gerar relatórios de logs de qualquer formato.

analisar_log.pl — log Apache/Nginx
#!/usr/bin/env perl
use strict;
use warnings;
use autodie;

# Formato Combined Log do Apache/Nginx:
# 192.168.1.1 - - [15/Mar/2024:10:30:00 +0000] "GET /index.html HTTP/1.1" 200 1234

my $log_regex = qr/
    ^(\d+\.\d+\.\d+\.\d+)          # IP do cliente
    \s+-\s+-\s+
    \[([^\]]+)\]                    # data/hora
    \s+"(\w+)\s+(\S+)\s+HTTP\/\S+"  # método e URL
    \s+(\d{3})                      # código HTTP
    \s+(\d+|-)                      # bytes enviados
/x;

my %estatisticas = (
    total      => 0,
    erros_4xx  => 0,
    erros_5xx  => 0,
    ips        => {},
    urls       => {},
    codigos    => {},
);

my $arquivo = $ARGV[0] // '/var/log/nginx/access.log';

open my $fh, '<', $arquivo;
while (my $linha = <$fh>) {
    chomp $linha;
    $estatisticas{total}++;
    
    if ($linha =~ $log_regex) {
        my ($ip, $data, $metodo, $url, $codigo, $bytes) =
            ($1, $2, $3, $4, $5, $6);
        
        $estatisticas{ips}{$ip}++;
        $estatisticas{urls}{$url}++;
        $estatisticas{codigos}{$codigo}++;
        $estatisticas{erros_4xx}++ if $codigo =~ /^4/;
        $estatisticas{erros_5xx}++ if $codigo =~ /^5/;
    }
}
close $fh;

# Relatório
printf "Total de requisições: %d\n", $estatisticas{total};
printf "Erros 4xx: %d (%.1f%%)\n",
    $estatisticas{erros_4xx},
    $estatisticas{erros_4xx} / $estatisticas{total} * 100;
printf "Erros 5xx: %d\n", $estatisticas{erros_5xx};

say "\nTop 5 URLs:";
my @top_urls = (sort { $estatisticas{urls}{$b} <=> $estatisticas{urls}{$a} }
                 keys %{$estatisticas{urls}})[0..4];
printf "  %-40s %d\n", $_, $estatisticas{urls}{$_} for @top_urls;

LWP e HTML::TreeBuilder: Web Scraping

LWP::UserAgent realiza requisições HTTP em Perl. Combinado com HTML::TreeBuilder (ou Mojo::DOM), permite extrair dados de páginas HTML de forma programática.

scraper.pl — extrair dados de uma página
#!/usr/bin/env perl
use strict;
use warnings;
use LWP::UserAgent;
use HTML::TreeBuilder;
use Encode qw(decode);

my $ua = LWP::UserAgent->new(
    agent   => 'MeuBot/1.0 (perl script educacional)',
    timeout => 15,
);

# Requisição GET
my $resposta = $ua->get('https://example.com');

unless ($resposta->is_success) {
    die "Erro HTTP " . $resposta->code . ": " . $resposta->message;
}

my $html = $resposta->decoded_content;

# Parsear HTML com TreeBuilder
my $tree = HTML::TreeBuilder->new_from_content($html);

# Extrair o título
my $title = $tree->look_down(_tag => 'title');
say "Título: " . ($title ? $title->as_text : "N/A");

# Extrair todos os links
my @links = $tree->look_down(_tag => 'a', href => qr/.+/);
foreach my $link (@links) {
    my $href = $link->attr('href');
    my $texto = $link->as_text;
    printf "%-50s  %s\n", $href, $texto;
}

# Extrair tabela específica (por ID)
my $tabela = $tree->look_down(_tag => 'table', id => 'dados-principais');
if ($tabela) {
    my @linhas = $tabela->look_down(_tag => 'tr');
    foreach my $tr (@linhas) {
        my @cels = map { $_->as_text } $tr->look_down(_tag => qr/^t[dh]$/);
        say join(" | ", @cels);
    }
}

$tree->delete;  # liberar memória

Envio de Email com MIME::Lite

MIME::Lite é uma das formas mais simples de enviar emails em Perl, suportando texto simples, HTML, anexos e endereçamento completo. Para casos mais complexos (autenticação SMTP, TLS), usa-se Email::Sender.

enviar_email.pl
use strict;
use warnings;
use MIME::Lite;

# Email texto simples
my $msg = MIME::Lite->new(
    From    => 'remetente@empresa.com',
    To      => 'destinatario@exemplo.com',
    Cc      => 'copia@exemplo.com',
    Subject => 'Relatório automático — Perl',
    Type    => 'text/plain; charset=UTF-8',
    Data    => "Olá,\n\nSeguem os dados do processamento.\n\nAtt.",
);

# Adicionar anexo
$msg->attach(
    Type     => 'text/csv',
    Path     => '/tmp/relatorio.csv',
    Filename => 'relatorio.csv',
    Disposition => 'attachment',
);

# Enviar via sendmail local
$msg->send('sendmail');

# Enviar via SMTP com autenticação
MIME::Lite->send('smtp', 'smtp.gmail.com',
    Port     => 587,
    AuthUser => 'usuario@gmail.com',
    AuthPass => 'senha_de_app',
);
$msg->send();

say "Email enviado com sucesso";

Text::CSV: Parsear e Gerar CSV

O módulo Text::CSV (ou sua versão XS em C: Text::CSV_XS) é o padrão para manipulação de arquivos CSV em Perl. Lida corretamente com campos com vírgulas, aspas e quebras de linha.

csv_processamento.pl
use strict;
use warnings;
use autodie;
use Text::CSV qw(csv);

# Leitura básica com função csv()
my $dados = csv(
    in      => 'vendas.csv',
    headers => 'auto',          # primeira linha = cabeçalho
    encoding => 'utf-8',
);

# $dados é um arrayref de hashrefs
foreach my $linha (@$dados) {
    printf "%-20s R$ %8.2f\n", $linha->{produto}, $linha->{valor};
}

# Escrever CSV
csv(
    in      => $dados,
    out     => 'saida.csv',
    headers => [qw(produto valor quantidade)],
    encoding => 'utf-8',
);

# API orientada a objeto — mais controle
my $csv = Text::CSV->new({
    binary      => 1,
    sep_char    => ';',   # usando ponto-e-vírgula
    quote_char  => '"',
    auto_diag   => 1,
});

open my $in, '<:encoding(utf-8)', 'entrada.csv';
open my $out, '>:encoding(utf-8)', 'saida_filtrada.csv';

my $cabecalho = $csv->getline($in);
$csv->say($out, $cabecalho);           # copiar cabeçalho

while (my $row = $csv->getline($in)) {
    next unless $row->[2] > 100;     # filtrar valor mínimo
    $csv->say($out, $row);
}

close $in;
close $out;

Automação SSH com Net::SSH2

Net::SSH2 permite executar comandos em servidores remotos via SSH diretamente de scripts Perl. Alternativas incluem Net::OpenSSH (wrapper do binário ssh) e Net::SSH::Perl.

ssh_automacao.pl
use strict;
use warnings;
use Net::SSH2;

my $ssh2 = Net::SSH2->new();

# Conectar
$ssh2->connect('servidor.exemplo.com', 22)
    or die "Falha ao conectar: " . $ssh2->error;

# Autenticar com chave pública
$ssh2->auth_publickey(
    'usuario',
    "$ENV{HOME}/.ssh/id_rsa.pub",
    "$ENV{HOME}/.ssh/id_rsa",
) or die "Autenticação falhou";

# Executar um comando
my $canal = $ssh2->channel();
$canal->exec('df -h / && uptime');

while (my $linha = <$canal>) {
    print $linha;
}
$canal->close();

# Transferir arquivo via SFTP
my $sftp = $ssh2->sftp();
$sftp->put('/local/relatorio.csv', '/remote/uploads/relatorio.csv')
    or die "SFTP falhou: " . $ssh2->error;

$ssh2->disconnect();
say "Operações SSH concluídas";

# Alternativa mais simples: Net::OpenSSH (usa o ssh do sistema)
use Net::OpenSSH;
my $ssh = Net::OpenSSH->new('usuario@servidor.com');
my @saida = $ssh->capture('ls /var/log/*.log | wc -l');
chomp @saida;
say "Arquivos de log: $saida[0]";
