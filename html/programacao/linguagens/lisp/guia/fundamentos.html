Lisp - Fundamentos

Base Sintática e Semântica de Lisp

Fundamentos em Lisp não são decorados por palavras-chave: eles são entendidos como transformação e avaliação de listas.

Ao dominar átomos, listas e avaliação, você passa a ler qualquer código Lisp como uma árvore de dados em execução.

Tipos e Átomos

Em Common Lisp, átomos incluem números, símbolos, strings e valores lógicos. A dupla t/nil controla verdade e vazio lógico.

Lisp/Common Lisp
; números
42
3.14

; símbolos
'x
'meu-simbolo

; strings
"texto"

; booleanos
t
nil

; operações em pares/listas
(car '(10 20 30))   ; => 10
(cdr '(10 20 30))   ; => (20 30)
(cons 5 '(10 20))   ; => (5 10 20)

Listas e Estruturas

S-expressions são o formato universal de código e dados em Lisp. Isso simplifica análise sintática e habilita macros poderosas.

S-expression na prática

Uma chamada como (+ 1 2) é uma lista cujo primeiro elemento é a função e os demais são argumentos.

Lisp/Common Lisp
(defparameter *lista* '(1 2 3))

(list 1 2 3)
(quote (a b c))
(append '(1 2) '(3 4))      ; => (1 2 3 4)
(length '(a b c d))         ; => 4
(mapcar #'1+ '(1 2 3 4))     ; => (2 3 4 5)

Funções

Funções em Lisp são valores de primeira classe: podem ser criadas com lambda, passadas como argumento e invocadas dinamicamente.

Lisp/Common Lisp
; função nomeada
(defun quadrado (x)
	(* x x))

; função anônima
((lambda (x) (+ x 10)) 5)

; escopo léxico com let e let*
(let ((a 2) (b 3))
	(+ a b))

(let* ((a 2)
			 (b (+ a 3)))
	(* a b))

; aplicar e chamar funções
(apply #'+ '(1 2 3 4))
(funcall #'quadrado 9)

Condições e Loops

Condição
  • if para bifurcação simples
  • cond para múltiplas guardas
  • when e unless para blocos legíveis
Repetição
  • loop para iteração declarativa ampla
  • do para controle fino de estado
  • dolist para percorrer listas
Lisp/Common Lisp
(if (> 8 3)
		"maior"
		"menor")

(cond
	((< 10 5) "caso 1")
	((= 10 10) "caso 2")
	(t "default"))

(when t
	(format t "Executa quando verdadeiro~%"))

(unless nil
	(format t "Executa quando falso~%"))

(loop for n from 1 to 3 do (print n))

(dolist (x '(a b c))
	(print x))
