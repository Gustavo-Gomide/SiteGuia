Lua para Web

Back-end performatico com Lua

Do edge ao servidor de aplicacao, Lua oferece caminhos simples e eficientes para web moderna.

OpenResty e Lapis mostram como Lua pode escalar APIs de baixa latencia com excelente produtividade.

OpenResty (Nginx + LuaJIT)

OpenResty integra Lua ao ciclo de requisicoes do Nginx. Isso permite construir gateways, autenticacao e caching diretamente no edge.

handler.lua
-- Handler basico em OpenResty
ngx.say("API ativa")
if ngx.var.request_method == "GET" then
	ngx.say("metodo aceito")
end

Lapis Framework

Lapis e um framework web em Lua com estilo MVC. Rotas, controladores e modelos podem ser escritos de forma enxuta e organizada.

Quando usar

Bom para APIs e apps server-side em stack LuaJIT/OpenResty, com deploy simples e rotas declarativas.

Sockets, JSON e Banco

Em servidores simples, bibliotecas de socket permitem ouvir conexoes TCP. Para serializacao, use dkjson ou cjson. Em dados relacionais, luadbi e uma opcao classica.
