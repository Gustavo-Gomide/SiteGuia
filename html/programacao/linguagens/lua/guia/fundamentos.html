Fundamentos da Linguagem

Dominando a Base de Lua

Compreenda os blocos essenciais que sustentam qualquer projeto serio com Lua.

Quando voce domina tabelas, escopo local e funcoes, ja consegue modelar a maior parte dos problemas cotidianos em Lua.

Tipos Primitivos e Especiais

Lua possui oito tipos principais: nil, boolean, number, string, function, table, thread e userdata.

Leitura rapida

table e o coracao da linguagem. thread representa corrotinas. userdata conecta objetos externos (normalmente C/C++).

Variaveis Globais vs Locais

Sem local, uma atribuicao cria/acessa global. Em projetos reais, prefira locais para evitar colisao de nomes e melhorar performance.

escopo.lua
-- Global (evite em modulos)
contador = 0

local function incrementar()
	contador = contador + 1
	local mensagem = "contador = " .. contador
	print(mensagem)
end

Tabelas: Estrutura Universal

Array

Indices numericos sequenciais: lista[1], lista[2].

Dicionario

Chaves arbitrarias: cfg["porta"] = 8080.

Objeto

Dados + metodos no mesmo table, simulando objetos.

tabelas.lua
local usuario = { nome = "Ana", pontos = 42 }
usuario.inativo = false
print(usuario.nome, usuario.pontos)

Funcoes, Closures e Varargs

Funcoes sao valores de primeira classe. Lua suporta multiplos retornos e argumentos variaveis com ....

funcoes.lua
local function soma(...)
	local total = 0
	for _, n in ipairs({...}) do
		total = total + n
	end
	return total, "ok"
end

Controle de Fluxo

Os comandos principais sao if/elseif/else, while, repeat/until, for numerico e for generico com pairs/ipairs.

EstruturaUso recomendado
for numericoSequencias com indice conhecido.
for com ipairsListas lineares sem buracos.
for com pairsTabelas chave-valor gerais.
repeat/untilExecutar pelo menos uma vez.
