Ecossistema Lua

Pacotes, compilador, editor, testes e depuracao para um fluxo de desenvolvimento profissional.

Com as ferramentas certas, Lua deixa de ser apenas linguagem de script e vira uma plataforma robusta para sistemas completos.

LuaRocks

LuaRocks e o gerenciador de pacotes padrao da comunidade Lua.

Comandos basicos
# buscar pacotes
luarocks search json

# instalar pacote
luarocks install dkjson

Testes e Debugging

Para testes automatizados, Busted e muito usado com estrutura describe, it e assert.

exemplo_spec.lua
describe("math utils", function()
	it("soma valores", function()
		assert.are.equal(4, 2 + 2)
	end)
end)

Debug pode iniciar com print debugging e evoluir para depuracao remota com MobDebug.
