Lua - Introdução

Lua

Uma linguagem pequena no tamanho, grande no impacto, especialmente em software embarcado e jogos.

Lua foi criada para ser simples de integrar, rapida de aprender e flexivel para resolver problemas reais com pouco codigo.

Historia e Contexto

Lua surgiu em 1993, na PUC-Rio, criada por Roberto Ierusalimschy, Waldemar Celes e Luiz Henrique de Figueiredo. O objetivo era oferecer uma linguagem de extensao para sistemas brasileiros de engenharia e dados.

Por que o nome Lua importa?

O nome reflete a origem brasileira do projeto e sua vocacao academica: uma linguagem feita com rigor teorico, mas com foco pratico em integracao com C e sistemas reais.

Caracteristicas Principais

Leve

Runtime pequeno e rapido para iniciar.

Embarcavel

Feita para ser embutida em engines, apps e firmwares.

Dinamica

Tipagem dinamica e tabelas versateis.

Multiparadigma

Procedural, funcional e orientada a prototipos.

Na pratica, isso significa produtividade alta para scripts e excelente custo-beneficio para automacao de regras de negocio.

Primeiro Script e REPL

hello.lua
-- Primeiro script em Lua
local nome = "Site Guia"
local versao = "5.4"
print("Ola, " .. nome .. "! Lua " .. versao)

Execute no terminal com lua hello.lua. Para testar instrucoes interativas, use o REPL: lua. Para checar bytecode e sintaxe, use luac -p hello.lua.

O comando luac -p nao executa o programa; ele apenas valida se o codigo esta correto sintaticamente.
