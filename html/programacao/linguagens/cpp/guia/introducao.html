C++ — Introducao

Introducao ao C++

C++ e uma linguagem de programacao de proposito geral com suporte a multiplos paradigmas: procedural, orientado a objetos, generica e funcional. Criada por Bjarne Stroustrup em 1979 como extensao do C, e a linguagem de escolha para sistemas de alto desempenho, jogos, sistemas operacionais e software embarcado.

C++ oferece controle manual sobre memoria, zero-cost abstractions e acesso direto ao hardware — sem abrir mao de abstraccoes de alto nivel. O lema da linguagem e: "What you don't use, you don't pay for."

Hello World e Compilacao

C++ — Hello World moderno (C++20+)
// hello.cpp
#include <print>      // C++23: std::println
#include <string>
#include <vector>

int main() {
  std::println("Ola, C++23!");

  // Inferencia de tipo com auto
  auto nome = std::string{ "Guia" };
  auto nums = std::vector<int>{ 1, 2, 3, 4, 5 };

  // Range-for (C++11)
  for (const auto& n : nums) {
    std::print("{} ", n);
  }
  std::println();   // quebra de linha
  return 0;
}
Shell — compilar com GCC e Clang
# GCC — compilar com C++23
g++ -std=c++23 -Wall -Wextra -O2 -o hello hello.cpp

# Clang
clang++ -std=c++23 -Wall -Wextra -O2 -o hello hello.cpp

# Debug com ASAN (Address Sanitizer) — detecta uso de memoria invalido
g++ -std=c++23 -g -fsanitize=address,undefined -o hello_debug hello.cpp

# Verificar flags de otimizacao disponiveis
g++ -O3 -march=native -o hello_fast hello.cpp

Paradigmas da Linguagem

Procedural

Heranca direta do C: funcoes, structs, ponteiros, arrays de tamanho fixo. Base para codigo de sistemas de baixo nivel, drivers e firmware.

Orientado a Objetos

Classes com encapsulamento, heranca simples/multipla, polimorfismo via virtual. RAII (Resource Acquisition Is Initialization) como idioma central de gerenciamento de recursos.

Genericidade

Templates — geracão de codigo em tempo de compilacao. Concept constraints (C++20) tornam erros de template legiveis. Toda STL e template-based.

Funcional

Lambdas com captura, std::function, algoritmos da STL (std::transform, std::reduce, std::views em C++20 Ranges). constexpr e consteval para computacao em tempo de compilacao.

C++ vs C vs Linguagens de Alto Nivel

Vantagens sobre C

  • Orientacao a objetos (classes, heranca, polimorfismo)
  • RAII para gerenciamento automatico de recursos
  • Templates e generic programming
  • Overload de operadores
  • Exceptions tipadas (try/catch)
  • STL: algoritmos, containers, iteradores
  • Smart pointers (unique_ptr, shared_ptr)
  • References (alternativa mais segura a ponteiros)

Desvantagens vs. linguagens GC

  • Gerenciamento manual de memoria (sem GC por padrao)
  • Compilacao mais lenta em projetos grandes
  • Curva de aprendizado inclinada (UB, dangling refs)
  • Erros de template pouco legíveis antes de C++20 Concepts
  • Sistema de build nao padronizado (CMake, Bazel, etc.)
  • Ausencia de package manager oficial (vcpkg/conan sao de terceiros)
