Visual Basic: Introducao

Visual Basic do Classico ao .NET

Uma familia de linguagens voltada a produtividade, com forte presenca em desktop corporativo e automacao Office.

Visual Basic evoluiu de VB6 para VB.NET, passou a rodar sobre o CLR e continua relevante em manutencao de sistemas, automacao e interfaces Windows.

Historia: de VB6 ao VB.NET

Visual Basic surgiu em 1991 como linguagem orientada a eventos para construir interfaces no Windows rapidamente. O salto grande veio com o VB.NET (2002), quando a linguagem passou a integrar a plataforma .NET e a compartilhar runtime com C# e F#.

  • 1991: Visual Basic 1.0 para Windows.
  • 1998: VB6 torna-se muito usado em software corporativo.
  • 2002: VB.NET chega com a primeira versao do .NET Framework.
  • 2017+: foco da Microsoft em estabilidade e interoperabilidade no ecossistema .NET.

.NET e CLR na pratica

No VB.NET, o codigo e compilado para IL (Intermediate Language) e executado pelo CLR (Common Language Runtime). Isso habilita gerenciamento de memoria, seguranca de tipos, excecoes estruturadas e interoperabilidade entre linguagens .NET.

visual basic - compilacao conceitual
' Codigo VB.NET
Module Program
	Sub Main()
		Console.WriteLine("Hello from VB.NET")
	End Sub
End Module

' VB Compiler -> IL -> CLR (JIT) -> Codigo nativo

Usos tipicos de Visual Basic hoje

Desktop Windows

WinForms e WPF em ambientes internos onde produtividade e manutencao previsivel sao prioridade.

Automacao Office

VBA para planilhas, relatorios e workflows de escritorio; VSTO para add-ins gerenciados.

Legado Corporativo

Migracao gradual de VB6 para .NET, mantendo regras de negocio que ja foram validadas por anos.

A Microsoft recomenda C# para novos cenarios amplos, mas Visual Basic segue suportado no .NET para cenarios de produtividade e manutencao.
