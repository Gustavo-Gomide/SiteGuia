Visual Basic: Fundamentos

Base Solida em VB.NET

Tipos, variaveis, fluxo de controle, metodos e orientacao a objetos no dia a dia.

O maior ganho em Visual Basic vem de escrever codigo simples, tipado e previsivel antes de pensar em recursos avancados.

Tipos e Variaveis

Em VB.NET, prefira Option Strict On para evitar conversoes implicitas perigosas. Tipos comuns incluem Integer, Double, String, Boolean e Date.

visual basic - declaracoes basicas
Option Strict On

Dim idade As Integer = 27
Dim salario As Double = 3500.5
Dim nome As String = "Ana"
Dim ativo As Boolean = True

Estruturas de Controle

Use If...ElseIf...Else, Select Case, For, While e For Each para deixar regras explicitas e legiveis.

visual basic - if, select e loop
If idade >= 18 Then
	Console.WriteLine("Maior de idade")
Else
	Console.WriteLine("Menor de idade")
End If

Select Case idade
	Case 0 To 12
		Console.WriteLine("Crianca")
	Case Else
		Console.WriteLine("Outro")
End Select

Functions e Subs

Sub executa acao sem retorno; Function retorna valor. Nomear bem esses blocos facilita testes e manutencao.

Sub

visual basic - sub
Sub ExibirMensagem()
	Console.WriteLine("Processo concluido.")
End Sub

Function

visual basic - function
Function Somar(a As Integer, b As Integer) As Integer
	Return a + b
End Function

Classes, Objetos e Encapsulamento

VB.NET oferece heranca, polimorfismo e encapsulamento completos. Comece com classes pequenas que representem conceitos reais do dominio.

visual basic - classe simples
Public Class Conta
	Public Property Titular As String
	Private Property Saldo As Decimal

	Public Sub Depositar(valor As Decimal)
		Saldo += valor
	End Sub
End Class

Evite classes gigantes com muitas responsabilidades. Separe regra de negocio, acesso a dados e interface.
