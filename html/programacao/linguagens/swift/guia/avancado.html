A etapa avancada de Swift conecta modelagem de tipos, concorrencia estruturada e interoperabilidade.

Generics e Protocol-Oriented Programming

Generics permitem reutilizacao sem perda de tipo. Protocolos com extensoes criam comportamento compartilhado de forma composicional.

Async/Await e Actors

Com structured concurrency, tarefas e cancelamento ficam mais previsiveis. Actors protegem estado mutavel contra data races.

swift - tarefa assincrona
func carregar() async throws -> [String] {
  let dados = try await api.fetch()
  return dados
}

ARC, Result Builders e Macros

ARC controla ciclo de vida de objetos. Result builders sustentam DSLs como SwiftUI. Macros (5.9+) reduzem boilerplate com geracao em compilacao.
