Swift - Introducao

Swift

Linguagem moderna da Apple para apps, backend e software de alto desempenho.

Swift combina seguranca de tipos, boa ergonomia e performance proxima de linguagens compiladas classicas.

Historia e Contexto

Swift foi anunciada em 2014 para substituir gradualmente Objective-C no ecossistema Apple. Em 2019, a estabilidade de ABI no iOS consolidou sua adocao em larga escala.

  • 2014: anuncio oficial na WWDC.
  • 2015: open source no GitHub.
  • 2019: ABI stable em plataformas Apple.
  • Atual: uso em iOS, macOS, servidor e tools CLI.

Filosofia da Linguagem

Os pilares sao seguranca, expressividade e velocidade. Optionals reduzem nulos inesperados, inferencia de tipos melhora produtividade e o compilador LLVM entrega performance forte.

swift - primeiro programa
import Foundation

let nome = "Swift"
print("Ola, \(nome)!")

Instalacao e Toolchain

Em macOS, o caminho padrao e Xcode. Em Linux/Windows, use Swift toolchain oficial e Swift Package Manager (SPM) para dependencias e build.

terminal - criando projeto
swift package init --type executable
swift build
swift run
