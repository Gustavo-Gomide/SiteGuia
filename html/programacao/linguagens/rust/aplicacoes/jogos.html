Rust em Jogos

ECS moderno, performance nativa e pipeline multiplataforma.

Com Bevy e ecossistema de plugins, Rust permite construir jogos 2D/3D com arquitetura limpa, boa escalabilidade e deploy para web via WASM.

Bevy ECS: Entities, Components, Systems e Schedules

Bevy adota ECS para separar dados (components) de comportamento (systems), facilitando paralelizacao e manutencao.

bevy minimo
// loop principal com sistemas de startup e update
			use bevy::prelude::*;

fn main() {
		App::new()
				.add_plugins(DefaultPlugins)
				.add_systems(Startup, setup)
				.add_systems(Update, movimentar)
				.run();
}

Fisica, Assets, Audio e Game Loop

Rapier oferece fisica robusta para colisao e dinamica. Bevy possui sistema de assets para sprites, animacoes e audio. O loop principal e controlado por schedules e estados.

  • Fisica: bevy_rapier2d / bevy_rapier3d.
  • Assets: carregamento assincro e hot-reload em desenvolvimento.
  • Audio: eventos de som por sistema e controle de canais.

WASM e Publicacao no itch.io

Compilar para WASM permite rodar no browser sem instalacao. Para distribuicao indie, itch.io e uma rota simples para web builds e executaveis desktop.

target web
rustup target add wasm32-unknown-unknown
cargo build --release --target wasm32-unknown-unknown
