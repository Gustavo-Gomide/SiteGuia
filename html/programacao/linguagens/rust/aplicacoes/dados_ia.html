Linfa, ndarray e Burn

ndarray e a base de computacao numerica; linfa cobre algoritmos de machine learning classico; Burn oferece framework moderno de deep learning em Rust.

ndarray basico
use ndarray::array;

fn main() {
		let x = array![[1.0, 2.0], [3.0, 4.0]];
		let media = x.mean().unwrap();
		println!("{}", media);
}

ONNX Runtime, PyO3 e WASM

Com bindings de ONNX Runtime, voce executa modelos treinados em outras stacks. PyO3 permite expor funcoes Rust para Python. E com WASM, inferencia leve pode ir para o browser.

  • ONNX para interoperabilidade entre frameworks de treino.
  • PyO3 para substituir gargalos Python por kernels Rust.
  • WASM para pre-processamento e inferencia client-side com baixa latencia.

Arquitetura Recomendada

Padrao recorrente em producao: treino em Python, serializacao em ONNX e servico de inferencia em Rust com API HTTP. Essa separacao equilibra produtividade cientifica e eficiencia operacional.
