Rust para Terminal e CLI

Ferramentas robustas, distribuidas como binarios unicos e rapidos.

Rust e excelente para CLIs por unir desempenho, portabilidade e um ecossistema maduro para UX de terminal.

Clap, Validacao e Subcomandos

clap derive api
// parser declarativo de argumentos e subcomandos
			use clap::{Parser, Subcommand};

#[derive(Parser)]
struct Cli {
		#[command(subcommand)]
		cmd: Cmd,
}

#[derive(Subcommand)]
enum Cmd {
		Init { path: String },
		Build { #[arg(long)] release: bool },
}

Com derive, ajuda, parsing e validacao ficam consistentes sem muito boilerplate.

Experiencia de Terminal Moderna

indicatif cria barras de progresso e spinners. crossterm controla raw mode e cores. dialoguer facilita prompts interativos. ratatui habilita TUIs completas.

  • CLI simples: clap + anyhow + indicatif.
  • Wizard interativo: adicionar dialoguer.
  • Painel em tempo real: migrar para ratatui.

Distribuicao com cargo-dist e Shell Completions

cargo-dist automatiza artefatos e installers para multiplas plataformas. Gere completions para bash, zsh, fish e powershell para elevar a produtividade do usuario final.

pipeline de release
cargo install cargo-dist
cargo dist init
cargo dist build
