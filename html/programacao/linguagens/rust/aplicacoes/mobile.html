Rust no Mobile

Core de negocio em Rust, interface nativa ou multiplataforma no topo.

No mobile, Rust costuma viver no core: criptografia, sync engine, regras de negocio e processamento pesado, expostos para iOS/Android via bindings.

UniFFI, C FFI e Interoperabilidade

UniFFI gera bindings para Swift/Kotlin a partir de interfaces declaradas. Quando necessario, C FFI puro continua sendo rota universal para integrar com apps nativos.

  • iOS: integracao via Swift Package Manager ou XCFramework.
  • Android: bibliotecas nativas com NDK e JNI.
  • Flutter/React Native: camada de ponte para chamar funcoes Rust.
