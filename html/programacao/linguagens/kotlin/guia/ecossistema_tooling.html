Kotlin — Ecossistema e Tooling

Ecossistema e Tooling Kotlin

Kotlin tem um dos ecossistemas mais produtivos da indústria por estar sobre a JVM e, ao mesmo tempo, oferecer ferramentas modernas para mobile, backend e multiplataforma.

Build com Gradle Kotlin DSL

build.gradle.kts essencial
plugins {
		kotlin("jvm") version "2.1.0"
		application
}

repositories {
		mavenCentral()
}

dependencies {
		testImplementation(kotlin("test"))
		implementation("io.ktor:ktor-server-core:2.3.12")
}

application {
		mainClass.set("com.siteguia.MainKt")
}

tasks.test {
		useJUnitPlatform()
}

Testes e Qualidade

JUnit 5 e MockK
import io.mockk.every
import io.mockk.mockk
import kotlin.test.Test
import kotlin.test.assertEquals

class Calculadora(private val fator: Int) {
		fun dobro(x: Int) = x * fator
}

class CalculadoraTest {
		@Test
		fun `deve calcular dobro`() {
				val c = Calculadora(2)
				assertEquals(10, c.dobro(5))
		}
}

Workflow CI simplificado
name: Kotlin CI

on: [push, pull_request]

jobs:
	build:
		runs-on: ubuntu-latest
		steps:
			- uses: actions/checkout@v4
			- uses: actions/setup-java@v4
				with:
					distribution: temurin
					java-version: 21
			- run: ./gradlew test detekt
