Kotlin — Introdução

Kotlin — Introdução Completa

Kotlin é uma linguagem moderna da JetBrains, tipada estaticamente, interoperável com Java e focada em produtividade. Hoje é linguagem principal para Android, muito forte no backend JVM e também presente em multiplataforma.

Kotlin combina segurança de tipos (especialmente null safety), sintaxe concisa e forte ecossistema. O resultado é menos código boilerplate, menos erros em runtime e maior velocidade de desenvolvimento.

Características-Chave

Null Safety

Tipos anuláveis explícitos evitam NullPointerException acidental.

Interoperabilidade Java

Pode chamar bibliotecas Java existentes sem wrappers complexos.

Coroutines

Concorrência assíncrona eficiente e legível para UI e backend.

Data Classes

Modelos de dados com equals/hashCode/toString automáticos.

Extensions

Permite estender APIs sem herança ou modificação da classe original.

Multiplatform

Compartilha código entre Android, iOS, desktop e backend.

Instalação e Primeiro Programa

Kotlin CLI
# SDKMAN (Linux/macOS)
sdk install kotlin

# Verificar
kotlinc -version

# Compilar e executar
kotlinc Hello.kt -include-runtime -d hello.jar
java -jar hello.jar
Hello.kt
fun main() {
		println("Ola, Kotlin!")

		val nome = "Site Guia"
		val versao = 2
		println("Projeto: $nome v$versao")

		fun dobro(x: Int) = x * 2
		println("Dobro de 21 = ${dobro(21)}")
}

Quando Kotlin é uma Boa Escolha

Use Kotlin quando:

  • Você desenvolve Android nativo moderno
  • Seu backend roda em JVM e quer mais produtividade que Java
  • Você precisa compartilhar código entre plataformas
  • Seu time valoriza segurança de tipos com sintaxe limpa

Pense duas vezes quando:

  • O ecossistema principal do projeto não é JVM
  • O time depende de libs não compatíveis com Kotlin/Native
  • Seu foco é ultra baixo nível (firmware bare-metal)
