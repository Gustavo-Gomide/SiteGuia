Kotlin — Desenvolvimento de Jogos

libGDX é o framework de jogos 2D/3D mais popular na JVM. Com KTX (LibKTX), recebe extensões idiomáticas Kotlin: DSLs para Scene2D, asset manager com coroutines e syntax sugar que elimina muito do verbosidade Java.

Kotlin traz null safety e extension functions ao desenvolvimento de jogos cross-platform. libGDX + KTX compila para Desktop (Windows/Linux/macOS), Android, iOS (via RoboVM) e HTML5 (GWT).

libGDX com Kotlin

A estrutura central de libGDX é a interface ApplicationListener. Em Kotlin, usamos KtxGame do KTX para adicionar gerenciamento de telas com type safety.

Kotlin — libGDX ApplicationAdapter
import com.badlogic.gdx.ApplicationAdapter
import com.badlogic.gdx.Gdx
import com.badlogic.gdx.graphics.GL20
import com.badlogic.gdx.graphics.Texture
import com.badlogic.gdx.graphics.g2d.SpriteBatch

class MeuJogo : ApplicationAdapter() {
    private lateinit var batch: SpriteBatch
    private lateinit var textura: Texture

    override fun create() {
        batch = SpriteBatch()
        textura = Texture("player.png")
    }

    override fun render() {
        val dt = Gdx.graphics.deltaTime  // tempo desde último frame
        Gdx.gl.glClearColor(0.1f, 0.1f, 0.15f, 1f)
        Gdx.gl.glClear(GL20.GL_COLOR_BUFFER_BIT)

        batch.begin()
        batch.draw(textura, 100f, 100f)
        batch.end()
    }

    override fun dispose() {
        batch.dispose()
        textura.dispose()
    }
}

KTX — Extensões Idiomáticas

LibKTX oferece módulos para Scene2D com DSL (scene2d {}), asset manager com coroutines, e extensões para Math, Input, AI e mais — sempre preservando a compatibilidade com a API padrão do libGDX.

Kotlin — KTX Scene2D DSL
import ktx.scene2d.*
import ktx.actors.*
import com.badlogic.gdx.scenes.scene2d.Stage

val stage = Stage()

// DSL para construir UI do jogo (HUD, menus)
val tabela = scene2d.table {
    setFillParent(true)
    pad(20f)
    label("Vida: 100") { cell().expandX() }
    row()
    textButton("Jogar") {
        cell().width(200f).height(60f)
        onClickEvent { iniciarJogo() }
    }
    row()
    textButton("Sair") {
        cell().width(200f).height(60f)
        onClickEvent { Gdx.app.exit() }
    }
}
stage.addActor(tabela)

Game Loop com Delta Time

O game loop em libGDX usa deltaTime para movimento frame-rate independent. Separar lógica (update) de renderização (render) é boa prática para facilitar testes e futura integração com física.

Kotlin — Game loop com entidades
data class Vec2(var x: Float, var y: Float)

class Player(
    var pos: Vec2 = Vec2(100f, 100f),
    var vel: Vec2 = Vec2(0f, 0f),
    val speed: Float = 200f
) {
    fun update(dt: Float) {
        pos.x += vel.x * speed * dt
        pos.y += vel.y * speed * dt
        // clamp nas bordas da tela
        pos.x = pos.x.coerceIn(0f, Gdx.graphics.width.toFloat())
        pos.y = pos.y.coerceIn(0f, Gdx.graphics.height.toFloat())
    }
}

class GameScreen : KtxScreen {
    val player = Player()

    override fun render(delta: Float) {
        // Input
        player.vel.x = when {
            Gdx.input.isKeyPressed(Input.Keys.RIGHT) -> 1f
            Gdx.input.isKeyPressed(Input.Keys.LEFT) -> -1f
            else -> 0f
        }
        // Update
        player.update(delta)
        // Render
        // ... batch.draw(playerTexture, player.pos.x, player.pos.y)
    }
}

Colisão e Física com Box2D

libGDX inclui Box2D para simulação física 2D. Bodies com fixtures definem forma e propriedades físicas. Um ContactListener detecta colisões typed-safe em Kotlin.

Kotlin — Box2D physics
import com.badlogic.gdx.physics.box2d.*
import ktx.box2d.*

val SCALE = 64f  // pixels por metro
val mundo = world(gravity = vec2(0f, -9.8f))

// Chão estático — KTX DSL
val chao = mundo.body {
    type = BodyDef.BodyType.StaticBody
    position.set(0f, 0f)
    box(width = 20f, height = 1f)
}

// Jogador dinâmico
val corpo = mundo.body {
    type = BodyDef.BodyType.DynamicBody
    position.set(5f, 5f)
    box(width = 1f, height = 2f) {
        density = 1f
        friction = 0.5f
        restitution = 0.3f
    }
}
