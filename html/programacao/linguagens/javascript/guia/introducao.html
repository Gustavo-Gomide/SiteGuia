JavaScript — Introdução

JavaScript

A linguagem da web: dinâmica, interpretada, orientada a eventos e onipresente — do browser ao servidor, de apps móveis a microcontroladores.

JavaScript foi criado em 10 dias em 1995 por Brendan Eich na Netscape e hoje é a linguagem mais usada no mundo segundo a pesquisa anual do Stack Overflow por doze anos consecutivos. Roda em navegadores, no Node.js, no Deno, no Bun e em chips IoT via Espruino.

Tipagem Dinâmica e o Sistema de Tipos

JavaScript — Tipos primitivos, coerção e typeof
// 7 tipos primitivos em JavaScript
const n = 42;              // number (IEEE 754 double)
const bigN = 2n ** 64n;   // bigint — inteiros arbitrários
const s = "olá";           // string
const b = true;            // boolean
const u = undefined;      // variável declarada sem valor
const nil = null;          // ausência intencional de objeto
const sym = Symbol("id"); // símbolo único e imutável

// typeof revela o tipo em run-time
console.log(typeof n);       // "number"
console.log(typeof null);   // "object" — bug histórico do JS!
console.log(typeof Symbol()); // "symbol"

// Coerção implícita: armadilhas clássicas
console.log("5" - 2);    // 3  (coerce string → number)
console.log("5" + 2);    // "52" (+ prefere string)
console.log(null == undefined);  // true  — loose equality
console.log(null === undefined); // false — strict equality
console.log(0 == false);         // true  — sempre use ===

Ambientes de Execução

Browser

Motor V8 (Chrome/Edge), SpiderMonkey (Firefox), JavaScriptCore (Safari). Acesso ao DOM, Web APIs, Service Workers.

Node.js

V8 + libuv. Runtime de servidor com event loop I/O assíncrono, npm, streams nativas, cluster. LTS: v22.

Deno

Criado por Ryan Dahl. TypeScript nativo, segurança por permissão, sem node_modules — usa URLs para importar.

Bun

JavaScriptCore + Zig. Objetivo: substituição all-in-one de Node.js mais rápida, com bundler e test runner embutidos.

Primeiros Programas

JavaScript (Browser) — Manipular DOM + evento
<!-- index.html -->
<button id="btn">Clique</button>
<p id="saida"></p>
<script type="module">
  const btn   = document.getElementById("btn");
  const saida = document.getElementById("saida");
  let contagem = 0;

  btn.addEventListener("click", () => {
    contagem++;
    saida.textContent = `Clicado ${contagem} vez(es)`;
  });
</script>
JavaScript (Node.js) — HTTP server nativo
// server.js — node server.js
import { createServer } from "node:http";

const server = createServer((req, res) => {
  res.writeHead(200, { "Content-Type": "application/json" });
  res.end(JSON.stringify({
    rota: req.url,
    metodo: req.method,
    horario: new Date().toISOString(),
  }));
});

server.listen(3000, () => console.log("http://localhost:3000"));
