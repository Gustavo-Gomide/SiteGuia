ncurses — A Arte das Interfaces de Terminal

Quando nao ha janelas graficas, ncurses transforma qualquer terminal em uma interface rica.

ncurses (new curses) e uma implementacao open-source da biblioteca curses original do BSD, mantida por Thomas Dickey desde 1994. E a biblioteca que sustenta ferramentas cli como htop, vim, mc (Midnight Commander) e muitos instaladores de sistemas operacionais. Trabalha enviando sequencias de escape para o terminal, posicionando o cursor e desenhando areas de texto com cores e atributos — tudo isso de forma portavel entre centenas de tipos de terminais.

1. Conceitos Fundamentais

WINDOW

Regiao retangular do terminal com seu proprio buffer. A janela especial stdscr representa a tela toda. Crie janelas com newwin(linhas, cols, y, x).

refresh() e wrefresh()

ncurses usa duplo buffer: operacoes escrevem no buffer interno. Apenas ao chamar refresh() o terminal fisico e atualizado com o minimo de sequencias de escape necessarias — evita flicker.

getch() e Teclas Especiais

getch() le uma tecla. Com keypad(stdscr, TRUE) habilitado, teclas especiais (setas, F1-F12, Home, End) retornam constantes como KEY_UP, KEY_F(1).

Pares de Cor

Defina pares foreground/background com init_pair(id, cor_texto, cor_fundo). Aplique com COLOR_PAIR(id) nos atributos. ncurses suporta ate 256 cores em terminais modernos.

2. Estrutura Basica

c — programa ncurses basico com cores
#include <ncurses.h>

int main(void)
{
    initscr();              /* inicializa ncurses */
    cbreak();               /* nao espera Enter para getch() */
    noecho();               /* nao exibe teclas digitadas */
    keypad(stdscr, TRUE);   /* habilita teclas especiais */
    curs_set(0);             /* esconde cursor */

    if (has_colors()) {
        start_color();
        init_pair(1, COLOR_CYAN, COLOR_BLACK);
        init_pair(2, COLOR_BLACK, COLOR_WHITE);
    }

    attron(COLOR_PAIR(1) | A_BOLD);
    mvprintw(5, 10, "Ola, ncurses!");
    attroff(COLOR_PAIR(1) | A_BOLD);

    attron(COLOR_PAIR(2));
    mvprintw(7, 10, " Pressione qualquer tecla para sair ");
    attroff(COLOR_PAIR(2));

    refresh();
    getch();
    endwin();   /* restaura o terminal ao estado normal */
    return 0;
}
shell — compilar com ncurses
gcc -o programa main.c -lncurses

3. Multiplas Janelas e Boxes

c — criar janela com borda
/* cria janela 10x40 na posicao (3, 5) */
WINDOW *win = newwin(10, 40, 3, 5);

/* desenha borda padrao com +, -, | */
box(win, 0, 0);

/* escreve dentro da janela (relativo a ela) */
mvwprintw(win, 1, 2, "Janela auxiliar");
mvwprintw(win, 3, 2, "Linha 2...");

wrefresh(win);  /* atualiza apenas esta janela */

/* ao final, destrua a janela */
delwin(win);

4. Paineis e Formularios

ncurses inclui extensoes para casos de uso mais ricos:

panel.h

Empilhamento de janelas com ocultamento automatico: janelas sobrepoe umas as outras corretamente. Use com -lpanel ao compilar. Operacoes: new_panel(), show_panel(), hide_panel(), update_panels().

form.h

Formularios TUI interativos: campos FIELD com validacao, navegacao com Tab/Enter, campos sensiveis a typos como inteiros, alfa, email. Linkado com -lform.

menu.h

Menus verticais e horizontais navegaveis com teclado. ITEM list, MENU struct e callbacks de selecao. Linkado com -lmenu. A base de muitos CLIs interativos como o instalador do Debian.

5. Quando Usar ncurses

Ideal Para

  • Ferramentas CLI interativas: dashboards, monitores, file managers (htop, mc, vim)
  • Ambientes sem display grafico: servidores, containers Docker, SSH
  • Instaladores e configuradores de sistema que rodam antes do ambiente grafico
  • Jogos roguelike e puzzles baseados em ASCII/terminal
  • Ferramentas de desenvolvimento (tmux, cmus, ranger, mutt) em C puro

Evitar Quando

  • O usuario espera uma interface grafica convencional com mouse
  • Precisa de imagens, graficos ou elementos visuais ricos
  • App sera distribuido para usuarios nao tecnicos
  • Windows: ncurses e POSIX; no Windows use PDCurses ou migrate para outra lib
