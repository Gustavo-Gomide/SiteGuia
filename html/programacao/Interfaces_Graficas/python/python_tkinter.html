Python Tkinter

Tkinter — A GUI que Vem com o Python

A biblioteca GUI padrao da linguagem: zero instalacao, zero dependencias externas.

Tkinter esta disponivel em toda instalacao padrao do Python desde a versao 1.5 (1999). O IDLE — o editor que acompanha o Python — e escrito em Tkinter. Ferramentas internas, scripts de automacao com GUI e prototipagem rapida sao os dominios naturais de Tkinter: quando voce precisa de uma janela e nao quer instalar mais nada.

1. O Que E Tkinter

Tkinter e o wrapper Python para o toolkit Tk, escrito em Tcl/Tk por John Ousterhout em 1991. O Tk renderiza widgets usando as primitivas graficas do sistema operacional — mas nao usa widgets nativos: desenha os proprios com aparencia aproximada do SO.

Tk vs ttk: o modulo tkinter.ttk (Themed Tk) introducido no Python 3.x fornece widgets com aparencia mais moderna e suporte a temas. Sempre prefira ttk.Button a tk.Button em projetos novos — visual mais limpo e suporte a estilos customizados.

Onde Vive

import tkinter — built-in. Nenhum pip install necessario. O interpretador CPython ja inclui a extensao C que faz bridge com a biblioteca Tk compartilhada do sistema.

Arquitetura

Python ⇔ Tcl interpreter (embutido) ⇔ Tk C library ⇔ sistema operacional. Toda chamada de widget atravessa essa cadeia. Tk mantem os widgets em memória; Python controla via handles.

Widgets Principais

Label, Button, Entry, Text, Frame, Canvas, Listbox, Combobox (ttk), Treeview (ttk — tabelas/arvores).

2. Hello World e Estrutura Basica

python — janela minima com Tkinter
import tkinter as tk
from tkinter import ttk

class App(tk.Tk):
    def __init__(self):
        super().__init__()
        self.title("Minha Janela")
        self.geometry("400x300")
        self.resizable(True, True)

        # StringVar: variavel observavel vinculada a widget
        self.nome_var = tk.StringVar()

        ttk.Label(self, text="Nome:").pack(pady=10)
        ttk.Entry(self, textvariable=self.nome_var, width=30).pack()
        ttk.Button(self, text="Saudar", command=self.saudar).pack(pady=10)
        self.result_label = ttk.Label(self, text="")
        self.result_label.pack()

    def saudar(self):
        nome = self.nome_var.get()
        self.result_label.config(text=f"Ola, {nome}!" if nome else "Digite um nome.")

if __name__ == "__main__":
    app = App()
    app.mainloop()  # inicia o event loop

3. Geometry Managers

Tkinter tem tres gerenciadores de layout que determinam a posicao dos widgets. Nunca misture pack e grid no mesmo Frame — causa conflito e o programa pode travar.

pack — Fluxo Linear

python — pack
# Empilha widgets em sequencia
ttk.Label(frame, text="A").pack(side="top",    fill="x")
ttk.Label(frame, text="B").pack(side="left",   padx=5)
ttk.Label(frame, text="C").pack(side="right")
# fill="x": expande horizontalmente
# expand=True: usa espaco extra disponivel

grid — Tabela de Celulas

python — grid
# Layout tabular com row e column
ttk.Label(frame, text="Nome").grid(row=0, column=0, sticky="w")
ttk.Entry(frame).grid(row=0, column=1, columnspan=2)
ttk.Label(frame, text="Idade").grid(row=1, column=0)
ttk.Spinbox(frame, from_=0, to=120).grid(row=1, column=1)
# sticky="nsew": ancora nas 4 direcoes

place — Posicao Absoluta

python — place
# Posicao fixa (pixels) ou relativa
btn.place(x=50, y=100)
btn.place(relx=0.5, rely=0.5, anchor="center")
# relx/rely: fracoes de 0.0 a 1.0 do container
# Evite place para layouts responsivos!

4. Eventos e Bindings

python — event binding e variaveis observaveis
# Bind: associar evento a callback
entry.bind("<Return>", lambda event: buscar())
canvas.bind("<Button-1>", on_click)       # clique esquerdo
canvas.bind("<Motion>", on_mouse_move)     # mover mouse
root.bind("<Control-s>", lambda e: salvar())  # atalho teclado

# StringVar: mudanca de valor dispara trace
var = tk.StringVar()
var.trace_add("write", lambda *_: atualizar_preview(var.get()))

# Rodar tarefa longa sem bloquear UI
import threading

def tarefa_demorada():
    resultado = calcular_algo()             # roda em background
    root.after(0, lambda: label.config(text=resultado))  # volta para UI thread

threading.Thread(target=tarefa_demorada, daemon=True).start()

5. Quando Usar Tkinter

Ideal Para

  • Ferramentas internas e scripts com GUI simples
  • Prototipagem rapida de interfaces antes de portar para Qt/PyQt
  • Ambientes sem acesso a pip (air-gapped, educacao basica)
  • Aprender os conceitos fundamentais de GUI (event loop, widgets, layout)
  • IDLE, configuradores, launchers pequenos

Evitar Quando

  • O app precisa de visual moderno e polish profissional — use PyQt/PySide
  • Precisa de suporte mobile — use Kivy ou Flutter
  • Precisa de graficos 2D/3D complexos — use Kivy ou PyGame
  • App grande com muitas telas: sem sistema de rotas nativo, codigo cresce confuso
  • Precisa de binding de dados bidirecional sofisticado — PyQt tem Model/View

6. Topicos Avancados

Temas com ttk.Style

ttk.Style() controla cores, fontes e padding dos widgets ttk. Bibliotecas como ttkbootstrap e sv-ttk (Sun Valley — tema Windows 11) transformam a aparencia padrao em interfaces modernas sem muito codigo.

Canvas — Desenho Vetorial

O widget Canvas e o corac do Tkinter para graficos: linhas, retangulos, arcos, imagens e texto com coordenadas. Usado para editores de diagrama, visualizacoes de dados simples e jogos 2D basicos.

Telas Multiplas com Frame Stacking

Sem sistema de rotas nativo, o padrao padrao e empilhar Frames e usar tkraise() para mostrar a tela ativa. O container ttk.Notebook fornece abas prontas.

Matplotlib e Pillow no Canvas

FigureCanvasTkAgg do Matplotlib embute graficos interativos em janelas Tkinter. PIL.ImageTk.PhotoImage exibe imagens de qualquer formato suportado pelo Pillow — expandindo muito o Tkinter alem do Tk nativo.

Thread safety: nunca modifique widgets de uma thread que nao seja a thread principal. Use root.after(0, callback) para agendar atualizacoes na UI thread a partir de background threads.

7. Ecossistema

ttkbootstrap

Temas modernos inspirados no Bootstrap para ttk. Fornece Light/Dark, cores de status (success, warning, danger) e componentes extras como DateEntry e Meter circular.

sv-ttk (Sun Valley)

Tema que imita o visual do Windows 11 com suporte a Light/Dark mode. Uma linha de codigo transforma completamente a aparencia padrao.

customtkinter

Biblioteca de widgets modernos para Tkinter com cantos arredondados, cores personalizadas e dark mode nativo. Popular para apps desktop Python de aspecto contemporaneo.

tkcalendar

Widget de calendario e DateEntry para selecao de datas. Integra-se com ttk e ttkbootstrap. Indispensavel em formularios com selecao de datas.

Pillow (PIL)

Biblioteca de imagens que expande o suporte a formatos do Tkinter (JPEG, PNG, GIF, BMP). Necessaria para exibir qualquer imagem alem de GIF e PNG basico via PhotoImage.

PyInstaller / cx_Freeze

Empacota o app Tkinter em executavel standalone (.exe no Windows, .app no macOS). Inclui o Python runtime e todas as dependencias — distribuivel sem instalar Python.
