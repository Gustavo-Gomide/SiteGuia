PHP-GTK — Desktop Nativo com PHP e GTK

Uma curiosidade historica e nicho: PHP saindo do servidor e entrando em janelas GTK nativas.

PHP-GTK e uma extensao PHP que fornece bindings para a toolkit GTK, permitindo criar aplicacoes desktop nativas em PHP. Foi criada por Andrei Zmievski e o projeto PHP-GTK 2 (baseado no GTK 2) teve desenvolvimento ativo no inicio dos anos 2000. A extensao expoe classes PHP para GtkWindow, GtkButton, GtkVBox e demais widgets GTK. Hoje e considerada legado: o desenvolvimento parou, ha pouca documentacao atualizada e o suporte a PHP moderno e limitado. O uso e principalmente historico e academico.

1. App PHP-GTK Basico

php — janela gtk com botao
<?php
// Requer a extensao php-gtk2 instalada
if (!class_exists('gtk')) {
    die("PHP-GTK nao esta instalado.\n");
}

$window = new GtkWindow();
$window->set_title('Ola, PHP-GTK!');
$window->set_default_size(300, 120);

$btn = new GtkButton('Clique aqui');
$btn->connect('clicked', function() {
    echo "Clicado!\n";
});

$window->add($btn);
$window->connect_simple('destroy', ['gtk', 'main_quit']);
$window->show_all();
Gtk::main();  // inicia o loop de eventos GTK

2. Status e Perspectivas

PHP-GTK 2 (GTK 2)

Versao mais usada historicamente. Baseada em GTK 2.x (deprecated). Desenvolvimento oficial parou por volta de 2012. Ainda funciona em distros antigas.

PHP-GTK 3

Tentativas esporadicas de port para GTK3, mas sem release oficial estavel. Nenhuma versao moderna amplamente testada existe.

Alternativas Modernas

Para apps desktop em PHP: Livewire (web), Tauri + PHP API (desktop via webview), ou simplesmente mudar de linguagem para Python/Go/Rust.

3. Quando Usar PHP-GTK

Cenarios Possiveis

  • Manutencao de sistema legado que ja usa PHP-GTK e nao pode ser migrado
  • Estudo historico de abordagens de GUI em linguagens de script

Evitar Quando

  • Qualquer projeto novo — nao ha suporte PHP 8+, sem comunidade ativa, sem atualizacoes de seguranca
  • Precisa de app desktop estavel: use Python/GTK3, Go/Fyne ou Rust/GTK-rs
  • App web reativo: use Livewire para PHP no browser
