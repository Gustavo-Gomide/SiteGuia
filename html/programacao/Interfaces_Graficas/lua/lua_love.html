LOVE2D — Framework de Jogos 2D em Lua

Simples de comecar, poderoso para jogos completos. O framework favorito da cena indie Lua.

LOVE2D (ou simply "love") e um framework open-source para criacao de jogos 2D em Lua, com bindings para OpenGL, OpenAL e backend de fisica (Box2D). A API gira em torno de tres callbacks centrais: love.load(), love.update(dt) e love.draw(). O framework fornece love.graphics para rendering, love.audio para som, love.physics para simulacao e love.filesystem para I/O. Jogos publicados em itch.io e outros portais indie frequentemente usam LOVE2D pela legeireza e pela curva de aprendizado suave de Lua.

1. Os Callbacks Fundamentais

lua — jogo basico com movimento de personagem
-- main.lua — arquivo principal do jogo LOVE2D
local jogador = { x=200, y=200, vel=150 }
local imgJogador

function love.load()
    imgJogador = love.graphics.newImage("jogador.png")
    love.window.setTitle("Meu Jogo")
    love.window.setMode(800, 600)
end

function love.update(dt)
    -- dt = delta time em segundos desde o ultimo frame
    if love.keyboard.isDown("right") then
        jogador.x = jogador.x + jogador.vel * dt
    end
    if love.keyboard.isDown("left") then
        jogador.x = jogador.x - jogador.vel * dt
    end
end

function love.draw()
    love.graphics.draw(imgJogador, jogador.x, jogador.y)
    love.graphics.print("X: " .. math.floor(jogador.x), 10, 10)
end

function love.keypressed(key)
    if key == "escape" then love.event.quit() end
end

2. Modulos Principais

love.graphics

Rendering 2D: draw, print, newImage, newFont, setColor, circle, rectangle, line. Tudo renderizado com OpenGL/OpenGL ES.

love.physics

Simulacao 2D com Box2D: World, Body, Shape, Fixture. Colisoes realisticas com callback beginContact/endContact.

love.audio

Sons e musica: love.audio.newSource() cria Source de tipo stream (musica) ou static (SFX). Suporte a panning e volume 3D.

love.filesystem

Leitura e escrita segura de arquivos. Dados de save ficam no diretorio de dados do usuario. love.filesystem.read() e write().

3. Quando Usar LOVE2D

Ideal Para

  • Jogos 2D indie: plataformers, roguelikes, puzzles, RPGs
  • Aprender desenvolvimento de jogos com uma linguagem simples (Lua)
  • Game jams: startup rapido, sem boilerplate, live reload com argumento --fused
  • Prototipagem de mecanicas de jogo antes de portar para outra engine

Evitar Quando

  • Jogo 3D — LOVE2D e puramente 2D (exceto shaders customizados)
  • Precisa de editor visual, tilemap integrado ou animacoes com editor — use Godot
  • Deploy em console (Switch, PS5) — LOVE2D nao tem suporte oficial a consoles
