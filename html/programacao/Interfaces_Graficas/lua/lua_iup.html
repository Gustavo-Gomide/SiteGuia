IUP — Toolkit GUI Nativo do Tecgraf/PUC-Rio

Pequeno, rapido e nativo. O toolkit GUI oficial para scripting Lua com widgets do SO.

IUP (Interface for User Programs) e um toolkit multiplataforma desenvolvido pelo Tecgraf / PUC-Rio, o mesmo laboratorio que criou Lua. Ele usa os widgets nativos de cada SO (Win32 no Windows, GTK no Linux/Mac) — o que significa aparencia e comportamento identico ao nativo sem instalar nada extra. A API e guiada por atributos (strings chave/valor) que controlam aparencia e comportamento. O binding Lua (iuplua) torna tudo ainda mais conciso e e a forma recomendada de usar IUP.

1. Janela com Botao

lua — app iup simples
require("iuplua")

local label = iup.label{ title="Ola, IUP!" }

local btn = iup.button{
    title = "Clique",
    action = function()
        label.title = "Clicado!"
        return iup.DEFAULT  -- retorna para continuar o loop
    end
}

local vbox = iup.vbox{
    label, btn;
    margin = "16x16", gap = "8"
}

local dlg = iup.dialog{
    vbox;
    title = "Meu App IUP"
}

dlg:show()
iup.MainLoop()  -- inicia o loop de eventos

2. Sistema de Atributos

lua — atributos controlam aparencia e comportamento
local campo = iup.text{
    value    = "",
    size     = "200x",    -- largura 200px, altura automatica
    mask     = iup.MASK_FLOAT,  -- aceita apenas numeros float
    readonly = "NO",
    border   = "YES"
}

-- Atributos podem ser lidos e escritos como propriedades Lua
campo.value = "3.14"
print(campo.value)   -- "3.14"

-- Mudanca de bg e fg color
btn.bgcolor = "0 120 215"  -- RGB como string
btn.fgcolor = "255 255 255"

3. Quando Usar IUP

Ideal Para

  • Scripts Lua que precisam de dialogo simples (input, alert, file picker) sem dependencia pesada
  • Apps embarcados onde Lua e usada como linguagem de extensao e IUP como frontend
  • Ferramentas internas leves no ecossistema Lua (ex: IM, CD, IUP do pacote Tecgraf)

Evitar Quando

  • Precisa de UI rica, animacoes ou estilos CSS — IUP e minimalista por design
  • Projeto nao e em Lua — IUP existe primariamente para o ecossistema Lua/C
