Rust Iced

Iced — GUI Elm-like para Rust com wgpu

State, Message, Update, View. A arquitetura Elm no Rust, com um renderer GPU moderno.

Iced e um framework GUI cross-platform em Rust inspirado na Elm Architecture: o estado da aplicacao e imutavel entre updates; a UI e uma funcao pura do estado; interacoes produzem mensagens que disparam update(). O renderer usa wgpu (WebGPU para Rust) por padrao, possibilitando UI acelerada por GPU com visual consistente entre plataformas. Ha tambem um backend HTML5 Canvas experimental. Iced e o framework GUI Rust com maior momentum em 2024–2025, junto com egui e Xilem.

1. O Padrao Message/Update/View

rust — contador clássico em iced
use iced::{widget::{button, column, text}, Element, Sandbox, Settings};

// 1. Estado da aplicacao
struct Contador { valor: i32 }

// 2. Mensagens possíveis
#[derive(Debug, Clone, Copy)]
enum Msg { Incrementar, Decrementar }

impl Sandbox for Contador {
    type Message = Msg;

    fn new() -> Self { Contador { valor: 0 } }
    fn title(&self) -> String { String::from("Contador Iced") }

    // 3. Update: transicao de estado pura
    fn update(&mut self, msg: Msg) {
        match msg {
            Msg::Incrementar => self.valor += 1,
            Msg::Decrementar => self.valor -= 1,
        }
    }

    // 4. View: UI como funcao pura do estado
    fn view(&self) -> Element<Msg> {
        column![
            text(self.valor).size(48),
            button("+").on_press(Msg::Incrementar),
            button("-").on_press(Msg::Decrementar),
        ].spacing(16)
         .padding(32)
         .into()
    }
}

fn main() -> iced::Result {
    Contador::run(Settings::default())
}

2. Widgets e Layouts

column! / row!

Macros para layouts verticais e horizontais. column![w1, w2].spacing(8) empilha widgets com espacamento. Similar ao Flutter Column/Row.

container

Envolve um widget com padding, alinhamento e fundo. container(conteudo).padding(16).center_x().into().

scrollable

Torna qualquer coluna/row rolavel. scrollable(column![...]).height(300). Suporte a direcionamento vertical e horizontal.

canvas / shader

Canvas 2D para desenho custom implementando Program. Shader WGSL para efeitos GPU. Ideal para visualizacoes de dados e jogos simples.

3. Quando Usar Iced

Ideal Para

  • Apps Rust desktop que precisam de UI moderna sem webview (sem Tauri)
  • Desenvolvedores que gostam da previsibilidade da Elm Architecture
  • Tools e utilitarios de sistemas onde a performance do Rust e valorizada
  • Apps que precisam de rendering GPU diretamente via wgpu

Evitar Quando

  • Precisa de widgets muito especializados (data grid, rich text editor) — Iced ainda esta madurando
  • Time nao sabe Rust — a curva de aprendizado e alta
  • App mobile — Iced e desktop-first; Tauri v2 cobre mobile melhor
