FXRuby — Toolkit FOX em Ruby

Binding Ruby para a biblioteca FOX GUI. Widgets nativos, leve e ainda mantido para uso desktop.

FXRuby e um binding para a biblioteca FOX Toolkit, um toolkit C++ que desenha seus proprios widgets sem depender de GTK ou Qt. A vantagem e que FOX tem uma pegada muito menor que GTK/Qt. FXRuby permite criar apps desktop completos em Ruby com FXApp, FXMainWindow, FXButton, FXTable e outros 100+ widgets FOX. E instalado como uma gem: gem install fxruby. Mantido por Lyle Johnson, ainda recebe atualizacoes ocasionais.

1. App Basico FXRuby

ruby — janela com botao e label
require 'fox16'
include Fox

app = FXApp.new('Meu App', 'FXRuby')

main = FXMainWindow.new(app, 'Ola, FXRuby!',
                         width: 300, height: 150)

vframe = FXVerticalFrame.new(main,
           LAYOUT_FILL_X | LAYOUT_FILL_Y,
           padding: 16)

label = FXLabel.new(vframe, 'Ola, Mundo!')

btn = FXButton.new(vframe, 'Clique')
btn.connect(SEL_COMMAND) { label.text = 'Clicado!' }

app.create
main.show(PLACEMENT_SCREEN)
app.run

2. Widgets Principais

FXTable

Planilha completa com celulas editaveis, selecao de linhas/colunas, ordenacao. Mais completo que muitos grids de outras toolkits menores.

FXTreeList

Arvore hierarquica com itens expansiveis. Equivale ao TreeView do GTK. Util para estrutura de arquivos e dados hierarquicos.

FXCanvas + OpenGL

Canvas para desenho customizado e suporte a OpenGL via FXGLCanvas. Permite visualizacoes 3D internas ao app FOX.

3. Quando Usar FXRuby

Ideal Para

  • Apps desktop Ruby leves que nao querem a pegada de GTK completo
  • Manutencao de apps legados ja escritos em FXRuby
  • Desenvolvedores que preferem a estetica e modelo de widget do FOX Toolkit

Evitar Quando

  • Projeto novo Ruby desktop com UI moderna — Shoes4 ou migrar para Python/Go seria mais suportado
  • Comunidade pequena: poucos tutoriais, Stack Overflow limitado, gem com poucos stars
