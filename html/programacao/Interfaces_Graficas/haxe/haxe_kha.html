Kha — Framework Multimedia de Baixo Nivel

A base sobre a qual Armory3D e outras engines Haxe sao construidas.

Kha (Khronos Haxe Application) e um framework multimedia de baixo nivel criado por Robert Konrad. E a camada que unifica acesso a GPU (OpenGL, Direct3D, Metal, Vulkan, WebGL), audio, entrada e janela em uma API Haxe portavel para 20+ plataformas incluindo HTML5, Windows, macOS, Linux, iOS, Android e consoles. Armory3D usa Kha internamente. Se voce quer construir sua propria engine ou precisa de controle total do pipeline grafico em Haxe, Kha e o ponto de partida.

1. Estrutura de App Kha

haxe — app kha com renderizacao 2D
import kha.Framebuffer;
import kha.Scheduler;
import kha.System;

class Main {
    public static function main() {
        System.start({title: "Kha App", width: 800, height: 600}, function(_) {
            // callback de render chamado a cada frame
            System.notifyOnFrames(function(framebuffers: Array<Framebuffer>) {
                var fb = framebuffers[0];
                var g2 = fb.g2;
                g2.begin();
                g2.color = kha.Color.White;
                g2.drawString("Ola, Kha!", 100, 100);
                g2.fillRect(50, 150, 200, 50);
                g2.end();
            });
        });
    }
}

2. APIs de Rendering

g1 — Sprite Batcher

API de imagens e sprites: draw, rotate, scale com batch automatico. Ideal para jogos 2D com muitos sprites na tela simultaneamente.

g2 — 2D Graphics

Retangulos, linhas, imagens e texto com transformacoes. API mais confortavel para rendering 2D geral sem precisar de shaders customizados.

g4 — Low-Level 3D

Acesso direto ao pipeline grafico: vertex buffers, index buffers, shaders GLSL/HLSL via Kore shader language. Para quem constroi engines proprias.

3. Quando Usar Kha

Ideal Para

  • Construir uma engine propria em Haxe com controle total do pipeline GPU
  • Apps multimedia de alta performance que precisam de acesso a GPU em multiplas plataformas
  • Projetos que usam Armory3D (que usa Kha internamente)
  • Pesquisa e experimentacao com graphics programming em Haxe

Evitar Quando

  • Precisa de engine completa pronta — use Heaps para jogos em Haxe
  • Nao tem experiencia com graphics programming de baixo nivel
  • Documentacao esparsa — Kha tem comunidade menor que Heaps/OpenFL
