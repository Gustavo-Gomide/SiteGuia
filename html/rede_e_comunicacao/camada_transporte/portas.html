Camada de Transporte

Portas TCP/UDP

Números de 16 bits que identificam processos — o endereço do processo dentro do host

Uma porta é um número de 16 bits (0–65535) que, junto com o IP e o protocolo (TCP ou UDP), identifica um socket — o endpoint de uma comunicação. Enquanto o IP identifica o host, a porta identifica o processo/serviço dentro daquele host.

Faixas de Porta

Well-Known Ports (0–1023)

Atribuídas pela IANA; exigem privilégio de root/admin no servidor para bind. Usadas por serviços padrão conhecidos. Ex: HTTP=80, HTTPS=443, SSH=22.

Registered Ports (1024–49151)

Registradas na IANA para aplicações específicas; não precisam de root. Ex: MySQL=3306, PostgreSQL=5432, Redis=6379, Minecraft=25565.

Dynamic / Ephemeral (49152–65535)

Usadas pelo SO como porta de origem para conexões de cliente (ephemeral ports). O Linux usa 32768–60999 por padrão; Windows usa 49152–65535; BSD 1024–5000.
