DHCP — Dynamic Host Configuration Protocol

Atribuição automática de IP, máscara, gateway e DNS — plug & play na rede

O DHCP (RFC 2131) elimina a configuração manual de IPs em cada dispositivo. O servidor gerencia um pool de endereços e empresta (lease) endereços para clientes por um tempo determinado. Opera em UDP: cliente na porta 68, servidor na porta 67.

Processo DORA

Troca DORA — cliente sem IP
Cliente (sem IP)                       Servidor DHCP
  |                                        |
  |-- DHCPDISCOVER (broadcast) ----------> |  src: 0.0.0.0:68  dst: 255.255.255.255:67
  |   "preciso de um IP!"                  |
  |<-- DHCPOFFER (broadcast/unicast) ----- |  "ofereço 192.168.1.100 por 24h"
  |                                        |
  |-- DHCPREQUEST (broadcast) ---------->  |  "aceito 192.168.1.100 do seu servidor"
  |   (informa qual oferta aceitou)        |
  |                                        |
  |<-- DHCPACK -------------------------   |  "confirmado! 192.168.1.100/24, GW: .1, DNS: .1"
  |   [cliente configura interface]        |
  |   [ARP probe opcional antes de usar]   |

O DHCPREQUEST é broadcast mesmo após receber a oferta; informa a todos os servidores DHCP qual oferta foi aceita (outros servidores recolhem suas ofertas).

DHCP Relay e Segurança

DHCP Relay Agent

Como o DHCPDISCOVER é broadcast e não atravessa roteadores, usa-se um DHCP Relay Agent (RFC 3046) no roteador de borda de cada sub-rede para encaminhar as mensagens DHCP como unicast ao servidor central:

Cisco IOS — DHCP Relay
interface GigabitEthernet0/0
 ip helper-address 10.0.0.1  ! IP do servidor DHCP

Rogue DHCP e DHCP Snooping

  • Rogue DHCP Server: dispositivo não autorizado respondendo com IPs falsos; pode causar MITM ao definir um gateway malicioso
  • DHCP Snooping (802.1Q): recurso de switches que marca portas como trusted (uplink ao servidor DHCP) ou untrusted (clientes). Bloqueia DHCPOFFERs vindos de portas untrusted. Habilita Dynamic ARP Inspection e IP Source Guard
Cisco — DHCP Snooping
ip dhcp snooping
ip dhcp snooping vlan 10

interface GigabitEthernet0/1   ! uplink ao servidor
 ip dhcp snooping trust
