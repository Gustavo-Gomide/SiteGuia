VLANs — IEEE 802.1Q

Segmentação lógica da rede: múltiplas redes broadcast isoladas sobre a mesma infraestrutura física

Uma VLAN (Virtual Local Area Network) cria domínios de broadcast independentes sobre o mesmo switch físico. Dispositivos em VLANs diferentes não se comunicam na Camada 2 — precisam de um roteador (ou switch L3) para interagir. Isso melhora segurança, performance e organização lógica da rede.

Tag 802.1Q

Um campo de 4 bytes inserido no frame Ethernet após o MAC de origem para identificar a qual VLAN o frame pertence:

Estrutura do Tag

802.1Q Tag — 32 bits
Bits 31-16: TPID = 0x8100   (Tag Protocol ID)
Bits 15-13: PCP (3 bits)    (Priority Code Point, QoS 0-7)
Bit  12:    DEI (1 bit)     (Drop Eligible Indicator)
Bits 11-0:  VID (12 bits)   (VLAN ID: 1 a 4094)

VLAN 0   = reservado (QoS sem VLAN)
VLAN 1   = padrão Cisco (default/native)
VLAN 1002-1005 = reservadas (Token Ring/FDDI)
VLAN 1006-4094 = extended VLANs

Portas Access vs Trunk

  • Access port: pertence a uma única VLAN; frames chegam sem tag do dispositivo final (PC, impressora); o switch adiciona a tag internamente ao encaminhar
  • Trunk port: transporta frames de múltiplas VLANs simultaneamente (tagged); usado em links switch↔switch e switch↔roteador
  • Native VLAN: em trunks, frames sem tag são atribuídos à native VLAN (padrão VLAN 1); risco de VLAN hopping se mal configurado — sempre mudar a native VLAN da VLAN 1

Configuração Cisco IOS

Criar VLANs e Portas Access

Cisco IOS — VLANs + Access
! Criar VLANs
vlan 10
 name FINANCEIRO
vlan 20
 name TI
vlan 30
 name VOIP
!
! Porta de acesso para PC de finanças
interface GigabitEthernet0/1
 switchport mode access
 switchport access vlan 10
 spanning-tree portfast
!
! Porta de acesso para telefone VoIP
interface GigabitEthernet0/2
 switchport mode access
 switchport access vlan 30
 switchport voice vlan 30

Trunk e Native VLAN Segura

Cisco IOS — Trunk
interface GigabitEthernet0/24
 switchport trunk encapsulation dot1q
 switchport mode trunk
 switchport trunk native vlan 999   ! muda de VLAN 1
 switchport trunk allowed vlan 10,20,30
!
! Verificar
show vlan brief
show interfaces trunk
show interfaces GigabitEthernet0/24 switchport

Roteamento Inter-VLAN

Router-on-a-Stick (ROAS)

Uma única interface física do roteador dividida em subinterfaces — cada uma com um endereço IP e encapsulação 802.1Q para uma VLAN diferente. Simples para redes menores.

Cisco IOS — ROAS
interface GigabitEthernet0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0
!
interface GigabitEthernet0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0

SVI — Switch Virtual Interface (L3)

Em switches L3 (com capacidade de roteamento), cria-se uma interface virtual por VLAN que serve como gateway padrão dos hosts dessa VLAN. Mais eficiente que ROAS para trunks de alta velocidade.

Cisco IOS — SVI L3
ip routing   ! habilita roteamento no switch L3
!
interface Vlan10
 ip address 192.168.10.1 255.255.255.0
 no shutdown
!
interface Vlan20
 ip address 192.168.20.1 255.255.255.0
 no shutdown
